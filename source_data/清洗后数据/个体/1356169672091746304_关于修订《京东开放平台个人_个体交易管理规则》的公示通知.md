# 关于修订《京东开放平台个人/个体交易管理规则》的公示通知

尊敬的商家朋友：
您好，为适应业务发展需求，将针对《京东开放平台个人/个体交易管理规则（https://learn-jdm.jd.com/knowledge/rule/detail?ruleId=1090224406216708096）》进行调整。现进行规则公示，公示期：2026年4月10日- 2026年4月26日，规则预计于 2026年4月27日生效，公示期满平台将另行发布正式生效规则。
**核心调整点：新增**延迟发货、缺货无货、未发货订单积压、发货异常等场景违约金档位标准**，新增****违约金档位**规则解读**，****修订**虚假发货异常情形。详见下文。
一、修订虚假发货异常情形
**修订前：**赠品先发，实际主商品晚发（商家上传订单物流单号非主商品物流单号）；
**修订后：**商家上传的物流单号非订单主商品真实运单号，如后台上传A运单，实际发B运单。
二、新增延迟发货、缺货无货、未发货订单积压、发货异常等场景违约金档位标准
1.****延迟发货、缺货无货
（1）订单实际发货时间超最晚发货时间3天及以内：扣收违约金6元/单（M1档）；
（2）订单实际发货时间超最晚发货时间3天以上：扣收违约金20元/单（M1档）；同时根据商家发货超时严重程度，扣除物流分0.1-5分。
（3）商家因自身原因、缺货无货等因素，确定无法按时发货时，订单最晚发货时间前：扣收违约金10元/单（M1档）；订单超最晚发货时间后：扣收违约金20元/单（M1档）。
（4）若订单超最晚发货时间7天仍未发货，再按订单实付金额所在区间扣除违约金，最低20元/单，最高500元/单。

2.****未发货订单积压
轻微积压扣除违约金50元/次（M1档）；严重积压扣除违约金200元/次（M2档）。为保障消费者体验、控制商家资损，自然月内，轻微、严重积压累计次数达极值时（取历史自然月轻微、严重积压违规次数最多的前20%作为极值基准值），在商家再次触发轻微、严重积压时，将直接限售24小时。
（二）发货异常
1.****未按承诺使用发货方式
若商家向消费者承诺发货形式与实际发货形式不符，京东有权视违规程度对商家扣除违约金200元到800元/次（M2档）。
2.****未按承诺时效履约
若商家在商品标题、主图、详情页等承诺送达时间但实际未能履约，京东有权对商家扣除违约金10元/单（M1档）。
3.****违规加收运费
若存在包邮商品商家要求消费者额外支付运费才能发货，京东有权视违规程度对商家扣除违约金200元到800元/次（M2档）。
（三） 配送异常
若商家物流停滞违规属实，京东有权对商家扣除5元/单的违约金（M1档）；在月度考核周期内，当成交单量大于100单，且物流停滞单量的占比大于5%时，京东有权视违规程度对商家扣除违约金200元到800元/次（M2档）。
若商家未规范使用厂家自送、用户自提、无需物流发货方式违规属实，京东有权视违规程度对商家扣除违约金200元到800元/次（M2档）。若产生物流纠纷客诉问题（如周期内客诉占比较高或引发投诉及舆情等情形），京东有权永久关闭三种发货模式。
（四） 签收异常
若商家签收异常违规属实，京东有权视违规程度对商家按订单实付金额所在区间扣除违约金。

（五）严重违规
若店铺发生虚假发货、虚假轨迹或欺诈发货行为，京东有权视违规程度对商家及其关联店铺采取处置措施，包含扣除违约金、延长自动确认收货时间、冻结扣除店铺资金（包括货款、保证金等)、扣除违规所得资金、管控、限权、关店等处置措施。 其中，管控包括但不限于违规商品下架且禁止上架、降低店铺星级、全店商品流量降权/屏蔽、延长结算周期、加收风险保证金等；限权包括但不限于限制新品标识、限制使用厂自模式、店铺清退等。
