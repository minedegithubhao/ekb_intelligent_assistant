# 京东开放平台出售假冒商品细则

# 1.****【违规定义】
（1）经司法、行政机关认定商家出售假冒、盗版商品的行为。
如：各级人民法院通过判决等司法文书或市场监管、海关等行政机关通过行政文书等认定商家出售假冒商品/盗版商品；
（2）经平台抽检或权利人或其授权的代理人鉴定为假冒、盗版商品。
如：京东抽检商家的商品，被权利人或其授权的代理人或京东认可的第三方检测机构鉴定为假并出具鉴定报告。
（3）商品不符合品牌正品基本事实特征，足以认定为假冒商品，且商家无法做出合理解释的。
如：权利人未生产的商品，商品外观、标识、包装、工艺与品牌官方正品存在实质性差异。
（4）经过京东识别，与认定出售假冒商品的行为存在强关联，且商家无法有效举证的，京东有权直接认定其出售假冒商品。
如：商家存在售假投诉，经核实后，商家所售商品在商品形态、供货渠道、交易行为等，与平台已认定出售假冒商品的店铺存在高度一致性。
（5）商家明确承认销售假冒商品，且无法提供相反证据的，京东有权认定其出售假冒商品。
如：商家与客户在京东平台沟通过程中，商家承认其所售商品为假冒、非正品或非品牌授权商品，无法提供有效资质、采购链路凭证等材料的。
（6）有其他合理依据认为商家所售商品为假冒或盗版商品的。
# 2.****【处理措施】
京东有权执行全店商品下架、永久禁止发布新商品、永久禁止上架待售商品，对涉事商家及店铺进行限权和采取管控措施。扣除最低1,000,000元（M7档）或违规商品近365天成交金额的10倍作为违约金（两者从高），并执行店铺清退。如前述违约金不足以弥补京东损失的，商家仍应赔偿京东损失。
京东对涉事商家店铺及关联店铺采取包括但不限于以下措施
京东有权视商家违规程度且不基于违规次数，对涉事商家、店铺及关联店铺进行违规商品下架、禁止发布新商品、禁止上架待售商品、禁止投放或提报活动、禁止投放或播放广告、品牌停用和限制申请品牌、类目停用和限制开通类目、搜索降权、推荐降权、全店商品下架、降低店铺星级、屏蔽降权、账户止付、定额冻结、提高涉事店铺应交保证金金额、延长结算周期、执行店铺清退，同时，不基于违规次数，京东有权对商家的关联方或关联店铺采取一项或多项的管控措施，具体管控措施详见《京东开放平台关联店铺管理规则》（https://rule.jd.com/rule/ruleDetail.action?ruleId=4680&type=0&btype=1），如违约金不足以弥补京东损失的，商家仍应赔偿京东损失，视商家违规影响，将扣除商家部分或全部违规经营所得（订单款项），并对商家予以追责。

2、假冒商品包括哪些？
假冒商品是指未经品牌方或权利人许可，在相同或相似商品上使用相同商标的商品。以下情形不按照本规则处理：
（一）根据商家商品的全部信息，不会误认为属于权利人或者其授权人生产的商品；
（二）进口商品来源国的权利人与持有相同商标的境内权利人不同，且商家是持有相同商标的境内权利人或者已取得相同商标的境内权利人相应授权许可的；
（三）商品使用的商标先于注册商标在中国使用的。
3、京东是如何发现商家“出售假冒商品”的？
京东发现的渠道包括但不限于采用京东抽检、行政部门通报、媒体报道、其他投诉反馈等。投诉反馈包括：权利人投诉、消费者投诉以及其他渠道投诉反馈等。
4、商家应做好哪些资料留存？
商家应保存在售商品和过往销售商品的相关授权资质和采购链路凭证，以备查验。
5、商家因出售假冒商品被开具违约单后，可以提交哪些资料进行申诉？
（一）商家可提供包含涉诉商品型号的可追溯至权利人/出版社/品牌方的合法进货凭证，包括但不限于购物发票、电子订单、采购合同、生产厂家订/出货单、报关单、与进货凭证相对应的付款流水单等；
（二）商家可提供包含涉诉商品型号的可追溯至权利人/出版社/品牌方的合法品牌授权，包括但不限于商标注册证、商标受理通知书、生产厂家委托/授权销售凭证、授权链体系凭证等。
6、商家关联方定义？
关联方是指存在关联关系的关联方，包括但不限于：以关联人士名义入驻的关联方；同一经营主体的关联方；不同经营主体但存在同一法人代表、投资关系等关系的关联方；或通过技术手段识别出由同一实际控制人经营的关联方。
# 附则：
1.京东开放平台商家的行为，发生在本规则生效之日以前的，适用当时的规则。发生在本规则生效之日以后的，适用本规则。
2.京东可根据运营情况随时调整本规则并以公告的形式向商家公示。
3.商家应遵守国家法律、行政法规、部门规章等规范性文件。对任何涉嫌违反国家法律、行政法规、部门规章等规范性文件的行为，本规则已有约定的，适用于本规则。本规则尚无约定的，京东有权酌情处理。但京东对商家的处理不免除其应承担的法律责任。商家在京东的任何行为，应同时遵守与京东及其关联公司签订的各项协议；若商家服务协议约定的违约金金额高于规则约定的，京东有权适用协议约定。
4.本规则于2026年3月31日修订，于2026年4月8日生效。
