# 男、女装品类POP店铺升降级机制公告

- active_time: 2025-07-23

为鼓励商家提升价格竞争力及销售额，对男、女装品类POP店铺建立店铺升降级机制，核心考核销售额指标，价格竞争力和广告投放金额作为观测指标，表现优秀的专卖店可升级为旗舰店，表现不好的旗舰店降级为专卖店。
一、 适用商家范围
在京东开放平台完成开店并启用时间超过6个月，主营类目是男、女装品类的POP旗舰店、专卖店。
二、 适用类目
男、女装类目。
三、 考核周期：
2025年8月起，季度考核一次，结束时间另行通知。
四、 考核规则
1. 旗舰店降级规则
① 男、女装主营类目下店铺，同品牌下旗舰店在男、女装类目近12个月的月均销售额排名在尾部10%，则降级为专卖店。
② 男、女装类目下月均销售额50万及以上，免降级。
2. 专卖店升级规则
① 男、女装主营类目下店铺，同品牌下专卖店在男、女装类目近12个月的月均销售额排名在头部10%，且男、女装类目下月均销售额 50万以上，则可以升级为旗舰店。
② 店铺可获取到品牌方给与的旗舰店开店授权。
五、 相关说明
1. 商家采取不正当手段（如刷单、套券、乱挂等）获得的店铺升级机会，不予升级。
2. 商家存在恶意竞争、影响消费者权益、影响京东声誉等违反市场公平竞争原则、诚实信用原则、公序良俗的行为，及存在价格虚高、严重质量问题等不予升级。
3. 考核周期内所有销售额计算口径都会剔除疑似刷单、套利的销售额。
4. 考核周期内获得升级机会的专卖店，需要取得品牌方旗舰店授权，才可以最终升级为旗舰店。
5. 2025年8月拟启动第一批店铺升降级落地，后续每3个月执行一次。
