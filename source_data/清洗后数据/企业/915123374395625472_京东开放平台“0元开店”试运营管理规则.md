# 京东开放平台“0元开店”试运营管理规则

# 第一章 概述
**第一条** 为营造良好的经营环境，扶持商家快速成长，依据《“京东JD.COM”开放平台在线服务协议》等，制定本规则，本规则适用于在京东开放平台经营的商家。
# 第二章 试运营规则
**第二条** 从2023年4月1日起，针对新入驻京东开放平台的POP模式商家，部分类目支持“0元开店”试运营，即试运营期间不缴纳保证金，试运营结束进入正式运营阶段后再缴纳保证金，参与“0元开店”试运营的类目可详见：0元试运营类目明细（https://rule-jd-com.s3.cn-north-1.jdcloud-oss.com/入驻文件/0元试运营类目明细（B店）_0407.xlsx）
**第三条**为保障消费者权益，在试运营期间的商家店铺将受到下列限制：
（一）冻结商家的京东钱包账户余额1,000元，作为消费者保障金；
（二）结算周期D+15天；
（三）店铺在售商品总量上限为500SPU。
注：试运营期间，商家可随时缴纳保证金以进入正式运营阶段，足额缴纳后会解除上述限制。
**第四条** 试运营期间，若商家店铺满足以下任一条件，则试运营结束，进入正式运营期，须在3个自然日内缴纳足额保证金：
（一）店铺累计销售额≥1万元；
（二）店铺累计成交订单高于100单；
（三）京东介入投诉单≥1单；
（四）增开非试运营的类目；
（五）违规次数≥1次。
**第五条** 试运营期结束后，若商家未及时足额缴纳保证金，京东有权视情节严重程度采取下列一项或多项管控措施：
（一）货款提现功能受限；
（二）全店商品下架；
（三）商品发布、销售等店铺经营相关功能受限；
（四）限制提报各类营销活动；
（五）禁止投放广告；
（六）店铺屏蔽。
**第六条** 若商家主体或其关联店铺历史存在违规行为或存在经营风险，平台有权限制其参与“0元入驻”，并有权视具体情况要求商家补缴保证金后方可开展经营活动。
# 第三章 附则
**第七条** 京东开放平台商家的行为，发生在本规则生效之日以前的，适用当时的规则。发生在本规则生效之日以后的，适用本规则。
**第八条** 京东可根据运营情况随时调整本管理规则并向商家公示。
**第九条** 商家应遵守国家法律、行政法规、部门规章等规范性文件。对任何涉嫌违反国家法律、行政法规、部门规章等规范性文件的行为，本规则已有约定的，适用于本规则。本规则尚无约定的，京东有权酌情处理。但京东对商家的处理不免除其应承担的法律责任。商家在京东的任何行为，应同时遵守与京东及其关联公司签订的各项协议。
**第十条** 本规则于2022年11月30日发布，于2025年2月13日修订，于2025年2月20日起生效。
