# 第一章 概述
1.1****目的
为保障京东车厘子/樱桃行业市场健康发展，提升消费者的购买体验，特制定本规范。
1.2****适用范围
适用于经营以下类目的商家：
生鲜-水果-车厘子/樱桃
# 第二章 商品页面规范
2.1****经营类目
经营车厘子/樱桃的商家，应将商品发布在生鲜-水果-车厘子/樱桃类目下。
2.2****商品命名
2.2.1 进口车厘子按照：品牌名称+产地名称+品种名称（不强制要求）+销售属性结构（净重*规格+营销卖点）格式命名。
2.2.2 国产樱桃按照：品牌名称+产地名称+品种名称+销售属性结构（净重*规格+营销卖点）格式命名。
其他注意事项：
- 命名信息内容顺序不做要求。
- 命名字数限制公共标题30个汉字，60个字符；销售属性15个汉字，30个字符。
- 不同品牌、不同品种、不同净含量、不同规格的商品须建立多个SKU。
- 商品品种变更，必须建立新的SKU，不能在老品上进行修改上柜销售。
- 命名中必须明确商品净重。
- 命名中规格的等级和分级标准需按照发货商品实际情况进行展示（等级与分级标准对照表如下，例如宣称大果 J，其分级标准果径需对应26-28mm或 克重需对应6-8g）。

# 第三章 预售或预定
3.1****预售/预定要求
在京东经营车厘子/樱桃的商家参加预售/预定活动应遵守《京东开放平台生鲜商家预售/预定管理规则》要求。
3.2****预定时长要求
在京东经营车厘子/樱桃的商家单次预定活动总时长不得超过30天。
# 第四章 商品质量
4.1****商品质量标准
在京东经营的车厘子/樱桃商家均需符合国家食品安全标准及京东《生鲜感官验收标准-车厘子/樱桃》（附件一），商家应依据此标准提供商品。
4.2****现场监督
在京东经营车厘子/樱桃的商家需接受并配合京东品控及委派的第三方机构对生产车间、发货仓库等地进行的现场驻场监督，监督检查项目包含原料验收、生产加工、现场环境、成品抽检等环节，并将依据《京东开放平台车厘子樱桃现场监督检查管理办法（https://rule.jd.com/rule/ruleDetail.action?ruleId=738331391913234432）》对商家违规行为进行处置。
4.3****飞行检查
售卖过程中，京东品控或委派的第三方机构会对经营车厘子/樱桃的商家进行不定期飞行检查，依据《生鲜感官验收标准-车厘子/樱桃》（附件一）对车厘子/樱桃进行感官检查，进口车厘子检查不合格按以下处理措施对商家进行处理。

# 第五章 商品售后
5.1****赔付标准
在京东经营车厘子/樱桃的商家对商品质量及发货延迟等问题应积极配合用户处理，优先与用户达成和解，减少纠纷产生，对于产生纠纷的情况京东将按照《京东开放平台交易纠纷处理总则》进行处理，其中商家存在违规行为的，京东有权以有利于用户的合理标准处理。
5.2****危机客诉
出现危机客诉，京东有权要求商家积极配合解决问题，并针对客诉问题提供整改报告，整改报告反馈至zhaojunwei25@jd.com。（涉及以下内容将按照6.6处理）
危机客诉场景：
- 行政投诉类：消费者通过政府工商官方APP、12345热线、12315热线等渠道进行投诉，问题包含但不限于商标、商品资质、商品品质、合同、反不正当竞争、广告宣传等。
- 行政举报类：消费者对问题商品进行举报，监管受理案件后立案，进入调查取证阶段。
- 热点舆情类：热点事件指因某一问题，短时间内引起客户集中咨询或可能引起客户集中咨询、造成社会负面舆论的外部新闻和内部事件；舆情事件指外部曝光涉及或关联京东事件（如微博、贴吧投诉，网站等新闻媒体曝光）。
# 第六章 违规处理
6.1****违规发布
商品页面发布任一项不符合规范，如标题宣称不规范、商详误导宣传、类目错挂等情形，要求1个自然日内完成整改，以商家为统计单位，1个自然年内累计发现2次不符合予以警告，累计3次及以上仍未改善，将对违规商品进行扣除100-2000元、加收风险保证金、搜推降权、下架等处理。
违规程度严重的，将限制提报营销活动7日、删除违规商品降低店铺综合体验分至2.5分30天、每次扣除违约金10000元、加收风险保证金等。
除上述处理措施外，如商家多次出现违规行为或因上述行为引起重大投诉的，京东有权视违规行为对商家采取包括但不限于店铺屏蔽，全店下架，扣除违约金100000元，冻结资金，加收风险保证金，清退等处理措施，同时可依据京东平台规则其他规则进行相应处置。
6.2****违规预售/预定
违规预售对于订单所产生的如履约、商品质量等客诉及其他违规问题，将依据《京东开放平台生鲜商家预售/预定管理规则》以及相关规则统一进行处理，相关成本由违规商家承担。对于预售订单延迟发货，将根据赔付标准：优先晚发赔，晚发赔赔付后客户仍不接受，可支持按照实付金额10%，上限200元赔付；若超7天仍未发货的订单占比达到50%及以上，则京东经用户同意后，有权指定其他商家或供应商承接相应订单的履约服务。
6.3****安全指标不合格
车厘子/樱桃安全指标不合格（比如农药残留超标），将按照《京东开放平台商品质量不合格细则（https://rule.jd.com/rule/ruleDetail.action?ruleId=3848&type=0&btype=1）》处理，如果造成严重影响（比如群发类舆情、政府监管指令等）将根据影响程度实施资金冻结、店铺清退。
6.4****虚假交易
商家通过不正当方式获取虚假的商品销量、店铺评分、商品评论或成交金额等，妨害、干扰消费者购物权益，损害平台正常经营秩序的行为。将依据《京东开放平台虚假交易细则（https://rule.jd.com/rule/ruleDetail.action?ruleId=3400&type=0&btype=1）》，京东有权视违规程度采取包括但不限于扣除违约金、店铺商品搜索降权、推荐降权、删除违规商品销量和评价、限制渠道专属技术服务费、限制商家下单和支付权益、禁止提报营销活动、禁止发布上新商品、禁止商家待售商品、全店下架、终止合作等处理措施。
6.5****感官体验抽检不合格
车厘子/樱桃感官体验抽检不合格（比如包装不符、品种不符、净重不足、规格不足、腐烂霉），将按照《京东开放平台水果商品品质抽检规范》对商家进行处置。

# 第七章 附则
**7.1**京东商家的行为，发生在本管理规则生效之日以前的，适用当时的规则。发生在本管理规则生效之日以后的，适用本规则。
**7.2**京东可根据京东运营情况随时调整本管理规则并以公告的形式向商家公示。
**7.3**商家应遵守国家法律、行政法规、部门规章等规范性文件。对任何涉嫌违反国家法律、行政法规、部门规章等规范性文件的行为，本规则已有规定的，适用于本规则。本规则尚无规定的，京东有权酌情处理，但京东对商家的处理不免除其应承担的法律责任。商家在京东的任何行为，应同时遵守与京东及其关联公司签订的各项协议。
**7.4**本规则于2025年3月18日修订，于2025年3月25日生效。
