# 京东开放平台地方特产类目资质标准

- rule_id: 2514
- source_url: https://learn-jdm.jd.com/knowledge/rule/detail?ruleId=2514
- label_names:
- active_time: 2022-05-25
- update_time: 2023-12-01
备注：特产馆模式入驻的商家请参考此标准，其他类目请参考对应类目资质。

</tr>
<tr>
<td colspan="2">招商说明： 1. 暂不接受个体工商户性质企业入驻，非中国大陆企业入驻请选择入驻京东国际。 资质材料说明： 1. 商标注册证，如商标处于“注册申请受理”状态（即“TM”商标），应当不侵犯其它商标专用权。 2. 商标注册证，核定使用商品/服务项目必须包含供应商所申请产品在商标网中二级群组的相关品类，详见《商标品类对应表》。 3. 若商标权利人为自然人，则需同时提供其亲笔签名的身份证复印件；如果商标权人为境外自然人，则需同时提供其亲笔签名的护照复印件。 4. 委托加工协议（国产），若品牌权利人为生产方，无需提供；若商标注册人非生产方需提供，提供品牌权利人到生产商的完整授权，若品牌权利人为自然人，可由其法人公司与生产单位进行委托加工协议。 5. 所有资质均需要提供扫描原件或复印件加盖入驻公司红章，法人身份证复印件加签字。</td>
</tr>

</tr>
<tr>
<td colspan="2">若经营预包装食品：还需提供以下资质： 1. 《食品经营许可证》： 入驻主体持有的《食品经营许可证》，若入驻商是生产商，且经营的产品全部为入驻商生产的产品可不提交《食品经营许可证》；若当地销售预包装食品仅需备案，请上传备案回执；若当地销售预包装食品备案工作未开展，可提供情况说明。 2. 《食品生产许可证》 （经营国产商品）： 生产厂商的《食品生产许可证》，生产类别需与检验报告产品相符； 3. 生产商营业执照： 产品生产厂商企业加载“统一社会信用代码”的营业执照原件或副本，企业名称需与检验报告生产单位企业一致； 4. 委托加工协议： ①若品牌权利人为生产方，无需提供 ②若品牌权利人非生产方，需提供品牌权利人到生产商的完整授权，若品牌权利人为自然人，可由其法人公司与生产单位进行委托加工协议签署； 5. 产品清单或商品外包装正反面图片： 请提供一份产品清单，点击下载（链接http://mjbbs.jd.com/thread-160079-1-1.html） 注：其他类目行业资质标准请查阅对应类目资质标准。</td>
</tr>
<tr>
<td>品牌资质</td>
<td>资质要求说明</td>
</tr>
<tr>
<td>商标注册证/商标注册申请受理通知书</td>
<td>① 需提供由国家商标总局颁发的商标注册证（R标）或商标注册申请受理通知书（TM标），如商标处于“注册申请受理”状态（即“TM”商标）， 应当不侵犯其它商标专用权。若商标发生过转让/变更/续展，还需提供相应证明。 ② 如经营进口商品，进口品牌需提供国内商标注册证（若提交的是马德里商标，需提供领土延伸证明）； ③ 商家若销售的新鲜水果、蔬菜和鲜蛋无品牌，品牌库可不提供商标注册证，同时勾选”水果蔬菜“品牌。</td>
</tr>
<tr>
<td>销售授权书</td>
<td>① 须提供以商标持有人为源头出发的完整授权链条，且授权书必须授权至设立店铺的公司主体，不接受只授权店铺的授权书； ② 所有授权需加盖红章，如是个人持有商标授权需同时上传授权人身份证，签字名称要与商标持有人名一致。</td>
</tr>
<tr>
<td>质检报告</td>
<td>提供近1年内有效的第三方权威质检机构出具的质检报告（须含有CMA认证或CNAS认证），成品检测报告内容须包含品牌名称、产品名称和各类产品所必须的各类检测项。注：如无品牌名称，提供含品牌名称及生产单位的产品实拍图。</td>
</tr>
<tr>
<td>报关单和入境货物检验检疫证明</td>
<td>若经营进口商品，需提供近一年内的中华人民共和国进口货物报关单以及近一年内的同批次（以合同号进行判断）商品的入境货物检验检疫证明或出入境检验检疫卫生证书，报关单上应展现对应品牌名称，及商品名称，卫检证明如有附页需提供完整。 委托报关协议：若报关单上的经营单位或收货单位不在授权链上，需提供委托报关授权。</td>
</tr>
<tr>
<td>特殊资质项补充</td>
<td>具体请参考申请类目的具体资质标准，如&lt;肉制品&gt;需提供入驻商的《动物防疫条件合格证》、《动物检验检疫证明》，若为猪肉制品，需再提供入驻商《生猪定点屠宰证》等等。</td>
</tr>

附则：
1.京东商家的行为，发生在本管理标准生效之日以前的，适用当时的标准。发生在本管理标准生效之日以后的，适用本标准。
2.京东可根据平台运营情况随时调整本管理标准并向商家公示。
3.商家应遵守国家法律、行政法规、部门规章等规范性文件。对任何涉嫌违反国家法律、行政法规、部门规章等规范性文件的行为，本标准已有规定的，适用于本标准。本标准尚无规定的，京东有权酌情处理。但京东对商家的处理不免除其应承担的法律责任。商家在京东的任何行为，应同时遵守与京东及其关联公司签订的各项协议。
4. 本标准于2018年9月20日发布，2022年5月19日修订，于2022年5月26日生效。
