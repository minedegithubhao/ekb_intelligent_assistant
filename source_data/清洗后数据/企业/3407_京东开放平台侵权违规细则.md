# 京东开放平台侵权违规细则

- rule_id: 3407
- source_url: https://learn-jdm.jd.com/knowledge/rule/detail?ruleId=3407
- label_names:
- active_time: 2026-04-07
- update_time: 2026-04-08
# 1.【违规定义】
商家发布的商品或信息涉嫌不当使用他人商标权、著作权、专利权、肖像权以及其他类型权利，或者商家发布、传播或授意他人发布、传播不利于京东的网络信息、音视频等的行为。
1.1 不当使用他人商标权/著作权/专利权的行为：指包括不限于商家所出售的商品或发布的商品信息中、所使用的店铺名、域名中，存在不当使用他人商标权、著作权、专利权等权利的行为；或相关行为产生消费者投诉举报、负面舆情等不良后果。
1.2 不当使用他人肖像权：指商家所发布的商品信息或出售的商品不当使用他人肖像权的行为。
1.3 知识产权投诉：指商家收到权利人发起的知识产权投诉，京东受理权利人所投诉的不当使用他人权利的行为。
1.4 发布影响京东名誉的信息：指商家发布、传播或授意他人发布、传播不利于京东的网络信息、音视频等的行为。
1.5 其他不当使用他人权利的行为：指商家其他不当使用他人权利的行为。
# 2.【处理措施】
2.1违规程度轻微的，下架或删除违规商品；不超过800元（M2档）；
2.2违规程度一般的，下架或删除违规商品，扣除违约金2000元-10000元不等；按照违规商品/商家近14日销售额的10%作为违约金，最低100元，最高10,000元（M3档）；
2.3违规程度严重的，下架或删除违规商品，扣除违约金5000元-40000元不等；按照违规商品/商家近14日销售额的20%作为违约金，最低300元，最高100,000元（M4档）；
2.4违规程度极其严重的，下架或删除违规商品，违规扣除违约金20000元-100000元不等；按照违规商品/商家近14日销售额的30%作为违约金，最低500元，最高500,000元（M5档）；
2.5 商家发布、传播或授意他人发布、传播不利于京东的网络信息、音视频等的，京东有权扣收不低于人民币1000000元的违约金；不足以弥补京东损失的部分，商家应继续予以赔偿。
2.6除上述措施外，京东有权视违规行为对违规店铺及其关联店铺采取包括但不限于扣除违约金、违规商品下架或修改（商品信息修改）、禁止发布新商品、禁止上架待售商品、限制投放或提报活动、限制投放或播放广告、品牌停用和限制申请品牌、类目停用和限制开通类目、搜索降权、推荐降权、全店商品下架、降低店铺星级、账户止付、定额冻结、提高涉事店铺应交保证金金额、延长结算周期、执行店铺清退等处理措施，并且京东有权视商家违规影响，扣除商家部分或全部违规经营所得（订单款项），并对商家予以追责。
# 3.【规则解读】
3.1 “侵权违规”是指不当使用他人的哪些权利？
指除出售假冒商品以外的不当使用他人商标权（包括但不限于商标近似、突出使用商标、未经授权以权利人的商标作为店铺名称等）；除出售盗版商品以外的不当使用他人著作权（包括但不限于盗用图片、字体侵权、盗用艺术设计/动漫周边/视频等）；不当使用他人专利权（包括但不限于外观专利、实用新型、发明专利等）；及其他不当使用他人权利的行为。
其中，对于不当使用他人商标权、著作权、专利权的情形，具体见下：
3.1.1 不当使用他人商标权：是指商家在出售商品或服务过程中侵犯他人商标权利，但不属于假冒的情形。包括但不限于以下情形：
3.1.1.1 未经权利人许可，在店铺页面、商品主图等位置突出展示权利人商标；
3.1.1.2 被投诉商品和权利人商品属同类商品，商家所使用的商标标识和权利人注册商标近似，且未取得有效的商标权；
3.1.1.3 被投诉商品被司法机关裁定或被行政机关认定构成商标侵权；
3.1.1.4 除上述情形外的其他商标侵权行为。
3.1.2 不当使用他人著作权：是指商家所售商品或在页面中侵犯他人著作权但不属于盗版的情形。包括但不限于以下情形：
3.1.2.1 未经权利人许可，在店铺、商品页面或商品包装上使用权利人拥有著作权的图片、文字作品、动漫形象及衍生品、奥林匹克标志、体育赛事视频、演出表演、人物肖像等情形；
3.1.2.2 商家所售的为正规出版社出版的图书，但权利人证明图书是在其对该出版社的版权授权已终止或超期后出版发行的；
3.1.2.3 被投诉商品被司法机关裁定或被行政机关认定为著作权侵权；
3.1.2.4 除上述情形外的其他著作权侵权行为。
3.1.3 不当使用他人专利权：是指商家出售的商品侵犯他人外观设计专利，实用新型专利或发明专利。包括但不限于以下情形：
3.1.3.1 未经权利人许可，被投诉商品使用了权利人专利的技术方案、内部结构或外观设计；
3.1.3.2 被投诉商品被司法机关裁定或被行政机关认定为专利侵权；
3.1.3.3 除上述情形外的其他专利侵权行为。
3.1.4 不当使用他人肖像权：是指商家未经肖像权人许可，在出售商品或服务过程中使用了肖像权人的肖像。包括但不限于以下情形：
3.1.4.1 未经肖像权人许可，在店铺、商品页面或商品包装上使用肖像权人拥有肖像权的图片；
3.1.4.2 在肖像权人授权已终止或超期后，商家仍然在店铺、商品页面或商品包装上使用肖像权人拥有肖像权的图片；
3.1.4.3 除上述情形外的其他肖像权侵权行为。
3.2 京东如何判定商家侵权违规的违规程度？
京东有权基于商家“违规次数”、“违规个数”、“违规恶意程度”、“商家因违规所造成的影响和损失”等情况综合判定商家违规程度。
3.3 什么叫“权利人”？
权利人是指拥有商标权、著作权、专利权等知识产权或肖像权、其他合法权利的法人、其他组织或自然人。
3.4 短期内商家会被多次投诉么？
京东处理投诉及商家接到投诉后进行整改需要时间，同一权利人在3日内对同一商家的投诉视为一次投诉。
3.5 如何查询店铺是否已收到权利人的投诉？对于权利人的投诉，京东会进行如何处理？
商家可登陆京东知产维权平台（https://jdgroupip.com/）查看投诉详情。京东会根据同一店铺在年度内的累计被投诉成立的次数，综合判定做出处理。
3.6 侵权违规申诉材料如何准备？
商家可提供包括但不限于以下资料进行申诉：
3.6.1 针对商标侵权投诉申诉凭证。
3.6.1.1 完整合法正规进货凭证：指向商标权利人或其授权经销商的进货凭证（发票/报关单/加盖出货方公章的进货单据/正规商场小票），并体现品牌、具体名称、型号或款式信息；
3.6.1.2 合法销售授权凭证：授权销售证明链完整，应明确指向权利人至申诉方、授权凭证所涵盖的商品范围应包括被举报商品、授权凭证应在有效期内，应有权利人签字或盖章；
3.6.1.3 商标使用授权凭证：授权链完整，应明确指向权利人至申诉方、授权凭证所涵盖的商品范围应包括被投诉商品种类、授权凭证应在有效期内，应有权利人签字或盖章；
3.6.1.4 其他材料，如未使用权利人商标的截图、书面说明等、商家自有的商标注册证扫描件、与本投诉商品有关的生效司法判决，行政裁定等证明文件（若有）等。
3.6.2 针对专利侵权投诉申诉凭证。
3.6.2.1 被控侵权商品来自于投诉人的凭证；
3.6.2.2 自有专利权或他人专利权合法生产的凭证（权属证明和/或授权证明）；
3.6.2.3 与投诉人外观设计专利不相同且不相近似的对比说明；
3.6.2.4 与投诉人发明、实用新型专利采用的技术不同的对比说明；
专利申请日之前已公开相同技术方案或外观设计的文献资料或销售记录；
3.6.2.6 与本投诉商品有关的生效司法判决，行政裁定等证明文件（若有）等。
3.6.3 针对著作权侵权投诉申诉凭证。
3.6.3.1 著作权许可协议，证明商品合法来源的凭证，著作权登记证书或其他相关权利证明等；
3.6.3.2 针对盗用图片的投诉，可提交被投诉图片设计原稿及其他可证明被投诉图片系其自行创作的证明材料；
3.6.3.3 针对盗用视频的投诉，可提交被投诉视频的独立拍摄底稿及其他可证明被投诉视频系其自行创作的证明材料；
3.6.3.4 未使用权利人著作权的截图、书面说明等；
3.6.3.5 被投诉商品相关的生效司法判决、行政裁定等证明文件（若有）等。
3.6.4 针对肖像权侵权投诉申诉凭证。
3.6.4.1 肖像权许可协议，可提交至肖像权人至商家的逐级肖像权授权许可证明，授权链应当清晰完整，授权凭证应在有效期内，应有权利人签字或盖章等；
3.6.4.2 未使用肖像权人肖像图片的证据，店铺、商品页面或商品包装未使用肖像权人图片的截图、书面说明等；
3.6.4.3 相关的生效司法判决、行政裁定等证明文件（若有）等。

# 4.****附则：
4.1商家的行为，发生在本规则生效之日以前的，适用当时的规则。发生在本规则生效之日以后的，适用本规则。
4.2京东可根据运营情况随时调整本规则并以公告的形式向商家公示。
4.3商家应遵守国家法律、行政法规、部门规章等规范性文件。对任何涉嫌违反国家法律、行政法规、部门规章等规范性文件的行为，本规则已有约定的，适用于本规则。本规则尚无约定的，京东有权酌情处理。但京东对商家的处理不免除其应承担的法律责任。商家在京东的任何行为，应同时遵守与京东及其关联公司签订的各项协议；若商家服务协议约定的违约金金额高于规则约定的，京东有权适用协议约定。
4.4本规则于2026年3月31日修订，于2026年4月8日生效。
