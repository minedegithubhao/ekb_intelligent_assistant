# 关于修订《京东开放平台本地生活/旅游出行/到家服务/二手回收类目资质标准》的公示通知

- active_time: 2025-05-14
- update_time: 2025-05-08
尊敬的商家/供应商朋友：
您好，为适应业务发展需求，将针对《京东开放平台本地生活/旅游出行/到家服务/二手回收类目资质标准》进行调整。现进行规则公示，公示期：2025年5月8日- 2025年5月14日，规则预计于 2025年5月15日生效，公示期满平台将另行发布正式生效规则。
核心调整点：①将“酒店预定”调整为“酒店预订”；②增加国内酒店三级类目行业资质要求。
修订具体内容如下：
酒店预订
热销酒店
1. 酒店集团：（1）提供完整的公司资质，无需提交类目资质；（2）开通酒店套餐类目需提交食品经营许可证。
2. 单体酒店：（1）《卫生许可证》；（2）《特种行业许可证》；（3）开通酒店套餐类目需提交食品经营许可证。酒店网络平台服务商：平台下至少有3家合作酒店。
3. 旅行社：（1）总社：《旅行社经营许可证》；《旅行社责任险保险单》；（2） 分社：《分社备案登记证明》、（保险若为总社统一购买请提供总社的）、总社的《旅行社业务经营许可证》。
4. 酒店代理商：需提交资质担保书或与酒店合作协议。

国内酒店
国内酒店提供行业资质二选一：
1.特种行业许可证 规格要求：原件照片图片清晰，复印件要求加盖酒店公章（公章要求与特行证酒店名称一致）。
2.酒店门头照 规格要求：酒店必须有明显的门头招牌，照片要求白天自然光下拍摄清晰的明显的门头招牌。
营业执照经营范围需有：住宿服务相关内容。
