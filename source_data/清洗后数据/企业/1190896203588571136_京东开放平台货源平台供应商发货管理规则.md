# 京东开放平台货源平台供应商发货管理规则

# 第一章 概述
1.1****目的及依据
为营造京东货源平台良好的交易环境， 现规范供应商的发货及交易行为，保障买家权益，依据《京东开放平台发货管理规则》（https://rule.jd.com/rule/ruleDetail.action?ruleId=2488&btype=1）等平台已有规则及要求，特制定本规则，旨在共同维护京东货源平台的生态秩序和氛围。 本规则适用于在货源平台经营的供应商。
1.2****京东发货原则
供应商在京东货源平台经营，应按照对买家的承诺，遵守按时发、真实发、准确发、送达快的原则，积极履行发货义务。
# 第二章 发货要求
2.1****发货标准
## 2.1.1****发货相关定义
（1）发货是指供应商将快递运单号上传至京东系统并点击出库（以京东系统内记录的出库时间为准）；
（2）揽件是根据运单号在对应物流公司系统查看到的第一条物流信息 (包含揽件、中转、派送、签收等)；
（3）在线付款的订单买家成功付款后，供应商需按照订单的承诺发货时效完成对相应买家订单的发货及揽件。
2.1.2****发货方式要求
买家下单后，供应商可根据自身的配送能力和买家的需求，在规定的时效内，采用最适当的发货方式履行配送服务。发货包装中不得有其他平台的店铺信息（包括但不限于链接、二维码、联系信息），或与该店铺商品无关的信息。供应商发货若要采用合单发货的方式发货，必须提前与买家进行协商。发货方式以系统后台允许供应商使用的方式为准。
2.1.3****发货时效要求
（1）Promise设置：店铺商品必须设置承诺发货时效，同时供应商可自主设置承诺配送时效，在商详页、结算页和订单全程跟踪页将展示供应商设置的承诺发货时效及配送时效，从而增强店铺潜在转化率及买家体验，时效越快越精准将有机会在前台获得更多流量倾斜。设置及操作详见：https://mtt.jd.com/article/articleView/14affa94-327e-4322-8ffb-efd5de876a27?rootId=null（https://mtt.jd.com/article/articleView/14affa94-327e-4322-8ffb-efd5de876a27?rootId=null）；设置承诺发货时效后，供应商应在设置时效内完成对相应买家订单的发货及揽件；没有设置承诺发货时效的商品，供应商需要在买家支付成功后的48小时内完成对相应买家订单的发货及揽件。
（2）如供应商遇国家大型活动、天气因素、系统对接问题、停电、库房盘点、仓库搬迁等特殊事项，可进行延迟发货报备，报备流程详见《京东开放平台服务延迟报备规则》（https://rule.jd.com/rule/ruleDetail.action?ruleId=972685729052364800&btype=1）。经评估报备成功的店铺，需在商详页、店铺首页等“显著位置”做出公告，发货时间以公告页面为主。“显著位置”标准：包括但不限于商详页、店铺首页等，该版块须靠前或居中位置，字体大于该版块其他编辑字体并加粗。
2.2****配送标准
2.2.1****配送方式要求
供应商在商详页承诺或与买家约定特定承运人（如特定物流公司）和承运方式（如全程冷链运输）的情况下，供应商应按照承诺内容积极履约。若供应商承诺未兑现，买家有权拒收商品，相应损失由供应商承担。如商品配送存在超区问题，供应商应在商详页说明或与买家另行约定，否则供应商需提供免费转运服务，如买家同意自行提货，由此产生的自提费用由供应商承担。
2.2.2****配送周期要求
除新疆、西藏、内蒙古、宁夏、甘肃、青海偏远地址和定制、大件、拍卖的商品外，供应商应当在商详页中对商品的生产周期、配送周期予以说明，如商详页有说明的以说明为准，无说明的配送周期默认为7天；配送周期超时（从下单到收货），供应商应积极协助买家查询配送周期超时的原因及商品下落。
# 第三章 发货场景定义
3.1****发货超时
3.1.1延迟发货：指供应商设置商品的Promise承诺发货时效后，但揽件时间超过承诺的发货期限或本规则规定的发货期限。
（1）完成揽件：指供应商在承诺发货时效内，上传订单对应的物流单号至京东并有物流公司揽件信息。
（2）揽件信息：指根据上传的运单号在对应物流公司系统查看到的第一条物流信息 (含揽件、中转、派送、签收等)。
3.1.2缺货无货：指供应商告知京东或买家无法发货；或者京东或买家无法在发货时限内联系上供应商。包括但不限于以下情形：
（1）供应商告知京东或买家无法发货。
（2）订单超承诺发货时间3天以上仍然没有发货。
3.2****配送异常
配送过程中，配送异常包括但不限于配送超区、订单揽收后物流信息长时间未更新、配送周期超7天未收到货等情形等场景。其中，订单揽收后物流信息长时间未更新具体指：揽件-派送过程中任意轨迹截点超过规定时效未更新，归为停滞，具体判定规则见下表。

3.3****虚假发货
指供应商上传至后台的订单物流单号对应的物流信息存在明显异常等情形，及供应商未真实发货的其他情形。包括但不限于以下几种情况
（1）供应商上传物流单号后，24小时内在平台内仍无揽收记录；
（2）赠品先发，实际主商品晚发（供应商上传订单物流单号非主商品物流单号）；
（3）其他订单物流信息异常的情形。
3.4****虚假轨迹
指供应商上传至后台的订单物流单号对应的物流轨迹存在明显异常的情形，及供应商上传物流单号后买家在合理期限内未实际收到所购商品。包括但不限于以下几种情况
（1）供应商上传的物流单号，出现首条物流轨迹信息后的24小时内没有后续的物流轨迹信息更新；
（2）供应商上传的物流单号对应的物流轨迹与买家订单显示的收货地址不符；
（3）供应商上传的物流单号，已有不同收货人或不同收货地址的订单使用该物流单号发货；
（4）供应商上传的物流单号存在其他物流轨迹明显异常或买家在供应商发货后合理期限内未实际收到所购商品的情形，如揽收时间早于订单下单时间等；
（5）供应商上传的订单物流单号存在异常、重复物流信息；
（6）其他订单物流轨迹异常的情形。
3.5****欺诈发货
指供应商未真实发货，在发货过程中向买家发送空包裹或与商品描述明显不符的物品，或者实施其他情节严重的欺诈行为。包括但不限于下列场景
（1）供应商上传订单物流单号后，买家收到空包裹；
（2）供应商上传订单物流单号后，买家收到与商品描述明显不符的其他物品；
（3）供应商在发货过程中实施欺诈行为（如未经买家同意私自修改买家订单信息等），导致买家在发货后合理期限内未实际收到所购商品，情节严重的情形；
（4）其他存在欺诈发货的情形。
3.6****签收异常
签收异常：签收过程中，签收异常包括但不限于商品出现少件、错件等场景。
3.6.1少件：供应商未按照订单明细和网页商品描述或者商品实际重量/配件等打包发送订单商品，买家投诉收到的订单商品存在少配件、少赠品，少商品的情况。
3.6.2错件：供应商在非主观恶意的情况下，给买家发送的商品与达成交易时的订单商品不一致，特指同一商品物理属性问题，如：尺码、尺寸或颜色等。
# 第四章 发货场景违规处置
4.1****发货超时
订单实际发货时间超承诺发货时间3天及以内：扣收违约金6元/单；
订单实际发货时间超承诺发货时间3天以上：扣收违约金20元/单；
供应商因缺货无货等因素，确定无法发货时，订单承诺发货时间前：扣收违约金10元/单；订单超承诺发货时间后：扣收违约金20元/单。
一个订单发货超时被记作一次违规行为，除按单扣收违约金外，店铺月度违规首次警告，2-5次下架违规商品，5次以上将店铺在货源平台中移除。

4.2****配送异常
若供应商配送异常违规属实，京东有权对供应商扣除5元/单的违约金。
4.3****签收异常
若供应商签收异常违规属实，京东有权视违规程度对供应商扣除不低于10元/单，不高于400元/单的违约金。
除按单扣收违约金外，店铺月度违规首次警告，2-5次下架违规商品，5次以上将店铺在货源平台中移除。
4.4****严重违规
若店铺发生虚假发货、虚假轨迹或欺诈发货行为，京东有权视违规程度对供应商及其关联店铺采取处置措施，包含扣除违约金、冻结扣除店铺资金（包括货款、保证金等)、扣除违规所得资金、管控、限权、关店等处置措施。 其中，管控包括但不限于违规商品下架且禁止上架、降低店铺星级、全店商品流量降权/屏蔽、延长结算周期、加收风险保证金等；限权包括但不限于限制新品标识、移除货源平台等。
其中，虚假发货和虚假轨迹扣除违规订单实付金额1~10倍的违约金（单次违规违约金最低不少于200元）；欺诈发货扣除店铺月成交金额10%以上的违约金（单次违规违约金最低不少于500元）。
4.5****其它补充
供应商经营过程中，因晚发、不发、未按承诺发货、无货或其他情形导致买家投诉，为保障更多买家体验，京东有权根据严重程度采取警告、商品下架、移除货源平台等措施，进行风险防控。若供应商出现发货异常或签收异常的情形，引发买家集中性投诉，京东有权全店商品下架且禁止上架，同时冻结供应商钱包、严重时移出货源平台处置。
# 第五章 附则
供应商的行为，发生在本规则生效之日以前的，适用当时的规则。发生在本规则生效之日及以后的，适用本规则。京东可根据运营情况随时调整本规则并向供应商公示。
供应商应遵守国家法律、行政法规、部门规章等规范性文件。对任何涉嫌违反国家法律、行政法规、部门规章等规范性文件的行为，本规则已有约定的，适用于本规则，本规则尚无约定的，依照法律法规处理。供应商依据相关规则承担的相关责任并不免除其应承担的法律责任。供应商在京东的任何行为，应同时遵守与京东及其关联公司签订的各项协议。若供应商服务协议约定的违约金金额高于规则约定的，京东有权适用协议约定。
本规则于2024年12月30日发布，于2025年1月6日生效。
