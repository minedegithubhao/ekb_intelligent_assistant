# 京东开放平台文娱类目资质标准

行业/特殊资质及要求
1. 销售音像制品商品，营业执照经营范围需包含“音像制品零售/批发”等项目；销售数字藏品类目商品，营业执照经营范围需包含版权服务相关项目（版权代理、知识产权服务、数字文化创意内容应用服务、数字内容制作服务等，至少满足一项）。
2. 经营进口CD，需提供进出口公司的销售协议，如中间存在经销商，可逐层授权。
3. 经营数字藏品，需签署承诺函。附：承诺函模板（http://box.jd.com/sharedInfo/32F8AAA97C8A3E3FEEC946AFBC5707A7），访问密码: 7a9byz；发品前需提交版权权利证明及授权书（商品维度资质）。
4. 经营国产音像类，需提供《出版物经营许可证》，《出版物经营许可证》的公司基础信息需与营业执照一致。
5. 经营进口音像类，需提供《出版物经营许可证》，以及委托进出口公司的《出版物经营许可证》，《出版物经营许可证》经营范围需包含“进出口音像”、“音像零售”、“出版物零售”等。
6. 经营游戏—非光盘类，如为游戏发行方，需提供广电总局出具的《关于同意出版运营国产网络游戏〈XXXX〉的批复》或《关于同意进口网络游戏〈XXXX〉的批复》或广电总局出具的《网络游戏出版物号核发单》；如为代理方，需提供《网络文化经营许可证》或《增值电信业务经营许可证》，以及游戏版权方的授权资质。
7. IP在线服务为定向经营类目，在新增经营类目时需经审核后方可运营。

## 附则
1. 京东商家的行为，发生在本管理标准生效之日以前的，适用当时的标准。发生在本管理标准生效之日以后的，适用本标准。
2. 京东可根据平台运营情况随时调整本管理标准并向商家公示。
3. 商家应遵守国家法律、行政法规、部门规章等规范性文件。对任何涉嫌违反国家法律、行政法规、部门规章等规范性文件的行为，本标准已有规定的，适用于本标准。本标准尚无规定的，京东有权酌情处理。但京东对商家的处理不免除其应承担的法律责任。商家在京东的任何行为，应同时遵守与京东及其关联公司签订的各项协议。
4. 本规则标准于2025年9月15日修订，于2025年9月22日生效。
