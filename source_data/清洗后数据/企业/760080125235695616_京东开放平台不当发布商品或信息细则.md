# 京东开放平台不当发布商品或信息细则

# 1.****【违规定义】
商家未按照国家法律法规、京东规则或其他标准要求（包括但不限于《京东开放平台商品信息规范总则（https://rule.jd.com/rule/ruleDetail.action?ruleId=1126064694289895424）》、规范、行业标准等），通过发布商品或信息而损害消费者合法权益的行为。不当发布商品或信息包括但不限于以下情形：品牌信息不一致、标题/主图/属性违规、不当发布营销类信息、商品发布渠道不合规、重复铺货、更换商品、商品信息综合违规、商品信息恶意对抗等。
1.1****商品发布渠道不合规：
**1.1.1虚拟商品发布渠道不合规：**指未按照《京东行业运营标准》规定的通过虚拟商品发布渠道进行发布商品的违规行为，虚拟商品发布操作路径详见：http://psm.jd.com/article/articledetail.html?id=671。
**1.1.2****非授权第三方软件发品：**指通过使用非京东官方软件工具，也非服务商在京麦服务市场中提供的软件工具进行发品的违规行为，管理公告详见：https://mtt.jd.com/article/articleView/b6bc1d74-a2b3-48b5-97ad-2d77c183f55b。
1.2****重复铺货：
商家的店铺中同时出售两件及以上同款商品。包含以下情形：
**1.2.1****同店铺重复铺货：**同一店铺内发布两件及以上同款商品的行为；
**1.2.2****跨店铺重复铺货：**与存在关联关系的店铺发布两件及以上同款商品的行为。
**1.3****SPU****乱绑：**指未按照绑定规则发布商品的情形，包括但不限于配置为单选且必填的公共属性，对应不同属性值的商品放置在一个商品链接中售卖。
例：类别属性为单选且必填，则不同类别的商品不能放置在一个商品链接中售卖，需创建多个商品链接进行售卖。
**1.4****品牌错误使用：**指商品所挂靠的品牌和类目，超出品牌对应的商标分类范围，包括但不限于商品品牌挂靠类目与商标分类不一致、商品品牌无真实对应商标、商品品牌未获得品牌方授权等。
# 2.****【处理措施】
2.1****商品发布渠道不合规
**2.1.1**违规程度轻微：警告，违规商品搜索、推荐降权、限制在售商品阈值；
**2.1.2**违规程度一般：限制创建商品、限制通过接口批量操作商品、违规商品屏蔽；
**2.1.3**违规程度严重：禁止上架待售商品、违规商品下架；
**2.1.4**违规程度极其严重：禁止发品、违规商品删除；
**2.1.5**除上述处理措施外，如商家多次（两次及以上）或大量在售商品出现以上违规行为，或因上述行为引起重大投诉或舆情的，京东有权视违规行为对商家采取包括但不限于删除商品、冻结保证金、冻结货款、限制提报营销活动、扣除保证金等处理措施。
2.2****重复铺货
**2.2.1**违规程度一般：违规商品搜索、推荐降权50%，限制申请新品标签（10日）；
**2.2.2**违规程度严重：违规商品搜索、推荐降权70%，限制参与营销活动（7日），限制申请新品标签（10日），扣除违约金1000元；
**2.2.3**违规程度极其严重：违规商品搜索、推荐降权100%，广告屏蔽（60日），限制参与营销活动（15日），限制申请新品标签（10日），扣除违约金2000元；
**2.2.4**除上述处理措施外，如商家多次（两次及以上）或大量在售商品出现以上违规行为，或因上述行为引起重大投诉或舆情的，京东有权视违规行为对商家采取包括但不限于限制创建商品、禁止上架待售商品、永久限制通过接口批量操作商品、限制提报营销活动、店铺屏蔽、商品屏蔽、品牌停用、类目停用、店铺清退、冻结保证金、冻结货款、跨类目经营限制等处理措施。
2.3SPU****乱绑
**2.3.1**违规程度轻微：警告，违规商品解绑；
**2.3.2**违规程度一般：警告，违规商品搜索、推荐降权（降权比例基于严重程度25%-100%不等）及解绑；
**2.3.3**违规程度严重：警告，违规商品搜索、推荐屏蔽及解绑；
**2.3.4**违规程度极其严重：警告，违规商品下架；
**2.3.5**除上述处理措施外，如商家多次（两次及以上）或大量在售商品出现以上违规行为，或因上述行为引起重大投诉或舆情的，京东有权视违规行为对商家采取包括但不限于删除商品、资金冻结、扣除保证金、页面止付等处理措施。
2.4****品牌错误使用
**2.4.1****违规情节轻微：**警告，违规商品需在提示整改期内完成整改；
**2.4.2****违规情节一般：**警告，违规商品屏蔽；
**2.4.3****违规情节严重：**警告，违规商品下架；
**2.4.4****违规情节极其严重：**除上述处理措施外，如商家多次（两次及以上）或大量在售商品出现以上违规行为，或因上述行为引起重大投诉或舆情的，京东有权视违规行为对商家采取包括但不限于限制创建商品、禁止上架待售商品、永久限制通过接口批量操作商品、限制提报营销活动、店铺屏蔽、商品屏蔽、品牌停用、类目停用、店铺清退、冻结保证金、冻结货款、跨类目经营限制、页面止付等处理措施。
# 【规则解读】
1****、虚拟商品应如何发布？
虚拟商品需依据《京东行业运营标准》，通过虚拟商品发布平台进行发布，虚拟商品发布操作路径详见http://psm.jd.com/article/articledetail.html?id=671（http://psm.jd.com/article/articledetail.html?id=671）。
2****、如何规避类目乱挂违规？
商家在创建商品时，第一步即选择商品所在的类目，在类目选择框下录入类目时务必保证类目准确，部分类目可参考类目释义。

4、 重复铺货违规中同款商品是指怎样的商品？
同款商品是指完全相同以及商品的重要属性完全相同或高度相似的商品。对于不同的商品，商家必须在商品的标题、描述、图片、重要属性等方面体现商品的不同，否则京东有权判定为同款商品。
5、哪些情形属于在店铺发布两件及以上同款商品（SKU）？
同款商品包括但不限于以下情形：
（1）店铺中同时出售两件以上完全相同的商品，两件完全相同商品在不同的SPU下或者两件商品在相同的SPU下；
图片：https://img30.360buyimg.com/venderrule//jfs/t1/135044/23/33815/85283/64914f18F64d0c94e/159585bd642c1737.png
图片：https://img30.360buyimg.com/venderrule//jfs/t1/96967/7/41250/79931/64914f1cFef9ea85e/5d13d42b4ba8d1a1.png
（2）同款商品，通过更改其价格、时间、数量、组合方式及其它发布形式进行多次发布；
图片：https://img30.360buyimg.com/venderrule//jfs/t1/191390/35/30043/25001/64914f4cF8bd322cc/dcc84b562bd5af31.png
（3）各类目下商品，出现诸如以下任一情形的：
1）同款商品不同颜色分别发布：例如，一条连衣裙，商家店铺中出售的一件商品，其主图中分别展示了红色款、白色款，但店铺中又单独出现了该连衣裙的红色款或白色款；
2）同款商品以大小规格不同分别发布；
3）同款商品附带不同的赠品分别发布；
4）专车专用脚垫、专车专用座套、专车专用后备箱垫、DVD导航类目的专车专用商品在不同车型分别发布。
6****、针对在店铺中同时出售两件及以上同款商品，京东将如何处理违规商品？
京东有权保留同款商品中商品质量得分最高的一件，删除其他同款商品。
7****、 商家应如何关注重复铺货预警？
在商家后台 >体检中心（https://illegal.shop.jd.com/duplicateAlert） > 违规预警（https://illegal.shop.jd.com/legalWarningList） > 重复铺货预警中，可查看每日预警信息，商家需要对表中违规商品(SPU)进行自查处理，并根据系统提示的商品信息质量分以及GMV等信息对商品做出下架、删除等处理。 若预警后仍不整改，除删除违规商品外，京东有权视违规程度采取包括但不限于扣分、删除违规商品、限制创建商品、限制新品标签权益、店铺商品搜索降权及限制提报营销活动等处理措施。
8****、商家更换商品违规包含哪些情形？
商家更换商品包括但不限于以下情形：
1）将商品A更换成不同的商品B
2）将虚拟链接（如赠品链接、补差价链接）更换成实体商品
3）将商品更换为不同颜色的同款商品，比如白色裤子变成黑色裤子
4）将商品更换为不同型号的商品
5）将商品更换为不同规格的同款商品，比如100ml更换为200ml
6）将商品更换为不同类目的商品，比如将手机壳更换为手机
7）将单一商品更换为同类商品组合装，比如将洗发水更换为洗发水+护发素的组合装
9****、商家更换商品违规会造成什么不良影响？
（1）消费者在浏览商品时，会在商品评价界面看到两种不同商品的评价或者消费者在初次购买之后想要对某商品进行复购或评价，进入商详页发现商品更换，产生不好的消费体验；
（2）商家利用原商品的流量或者评论来提高更换后新商品的销量，涉嫌不正当竞争。
10****、如何规避更换商品违规？
（1）通过正规流程发布新品
商家如需上架新商品，需遵守商品信息发布规范且按流程操作，避免因直接更改已有商品关键信息而违规；
（2）关注预警提醒，及时整改
预警提醒查看路径:商家后台-体检中心-违规预警-更换商品预警(如下图)：
图片：https://img30.360buyimg.com/venderrule//jfs/t1/119827/30/35274/63740/64914f62F8bd4f4cb/f5a6a3653323085f.png
（3）提交申诉时确保有理有据
商家如果收到更换商品的违规预警提醒，请第一时间整改。如收到违约单并对违约单内容有异议，请在规定时间内提交申诉并附上证据证明未更换商品，避免提交无效申诉。一旦违约单生效，被下架的商品不支持再次上架。
11****、商家出现不当发布商品或信息违规后，应如何做？
（1）销售商家需及时对店铺装修页面、商品详情页面进行自查自检，检查商品标题、主图、属性等是否违规并及时修改；
（2）销售商家需注意查看预警信息及违约单信息，预警渠道为：商家后台-体检中心-违规预警。及时删除违规信息，并按规定修改商品信息，避免因此类商品违规造成商品下架等。
12****、什么样的行为会造成不当发布商品或信息中的严重、极其严重违规？
包含但不限于以下情形：
1）店铺内大量商品或者全部商品存在不当发布商品信息行为；
2）店铺多次发生“违规程度轻微或者严重”情形；
3）商家无视规则，重复上架违规商品；
4）商家逃避、对抗，或通过滥发商品信息获取不正当利益；
5）商家因不当发布商品或信息行为，造成重大客诉舆情。

## 附则：
1.商家的行为，发生在本规则生效之日以前的，适用当时的规则。发生在本规则生效之日以后的，适用本规则。
2.京东有权对违规商品/信息采取直接修改或删除违规标题、主图、属性、详情页等以保障商家合规经营。如商家对京东的处置方式有异议，可通过申诉渠道提交申诉，京东将根据实际情况进行审核。
3.商家应遵守国家法律、行政法规、部门规章等规范性文件。对任何涉嫌违反国家法律、行政法规、部门规章等规范性文件的行为，本规则已有约定的，适用于本规则。本规则尚无约定的，京东有权酌情处理。但京东对商家的处理不免除其应承担的法律责任。商家在京东的任何行为，应同时遵守与京东及其关联公司签订的各项协议；若商家服务协议约定的违约金金额高于规则约定的，京东有权适用协议约定。
4.京东可根据运营情况随时调整本规则并以公告的形式向商家公示。
5.本规则于2026年4月1日修订，于2026年4月8日生效。
