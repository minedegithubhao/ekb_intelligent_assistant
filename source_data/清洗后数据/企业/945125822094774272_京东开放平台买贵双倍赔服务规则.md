# 第一章 概述
**第一条** 为保障消费者价格体验，倡导优质、公平的价格服务，制定《京东开放平台“买贵双倍赔”服务规则》。
**第二条** 本规则适用于在京东开放平台JD.COM入驻开店的第三方商家。
**第三条**买贵双倍赔服务指消费者购买标有此服务标识的商品后，若其订单实付金额高于京东或其他特定平台上成交同款商品的实付价格（含运费），消费者可在订单支付后买贵双倍赔服务申请有效期内凭有效凭证申请与订单实付金额双倍差额的补偿。京东将按1：100的比例将补偿折算为京豆，发放至消费者的京东账户中。
（一）比价平台：拼多多、淘宝（不含阿里拍卖、淘工厂、淘宝买菜）、天猫（不含阿里拍卖、淘工厂、淘宝买菜）、京东、抖音、唯品会 (含前述平台的网站、APP、微信公众号、微信商城及小程序等)、途虎养车APP。
注：以上比价平台仅自营/官方旗舰店/旗舰店/专营店/专卖店商品参与比价；带有秒送标识的商品不参与京东站内比价。
（二）同款商品:同款商品是指一定销量（前台展示销量≥30件）且评价数量（前台展示评价≥30条），且商品品牌、型号(包括但不限于、颜色、数量、存储容量、口味、香型、包装、款式、材质、成分、尺寸、尺码、规格、生产日期、赠品、货号、是否升级款)、货源（京东国际商品仅可对比京东国际、天猫国际、拼多多全球购、唯品国际、抖音全球购商品） 等单一商品维度的基本信息均一致的商品，同款商品判定以商品商详展示的商品信息为准。
注：以下情形除外：
①定制商品由于其特殊性，需单独识别。
②来源存在合法合规风险的商品，例如药品价格低于国家或地方药品集中采购价格等。
（三）消费者实付金额：包含使用京豆、E卡、余额支付的订单金额，不包含优惠券等平台普惠促销；
（四）同款商品的价格：消费者成交同款商品的实付金额(含运费)；
（五）买贵双倍赔赔付金额=（消费者订单实付金额-特定平台上消费者购买同款商品时订单实付金额）×2。
# 第二章 服务细则
## 第一节 开通与退出
**第四条** 商家在报名大促活动时，会通过签署协议的方式自动开通买贵双倍赔服务，商家签署该协议后即承诺提供 “买贵双倍赔”服务。大促活动包括但不限于参与超级补贴、超级品牌日、超级品类日、超级新品日等，开通成功的活动商品均适用“买贵双倍赔”服务。
**第五条** 商家可以申请退出“买贵双倍赔”相关活动，包含但不仅限于京东大促、超级补贴等。实时退出或审批通过后退出，将视活动规则而定。若商家成功退出活动，等同于退出买贵双倍赔服务，商品将不展示“买贵双倍赔”标识，京东有权不再支持商品享有“买贵双倍赔”相关活动资源。
**第六条** 经审核若商家退出支持买贵双倍赔的营销活动，退出活动时已产生的交易订单仍需按本规则、协议等向消费者履约。
**第七条** 商家可在商家后台（shop.jd.com（https://shop.jd.com/））“交易—价保管理—买贵双倍赔”查看订单相关信息。
## 第二节 补贴方式
**第八条** 消费者发起“买贵双倍赔”服务申请后，京东有权依据消费者提交的凭证进行差额补贴审核。
**第九条** 审核通过后，官方补贴商品将由京东承担差额赔付，非官方补贴商品将由京东垫付赔付差额后与商家结算。
**第十条** 若商品支持“价格保护”服务，同时满足“买贵双倍赔”服务条件的，消费者仍可发起“买贵双倍赔”服务申请。
**第十一条**商家授权并同意，京东有权在【买贵双倍赔】活动商品订单生成后限制商家对应订单金额的资金提现。当对应订单【买贵双倍赔】服务期满3个自然日或消费者【买贵双倍赔】申请审核结束且已向消费者补偿（如需）后（以较晚时间为准），京东将解除商家对应订单金额的资金提现限制。
# 第三章 商家申诉
**第十二条**若商家对买贵双倍赔订单存在异议，可通过商家后台（shop.jd.com（https://shop.jd.com/））“交易—价保管理—买贵双倍赔—申诉路径“，向京东进行申诉。针对每笔订单商家仅可申诉一次，收到客服一次审核结果后，商家需在24小时内提交相关凭证。
**第十三条** 商家申诉时需注明申诉原因，如多赔、错赔等，及提供清晰、真实的申诉材料：
（一）同品问题: 商家针对消费者所提交的竞品平台同品物品不予认同，商家需提供注明二者不同之处的截图与情况说明；
（二）价格问题：商家对消费者在竞品平台成交同款商品的应付金额不予认同，需商家在同品商品的下单界面的应付金额截图与情况说明。
**第十四条** 商家申诉后将由专属客服进行审核，并于申诉后24小时内反馈。若申诉成功，将由京东承担买贵双倍赔服务成本，商家可对相关订单向京东申请结算。
# 第四章 服务管理
**第十五条**商家在履行“买贵双倍赔”服务期间，需履行京东“买贵双倍赔”服务承诺，遵守法律法规、协议及京东相关规则（https://rule.jd.com/）。
**第十六条**商家不得未经提报自行在商品主图添加“买贵双倍赔”的标识。
**第十七条**商家若存在任何违规行为（包括但不限于提供虚假凭证、进行虚假交易、使用外挂或插件等非正常手段、洗钱、恶意套现刷信誉、活动作弊等），京东将终止其享受“买贵双倍赔”服务的资格并有权撤销违规交易、追回京东损失；同时京东将依照相关规定进行处理并追究商家违约责任及其他相关法律责任。
**第十八条** 如商家参与本活动的商品包含临期商品，则须在商品主图、详情页等显著位置标明为临期商品, 对于临期商品，京东将按下述规则提供用户买贵双倍赔服务：
（一）同款商品与下单商品均为临期商品，支持赔付；
（二）同款商品为临期商品，下单商品为非临期（正常）商品，不支持赔付；
（三）同款商品为非临期（正常）商品，下单商品为临期商品，支持赔付。
# 第五章 附则
**第十九条**除非本规则另有规定，本规则中的用语具有与京东协议、京东规则中相应用语相同的定义。
**第二十条**京东开放平台商家的行为，发生在本规则生效之日以前的，适用当时的规则。发生在本规则生效之日以后的，适用本规则。
**第二十一条**商家应遵守国家法律、行政法规、部门规章等规范性文件。对任何涉嫌违反国家法律、行政法规、部门规章等规范性文件的行为，本规则已有规定的，适用于本规则，本规则尚无规定的，依照法律法规处理。商家依据相关规则承担的相关责任并不免除其应承担的法律责任。商家在京东的任何行为，应同时遵守与京东及其关联公司签订的各项协议。
**第二十二条**本规则于2026年3月13日修订，于2026年3月20日生效。
