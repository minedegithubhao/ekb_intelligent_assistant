# 【修订】京东开放平台发布或推送第三方信息细则

尊敬的商家朋友们：
您好！
为维护京东开放平台商家的合规经营，保护消费者隐私信息，提升消费者购物体验，对《京东开放平台发布或推送第三方信息细则（https://rule.jd.com/rule/ruleDetail.action?ruleId=760077009186590720&btype=1）》中违规定义、处理措施和规则解读进行调整。本次调整于2024年4月19日公示，于2024年4月26日起生效。
| **调整点** |**修订前** |**%修订后** |
| 违规定义 | 1.3其他方式发布或推送第三方信息：通过其他方式诱导消费者跳转至第三方网站或客户端或线下进行交易的行为。 | 1.3其他方式发布或推送第三方信息：商家联系消费者添加京东外联系方式，或引导消费者跳转至第三方网站或客户端或线下进行交易。 |
| 处理措施 | 2.1.2违规程度严重：下架违规商品、每次扣除违约金1000元；
2.1.3违规程度极其严重：限制提报营销活动7日、减少店铺曝光3日、删除违规商品，延长结算21日、咚咚禁言、降低店铺星级至2.5星30天、每次扣除违约金3000元；
2.1.4除上述处理措施外，如商家多次出现违规行为或因上述行为引起重大投诉的，京东有权视违规行为对商家采取包括但不限于店铺屏蔽，全店下架，限制使用咚咚，扣除违约金40000元，清退等处理措施。 | 2.1.2违规程度严重：下架违规商品、每次扣除违约金1000元、加收风险保证金；
2.1.3违规程度极其严重：限制提报营销活动7日、减少店铺曝光3日、删除违规商品，延长结算21日、咚咚禁言、降低店铺星级至2.5星30天、每次扣除违约金10000元、加收风险保证金；
2.1.4除上述处理措施外，如商家多次出现违规行为或因上述行为引起重大投诉的，京东有权视违规行为对商家采取包括但不限于店铺屏蔽，全店下架，限制使用咚咚，扣除违约金100000元，冻结资金，加收风险保证金，清退等处理措施。 |
| 规则解读 | 违规程度极其严重包括但不限于：
（一）店铺多次发生“严重”违规程度的违规；
（二）商家引导消费者到线下或其他平台进行交易等事实发生；
（三）商家刻意研究、规避京东的管控方式，并加以传播、利用；
（四）商家存在其他恶意对抗管控的特殊情节。 | 违规程度极其严重包括但不限于：
（一）店铺多次发生“严重”违规程度的违规；
（二）商家联系消费者添加京东外联系方式，或引导消费者跳转至第三方网站或客户端或线下进行交易；
（三）商家刻意研究、规避京东的管控方式，并加以传播、利用；
（四）商家存在其他恶意对抗管控的特殊情节。 |
