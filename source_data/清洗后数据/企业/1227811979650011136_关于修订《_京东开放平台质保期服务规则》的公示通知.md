# 关于修订《京东开放平台质保期服务规则》的公示通知

- rule_id: 1227811979650011136
- source_url: https://learn-jdm.jd.com/knowledge/rule/detail?ruleId=1227811979650011136
- label_names:
- active_time: 2025-04-17
- update_time: 2025-05-27
尊敬的商家朋友：
您好，为适应业务发展需求，将针对《京东开放平台质保期服务规则》进行调整。现进行规则公示，公示期：2025年4月11日- 2025年4月17日，规则预计于 2025年4月18日生效，公示期满平台将另行发布正式生效规则。
核心调整点：
1、增加质保维修争议场景说明和处理标准：涉及无故拒绝维修场景、超时未维修/维修超时未寄回、维修费用争议、多次维修未解决、维修后争议等场景；
2、平台类目最低质保期时效更新。
修订具体内容如下：
# 第一章 概述
**第一条** 为保障消费者购物及售后体验，帮助商家合规经营，特制定本规则。
**第二条**“平台质保期服务”是指销售商家（以下简称“商家”）的商品出现质量问题，在质保期时效内客户申请维修时，按平台质保期时效为客户提供售后维修服务。
**第三条** 商家在发品时通过【商家后台-商品-功能设置-质保期】模块可设置质保期时效为客户提供质保期服务，如果未设置以国家法律法规和平台规定的“平台质保期服务”为准（两者以最有利于消费者的为准）。如果商家根据厂商政策、商品详情页承诺选择高优的质保期时效等为客户提供服务，以商家提供为准。
**第四条** 本规则适用于入驻京东开放平台JD.COM的商家（小时购、京喜拼拼、个人/个体店除外；非JD.COM平台上的商家除外）。
# 第二章 服务细则
**第五条** 商家需承诺按照不低于相关法律法规要求的水平提供售后维修服务，包括但不限于在无法按照相关规定提供维修服务时，延长退款、退换货、补寄（如有）等售后服务有效期。
**第六条** 商家在发布商品时，在【商家后台-商品-功能设置-质保期】模块可自行选择优于平台最低质保期时效为消费者提供服务承诺。具体见附件一平台类目质保期时效表。
**第七条** 当消费者申请售后维修服务，页面存在多个质保期时效时，以对消费者最有利承诺为准。
# 第三章 类目质保期时效
**第八条**除安装类商品外，质保期时间以自消费者签收商品之日起（自签收次日零时起）计算。安装类商品质保期时间自商品安装完成之日起计算。平台质保期服务时效即商家承诺为消费者提供的最低质保期服务时效，具体见附件一平台类目质保期时效表及附件二安装类目明细表。
# 第四章 违规及争议处理
**第九条**在质保期内，商家应提供但未提供或未在规定时间内提供“平台质保期服务”，或涉及维修争议场景的，京东有权依据法律法规、协议规则等处理、包括但不限于**《京东开放平台质保期服务规则》**（https://rule.jd.com/rule/ruleDetail.action?ruleId=1076394019191394304&type=0&btype=1）**、****《京东开放平台售后服务管理规则》**（http://rule.jd.com/rule/ruleDetail.action?ruleId=2595）、**《京东开放平台商家服务管理规则》**（https://rule.jd.com/rule/ruleDetail.action?ruleId=941277149179023360），同时商家对该商品仍需要向消费者提供质保服务。
**第十条**本规则中未约定的内容，按照法律法规、协议、**《京东开放平台售后服务管理规则》**（http://rule.jd.com/rule/ruleDetail.action?ruleId=2595）**、****《京东开放平台交易纠纷处理总则》**（https://rule.jd.com/rule/ruleDetail.action?ruleId=2392）或相关纠纷处理细则等进行处理。
## 第一节 无故拒绝维修申请
**第十一条**若消费者因商品维修申请售后服务的，商家应告知商品承诺维修时效，并按双方约定为消费者提供免费维修服务。若商家无故拒绝消费者维修申请的，包括
（一）商品未超质保时效，无故拒绝；
（二）无法举证商详页承诺或国家“三包”范围或商品保修卡质保说明，商家主张商品超质保；
（三）主张商品无质量问题，客户已提供商品问题基础凭证，商家未提供有效检测报告（检测报告有效性参照售后质量问题有效性标准）；
（四）系统维护质保时间大于页面说明质保时效，客户发起售后维修后商家以页面描述拒绝维修；
（五）商家让客户自行联系厂家、小程序维修且客户不认可。
商家需履行维修义务，若商家拒绝，支持客户退货退款。
## 第二节 超时未维修/维修超时未寄回
**第十二条**若消费者因商品维修申请售后服务的，商家应告知商品承诺维修时效，并按双方约定为消费者提供免费维修服务。若商家超时未维修或维修超时未寄回，
**第十三条**寄送维修场景下，若
（一）商家与客户未约定维修时效，商家应在商品发出后30天内维修完成并寄回。商家应在收货后7天内告知客户维修结论，若需要维修应在收货后23天内完成维修并寄回；若不需要维修应在7天内告知原返结论并在10天内告知原返物流单号。超时未寄回，支持客户退货退款；
（二）商家与客户约定维修时效，商家应在时效内维修完成并寄回，超时未寄回，支持客户退货退款。
**第十四条**上门检修场景下，若
（三）商家与客户约定上门时效后，商家超时/无故未上门，支持客户退货退款；若超时后客户同意继续等待，自商家违约之日起，每日按订单实付金额的2%赔付客户，直至商家上门检修（最长支持赔付15天）。超15天商家仍未上门检修，支持客户退款，商品需商家自行联系客户取回；若超时后客户要求自行维修，维修费用由商家承担；
（四）商家与客户没有约定上门时效，商家需在收到客户维修申请的23天内完成上门检修，超时未完成，支持客户退款，商品需商家自行联系客户取回；
（五）不支持拆卸的商品商家需支持上门检修，如商家拒绝，支持客户退款，商品需商家自行联系客户取回。
## 第三节 维修费用争议
**第十五条**若消费者因商品维修申请售后服务的，商家应告知商品承诺维修时效，并按双方约定为消费者提供免费维修服务。因维修费用产生争议，如
（一）商家承诺免费安装但未提前告知维修会产生其他费用，如材料费、辅料、拆装费用等（无论维修服务是商家或品牌方还是安装方提供）；
（二）商家收费超行业标准，无法提供费用收取合理的依据；
（三）保内维修商品寄出和返回快递费用均由商家承担。客户寄出商品时支付了快递费用或商家寄回商品时向客户收取了快递费用；
（四）符合保内维修场景但是商家需客户提供维修费用。
若商品已维修，商家需返回对应费用给客户；若商品未维修，支持客户退货退款。
## 第四节 多次维修未解决
**第十六条**若消费者因商品维修申请售后服务的，商家应告知商品承诺维修时效，并按双方约定为消费者提供免费维修服务。若商家在承诺质保期时效内多次维修（≥2次）未解决，
（一）客户接受调换同型号、同规格商品，支持换货处理；
（二）客户拒绝调换同型号、同规格商品，支持客户按照标准折价退款（未使用过的商品支持退货退款）；
（三）商家无同型号同规格商品，客户不愿调换其他型号、规格商品而要求退货的，商家应当予以退货（不涉及货值折价）。
## 第五节 维修后争议
**第十七条**若消费者因商品维修申请售后服务的，商家应告知商品承诺维修时效，并按双方约定为消费者提供免费维修服务。因客户反馈维修后收到商品外观破损//少件/少附件，如
（一）核实返件途中破损，即商家提供寄回时商品未损坏证明，物流原因导致商品破损。商家需重新给客户提供维修服务，如客户要求自行维修并保留维修凭证，商家需提供维修费用补偿；
（二）核实商家维修失误，商家需重新给客户提供维修服务。维修时效同第十三条说明；
（三）核实商家少发附件，商家需在24小时内为客户补发，超时未补发支持客户退货退款。
# 第五章 附则
**第十八条** 京东开放平台商家的行为，发生在本规则生效之日以前的，适用当时的规则。发生在本规则生效之日以后的，适用本规则。
**第十九条** 京东可根据运营情况随时调整本规则并以公告的形式向商家公示。
**第二十条** 商家应遵守国家法律、行政法规、部门规章等规则性文件。对任何涉嫌违反国家法律、行政法规、部门规章等规则性文件的行为，本规则已有约定的，适用于本规则，本规则尚无约定的，依照法律法规处理。商家依据相关规则承担的相关责任并不免除其应承担的法律责任。商家在京东的任何行为，应同时遵守与京东及其关联公司签订的各项协议。商家与京东协议约定的售后义务更有利于消费者时，京东有权适用协议约定。
**第二十一条** 本规则于2025年4月11日修订，于 2025年4月18日生效。
