# 第一章 概述
1.1****目的
为保障京东大闸蟹行业市场健康发展，提升消费者的购买体验，特制定本规范。
1.2****适用范围
适用于经营以下类目的商家：
生鲜-海鲜水产-蟹类-大闸蟹
生鲜-海鲜水产-海鲜卡券-大闸蟹礼券
# 第二章 商品发布
2.1****商品页面规范
2.1.1 经营地理标志商品的商家，需按《京东开放平台商家地理标志类商品管理规则（https://rule.jd.com/rule/ruleDetail.action?ruleId=734358177788858368&type=0&btype=1）》上传地标相关资质证明至品质楼层亮标，同时经营阳澄湖大闸蟹的商家，需满足《京东阳澄湖大闸蟹管控要求》要求（附件一）后，方可在商品标题、主图、属性、商详任意模块涉及地标连词和非连词宣称商品，特殊场景如有证明除外。
生鲜-海鲜水产-蟹类-大闸蟹和生鲜-海鲜水产-海鲜卡券-大闸蟹礼券底层已分别配置单蟹重量（实物）和单蟹重量必填属性，填写要求：公蟹x两-y两 或 母蟹x两-y两 （x,y为数值，可填写小数点），请商家按照填写要求和商品实际重量进行宣传。
2.1.2 经营大闸蟹活蟹的商家，应将商品发布在生鲜-海鲜水产-蟹类-大闸蟹类目下，经营大闸蟹礼券的商家，应将商品发布在生鲜-海鲜水产-海鲜卡券-大闸蟹礼券类目下。
2.1.3 关于商品命名，活蟹按照：【活蟹】品牌名称、只数信息、公母信息、规格信息、其他等格式命名；礼券按照：【礼券】品牌名称、只数信息、公母信息、规格信息、数字型号（如有不能超过4位数）、其他等格式命名；残蟹按照：【活蟹】（残蟹/九指蟹/断钳蟹/单脚蟹）品牌名称、只数信息、公母信息、规格信息、其他等格式命名；网兜蟹按照：【活蟹】（网兜蟹）品牌名称、只数信息、规格信息（总净重）、其他等格式命名。信息顺序不做要求，规格信息鼓励商家标注具体值，如标注区间值，则区间浮动范围必须≤0.3两（礼券商品不得标注区间值）；仅网兜蟹可标注只数区间值，区间浮动范围必须≤5只，可不标注公母信息和单只规格。例如：【活蟹】xx品牌 公蟹xx两/只 母蟹xx两/只 x对 x只；【礼券】xx品牌 公蟹xx两/只 母蟹xx两/只 x对 x只；【活蟹】（残蟹）xx品牌 公蟹xx两/只 母蟹xx两/只 x对 x只；【活蟹】（网兜蟹）xx品牌 x总净重 x-x只。
注：阳澄湖大闸蟹个体规格需达到母蟹≥2.5两，公蟹≥3.0两。
2.1.4 品质服务保障：经营礼券、活蟹、阳澄湖活蟹、阳澄湖礼券的商家需满足京东对应的品质服务保障，如下：
礼券：兑换无忧、30天价保、足斤足两、保鲜活
活蟹：足斤足两、保鲜活、全程冷链、专线专配
品质服务保障展示图片内容，详见附件二。
2.2****礼券发布规范
2.2.1 经营礼券的商家，需签署《京东卡券兑换产品服务协议（https://rule.jd.com/rule/ruleDetail.action?ruleId=858631791009992704&btype=1）》，协议内容包括需接入京东平台大闸蟹卡券管理系统、质保金、资金冻结等要求。商家接入京东平台大闸蟹卡券管理系统生成卡号，并进行卡券印刷与核销、提货数据互通，同时需遵守以下接入标准：商家同步的卡券提货数据应为通过京东平台销售的卡券的兑换数据，兑换数据应为京东平台客户已完成提货的兑换数据（不含预约成功实际未提货的），商家必须保证同步到卡券系统中的所有数据真实有效。
注：接入京东平台大闸蟹卡券管理系统时，后台必须绑定订单与卡券序列号。
2.2.2 经营大闸蟹礼券及活蟹的商家，其资质需符合《京东开放平台生鲜类目资质标准（https://rule.jd.com/rule/ruleDetail.action?ruleId=2516&btype=1）》要求，同时为保证消费者的兑换权益，商家最低入驻时长不得低于三年。
2.2.3 经营大闸蟹礼券的商家，其礼券商品需全部接入兑换无忧服务，详见《京东开放平台兑换无忧服务规则》（https://rule.jd.com/rule/ruleDetail.action?ruleId=587324745217871872）。
2.2.4 经营大闸蟹礼券的商家，礼券版面印制需按照如下板式，并包含以下信息印制礼券前商家需反馈电子版卡面信息至7fresh-qc@jd.com进行审核。

2.3****礼券入库规范
经营大闸蟹礼券且入京东仓库的商家，制作时礼券外包装必须露出条形码及序列号数字，京东物流需直接对券面的条形码扫描，否则无法在京东平台售卖。建议将内卡固定，防止内卡挪动位置，被卡套挡住条形码。

2.4****快递面单展示信息：
京东渠道销售的活蟹及礼券的快递面单需展示必须展示商品编号、商品名称、规格信息、京东渠道销售、活蟹或礼券等信息避免难以区分。
2.5****预售和预订/预定要求
京东渠道销售的活蟹及礼券商品提前销售行为需符合类目要求规定，礼券商品不允许参与任何预售/预订/预定等提前销售性质的活动，活蟹商品不允许参与任何预售活动，仅允许设置不超过7天的预订/预定时效。
# 第三章 商品质量
3.1****商品质量标准
在京东经营的大闸蟹商家均需符合国家食品安全标准及《京东生鲜感官标准-大闸蟹（活体）》标准（附件三），商家应依据此标准提供商品。
3.2****现场监督
在京东经营大闸蟹的商家需接受并配合京东品控及委派的第三方机构对生产车间、发货仓库等地进行的驻场监督、飞行检查、协同仓检查等，监督检查项目包含原料验收、暂养条件、生产环境、成品抽检等环节。现场监督检查细则详见《京东开放平台大闸蟹现场监督检查管理办法（https://rule.jd.com/rule/ruleDetail.action?ruleId=738331391913234432）》。
3.3****抽检方式
售卖过程中，京东品控或委派的第三方机构会进行不定期抽检，食品安全类抽检依据《京东开放平台生鲜类商品品质抽检规范》（https://rule.jd.com/rule/ruleDetail.action?ruleId=3849&type=0&btype=1）。
感官类抽检依据《京东生鲜感官标准-大闸蟹（活体）》标准 （附件三），感官抽检频次依据各SKU周销售数量划分，具体如下表：

3.4****数据预警
京东品控依据客户体验大数据对“死蟹、缺斤短两、兑换难”等问题设定监控值，每周超出监控值的商品、品牌进行品质管控，管控方式不限于预警、增加感官抽检次数、下架、清退等措施，并通过邮件、商家群等渠道对商家进行正式触达。商品如需再次上架，[需提供整改报告至7-fresh-qm@jd.com](mailto:需提供整改报告至7-fresh-qm@jd.com)进行审核，审核通过后重新上架。
# 第四章 商品售后
4.1****赔付标准
在京东经营大闸蟹的商家均按照《京东开放平台生鲜大闸蟹赔付管理规范（https://learn-jdm.jd.com/knowledge/rule/detail?ruleId=1189043937948602368）》进行售后服务。
4.2****危机客诉
出现危机客诉，京东要求商家积极配合解决问题，并针对客诉问题提供整改报告，整改报告反馈至7fresh-qc@jd.com。（涉及以下内容将按照6.10处理）
危机客诉场景：
① 行政投诉类：消费者通过政府工商官方APP、12345热线、12315热线等渠道进行投诉，问题包含商标、商品资质、商品品质、合同、反不正当竞争、广告宣传等。
② 行政举报类：消费者对问题商品进行举报，监管受理案件后立案，进入调查取证阶段。
③ 热点舆情类：热点事件指因某一问题，短时间内引起客户集中咨询或可能引起客户集中咨询、造成社会负面舆论的外部新闻和内部事件；舆情事件指外部曝光涉及或关 联京东事件（如微博、贴吧投诉，网站等新闻媒体曝光）。
4.3****兑换要求
4.3.1 礼券兑换选择发货日期需保证近7天内至少3天可供预约发货，每日开放可预约单量按照下表。京东将以日维度抽检礼券兑换情况，每日通知不合格商家，抽检采用不固定周期，每天进行多次抽检，并追踪落实改善状况。高峰兑换期间（9月25日/开湖时间~10月10日），每日开放可预约单量在常规兑换期间基础上增加至2倍及以上。
图片：https://img30.360buyimg.com/venderrule//jfs/t1/334610/40/16868/51572/68d3b96eF4b5cb559/67aaf277d3f1ebeb.png
4.3.2 在中秋节、国庆节（9月25日/开湖时间~10月10日）等预约高峰期，商家需提前反馈每日最大可预约单量，京东按高峰期间兑换率30%反推销售量，对超出销售量的商家进行限制销售。
注：客户卡券预约的提货日期为订单发货日期。
# 第五章 价格规范
5.1****价****格管控
售卖过程中，京东针对大闸蟹的价格虚高日维度管控。若商品为价格虚高或虚低，即售价与实际进销成本严重不符，与一般同类商品的市场售价相比虚高或虚低，则商家需在预警后4小时内自行修正价格虚高行为或提供商品价格合理性证明。若无法提供且不修正价格的，商品维度价格虚高违规次数进行累计并根据违规次数进行相应违规处理。商品如需再次上架，需提供整改报告至7fresh-qc@jd.com进行审核，审核通过后重新上架。
5.2****影响行业价格公平竞争力
为保证大闸蟹价格公平竞争力，京东针对大闸蟹的价格日维度排查管控，若涉及影响价格公平竞争力的违规行为，则商家需在预警后4小时内修正价格，对于不修正价格的商家，其商品维度违规次数将累计，京东根据违规次数对其进行相应违规处理。商品如需再次上架，需提供整改报告至7fresh-qc@jd.com进行审核，审核通过后重新上架。
# 第六章 违规处理
6.1 经调查“过水蟹”①问题证据确凿且属于商家责任的，将涉事公司法人及主体加入京东商家黑名单系统，永不合作。同时依据平台规则《京东开放平台出售假冒商品细则（https://rule.jd.com/rule/ruleDetail.action?ruleId=3393&type=0&btype=1）》，京东有权执行全店商品下架、永久禁止发布新商品、永久禁止上架待售商品，店铺清退，扣除1000000元或该店铺累计销售金额10倍的违约金（从高），并对涉事商家及店铺进行限权和管控。同时，不基于违规次数，京东有权对商家的关联方或关联店铺采取一项或多项的管控措施，具体管控措施详见《京东开放平台关联店铺管理规则（https://rule.jd.com/rule/ruleDetail.action?ruleId=4680&type=0&btype=1）》。
① 过水蟹，就是将不符合“阳澄湖大闸蟹”农产品地理标志要求的蟹，运送至阳澄湖区域以“阳澄湖大闸蟹”名义销售的假阳澄湖大闸蟹。
6.2 核查阳澄湖大闸蟹资质及和备案信息不符合要求时，如发现商家提交6证资质缺失错误、备案信息未提交等，以及抽检商品不符合要求时，如蟹扣和包装宣称使用不规范（非阳澄湖大闸蟹使用阳澄湖大闸蟹蟹扣或包装印刷阳澄湖字样），京东将扣除违约金10000元/次，并下架所有相关商品并取消阳澄湖大闸蟹售卖资格。
6.3 活蟹商品页面发布任一项不符合规范，如标题宣称不规范、商详误导宣传、类目错挂等情形，要求1个自然日内完成整改，前2次不符合予以警告，累计3次仍未改善的，将对违规商品下架处理。
6.4 礼券商品页面发布任一项不符合规范，如涉及类目错挂、标题宣称不规范、商详误导宣传等情形，一经发现直接对违规商品下架处理；针对礼券实物印刷信息不符合规范及销售非实物卡券行为，京东将以SKU维度扣除商家违约金1000元/次/SKU，如礼券版面信息、数字型号、序列号、版式等情形，同时未上架的不得上架销售，已上架的下架处理，直至修改合格后才可上架销售。
6.5 当活蟹或礼券作为任何赠品形式销售时，其赠品与本规则所有管控要求一致，如赠品涉及违规，将按照对应条例处置。
6.6 大闸蟹安全指标不合格（比如重金属、农药残留超标），将按照《京东开放平台商品质量不合格细则（https://rule.jd.com/rule/ruleDetail.action?ruleId=3848&type=0&btype=1）》中危害人身安全类条款处理，在未造成严重影响的情况下仅对商品下架，如果造成严重影响（比如群发类舆情、政府监管指令等）将根据影响程度实施资金冻结、店铺清退。
6.7 商家通过不正当方式获取虚假的商品销量、店铺评分、商品评论或成交金额等，妨害、干扰消费者购物权益，损害平台正常经营秩序的行为。将依据《京东开放平台虚假交易细则（https://rule.jd.com/rule/ruleDetail.action?ruleId=3400&type=0&btype=1）》，京东有权视违规程度采取包括但不限于扣除违约金、店铺商品搜索降权、推荐降权、删除违规商品销量和评价、限制渠道专属技术服务费、限制商家下单和支付权益、禁止提报营销活动、禁止发布上新商品、禁止商家待售商品、全店下架、终止合作等处理措施。
