# 1.****【违规定义】
## 1.1****危害国家安全、破坏政治稳定与社会稳定的有害信息类**（https://rule.jd.com/rule/ruleDetail.action?ruleId=1241927157672972288&btype=1）**：
1.1.1含有反动、破坏国家统一的商品及信息、破坏主权及领土完整、破坏社会稳定、涉及国家机密、扰乱社会秩序的商品及信息
1.1.2宣扬恐怖主义、宗教极端思想、民族仇恨等的商品及信息
1.1.3发布任何歪曲历史，美化他国侵略史实或宣扬军国主义思想，或伤害中国人民和中华民族感情的商品及信息
1.1.4未在境内出版发行或已停止发行销售的涉政书刊及收藏性的涉密书籍、音像制品、视频、文件资料、游戏以及特殊邮票
1.1.5违规发布国家机关用品或不当营销的商品及信息
1.1.6未经授权的含有国家领导人相关的商品或信息
1.1.7涉国家领导人关联的书籍、培训资料、教材等其他出版物，领导人姓名或职务错写、漏写、遮挡
1.1.8“问题地图”及衍生品
1.1.9违反《互联网宗教服务管理规定》相关管理规定，带有非法宗教、邪教、种族、性别、地域、职业歧视的相关商品或信息
1.1.10为完成党政工作提供违规服务

# 2.****【处理措施】
2.1商家不得通过任何方式规避本规范及其他京东开放平台禁售商品的相关管控措施，商家不得在包括但不限于京东商品页面（包括但不限于商品标题、商品描述、商品图片），店铺页面（包括但不限于店铺命名、店铺分类、店铺介绍、店铺公告、店铺留言），以及在线客服对话页面、社区和论坛等信息发布区块发布禁售商品相关信息；
2.2如果商家利用平台监管漏洞持续销售违法违规商品，或通过模糊商品描述、使用“马赛克”遮挡关键标识、在客服环节引流销售、拆分关键词等方式故意规避平台审核，平台将从严处理，并有权将相关违法违规线索上报主管部门；
2.3商家应遵守国家法律、行政法规、部门规章等规范性文件。对任何涉嫌违反国家法律、行政法规、部门规章等规范性文件的行为，本规则已有约定的，适用于本规则。本规则尚无约定的，京东有权酌情处理。但京东对商家的处理不免除其应承担的法律责任；
2.4如商家违反法律法规、监管政策或本规范的，京东有权依据法律法规、协议、《京东开放平台商家违规管理总则》以及本规范等对于该违规行为进行包括但不限于下架商品、扣除违约金、违规商品下架或修改（商品信息修改）、限制发布商品，店铺屏蔽、限制提报及投放营销活动、全店下架、清退等处理措施。

# 附则：
1.商家的行为，发生在本规则生效之日以前的，适用当时的规则。发生在本规则生效之日以后的，适用本规则；
2.京东可根据运营情况随时调整本规则并以公告的形式向商家公示；
3.本规则于2026年5月27日修订，于2026年6月3日生效。
