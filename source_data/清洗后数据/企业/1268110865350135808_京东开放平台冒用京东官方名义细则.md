# 京东开放平台冒用京东官方名义细则

- rule_id: 1268110865350135808
- source_url: https://learn-jdm.jd.com/knowledge/rule/detail?ruleId=1268110865350135808

- active_time: 2026-04-07

# 1.****【违规定义】
1.1冒用京东官方名义：指商家推广商品过程中，未经平台同意，直接使用与京东平台相关的logo、图案、名称、活动、排行榜等信息，或暗示商品或商家与京东平台存在品质、服务、供应等关联关系，以此来诱导或欺骗消费者进行点击或购买的行为。
# 2.****【处理措施】
2.1违规程度轻微：警告；
2.2违规程度一般：预警，超3天未整改下架违规商品；
2.3违规程度严重：下架违规商品、扣除违约金500元（M2档）；
2.4违规程度极其严重：下架违规商品、扣除违规商品近14天成交金额10%，最低100元，最高10,000元违约金（M3档）；
2.5除上述处理措施外，如商家多次出现违规行为或因上述行为引起重大投诉的，京东有权视违规行为对商家采取包括但不限于违规商品下架或修改（商品信息修改）、店铺屏蔽、全店下架、限制使用咚咚、扣除违约金、冻结资金、加收风险保证金、清退等处理措施。
# 3.****【规则解读】

本细则所述违规行为，根据其严重程度，适用的违约金档位范围为M3-M5档，具体如下：
1）非京东自营商家宣传“京东自营”“自营”等信息；
2）未经授权使用京东素材或卡通形象；
3）商家在商品信息中宣传“京东慧采”等信息；
4）使用“京直供”“京专供”“京品”“京选”等表述暗示商品与京东平台存在关联关系；
5）使用“自管”“白营”等表述，误导消费者为自营商品；
6）宣传不实的京东榜单信息，包括但不限于：
• 非京东榜单商品，宣传京东榜单在榜；
• 杜撰虚假榜单/使用三方榜单，冒充京东榜单；
• 榜单信息超过有效期限仍然使用。
# 4.****【附则】
4.1商家的行为，发生在本规则生效之日以前的，适用当时的规则。发生在本规则生效之日以后的，适用本规则。
4.2京东可根据运营情况随时调整本管理规则并以公告的形式向商家公示。
4.3商家应遵守国家法律、行政法规、部门规章等规范性文件。对任何涉嫌违反国家法律、行政法规、部门规章等规范性文件的行为，本规则已有约定的，适用于本规则。本规则尚无约定的，京东有权酌情处理。但京东对商家的处理不免除其应承担的法律责任。商家在京东的任何行为，应同时遵守与京东及其关联公司签订的各项协议；若商家服务协议约定的违约金金额高于规则约定的，京东有权适用协议约定。
4.4本规则于2026年4月1日发布，于2026年4月8日生效。
