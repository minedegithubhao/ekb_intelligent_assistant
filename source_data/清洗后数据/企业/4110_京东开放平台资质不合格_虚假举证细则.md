# **1.****【违规定义】**资质不合格/虚假举证包含以下情形
1.1提供虚假材料或信息：指商家向京东提供虚假资料或信息，如提供伪造、变造的商标、专利证书、质检报告或其他资质文件，在报备、申诉、不合理评价投诉环节提供虚假材料，误导审核人员等。
1.2不配合提交材料或材料不合格：指商家不配合提交材料或提交的材料不符合京东要求。
1.3不具备该类目经营权：指商家资质不具备该类目经营权，发布超范围经营的商品。
1.4伪造或提供与事实不符的申诉证据：指商家在使用平台申诉工具等时，故意伪造、盗用或其他方式向平台、消费者或第三方提供与事实不符的交易快照、聊天记录、物流凭证等申诉材料，误导审核人员判断等。
# 2.****【处理措施】
2.1****提供虚假材料或信息
2.1.1违规程度轻微：警告，停用相关品牌或类目。
2.1.2违规程度一般：扣除商家近14天成交金额10%，最低100元，最高10,000元（M3档），停用相关品牌或类目的处理。
2.1.3违规程度严重：扣除商家近14天成交金额20%，最低300元，最高100,000元（M4档），停用相关品牌或类目的处理。
2.1.4违规程度极其严重：扣除商家近30天成交金额10倍， 最低30,000元（M6档），并执行店铺清退。
2.2****不配合提交材料或材料不合格
2.2.1违规程度轻微：延迟结算、限制提报营销活动、搜索降权、推荐降权、下架商品、禁止商品上架、限制创建商品、限制上架待售商品；
2.2.2违规程度严重：延迟结算、限制提报营销活动、搜索降权、推荐降权、下架商品、禁止商品上架、限制创建商品、限制上架待售商品、店铺屏蔽、加收风险保证金；
2.2.3违规程度极其严重：执行店铺清退。
2.3****不具备该类目经营权
2.3.1违规程度轻微：扣除违约金6000元，全店商品下架，永久限制通过接口批量操作商品，限制创建商品14日，禁止上架待售商品14日，店铺搜索推荐降权25%（90日），店铺在售商品阈值上限降低为200个SPU（30日）；
2.3.2违规程度严重：扣除违约金20000元，全店商品下架，永久限制通过接口批量操作商品，限制创建商品，禁止上架待售商品30日，限制提报营销活动30日，店铺搜索推荐降权75%（90日），在售商品阈值下降为50个SPU（90日）；
2.3.3违规程度极其严重：扣除违约金40000元，并执行店铺清退。
2.4****评价投诉虚假举证
2.4.1情节轻微：扣除违约金50元（M1档）
2.4.2情节一般：扣除违约金800元（M2档）扣除违约金1000元；
2.4.3情节严重：扣除违规商家近14天成交金额10%，最低100元，最高10,000元（M3档），限制评价投诉权限90天。
2.5****伪造或提供与事实不符的申诉证据
2.5.1情节轻微的，限制使用申诉工具48小时并警告；
2.5.2情节一般的，扣除违约金500元；
2.5.3情节严重的，扣除违约金1000元；
2.5.4情节极其严重的，扣除违约金3000元。
2.6****除上述情形外，京东有权视违规情形，对关联店铺采取一项或多项的管控措施。
# 【规则解读】
一、资质/证据不合格包括哪些情形？
商家资质不合格包括但不限于以下情形：
1、未按京东要求时间更新、提交符合京东要求的最新资质或信息的；
2、在风险调查中不提交材料或提交的材料不符合平台要求的；
3、商家向京东提供虚假资料或信息；
4、因商家行为引发行政机关调查或法律诉讼的，若商家存在提供的联系方式无效，或怠于处置、消极应对、拒不配合提交相关资料等情形；
5、商家未提交符合京东要求的商品信息，包括但不限于商品的标识标志信息（如产品名称、生产厂厂名、厂址、产品质量检验合格证明、生产日期和安全使用期等）。
6、未更新或变更证照信息。未更新店铺证照信息，指商家未在其营业执照信息或其店铺经营业务有关的行政许可信息变更完成之日起十个工作日内完成更新店铺证照信息的行为；未变更证照信息，指商家实际经营信息发生变化但未变更其营业执照信息及其店铺经营业务有关的行政许可信息登记事项的行为。
二、什么是“风险调查”？“风险调查”过程中要求商家提供的资质有哪些？
京东将根据系统识别、媒体报道、用户投诉及商家举报等多维度对违规店铺进行判定，当识别到商家店铺疑似存在违规行为时，京东有权发起风险调查，要求商家提供以下材料，包括但不限于：包裹出库打包图片、包裹丢件证明、不可抗力等因素证明、仓储照片及合同证明、订单系统物流信息、法人辅助证明文件、运单承接方官方证明、进货凭证、快递公司合作证明、商标注册证/商标注册申请受理通知书、商品辅助证明、手持身份证照片、完税凭证、完整快递包裹面单、委托加工证明、消费者已签收的凭证、销售授权书、银行对账单、质检/检疫/检验报告等，如有特殊经营模式，商家应提供相关资质凭证及相关合作协议等。
三、商家向京东提供虚假资料或信息中违规程度严重或极其严重是指什么？
商家向平台京东提供虚假凭证致使消费者受到损失或者引发相关风险的；出现负面监管风险、舆情事件的或者提供虚假凭证致使平台受到损失或者引发相关风险的，都属于违规程度严重。违规程度严重包括但不限于如下情形：
1、招商入驻及经营过程中提供虚假资质证明：
（一）企业资质造假（如：企业营业资质造假、一般纳税人资质造假）；
（二）品牌资质造假（如：品牌授权书造假、品牌商标注册证造假）；
（三）特殊行业经营许可资质造假（如：食品经营许可证造假、图书经营许可证造假）；
（四）其他商品资质造假。
2、商家在“出售假冒商品”的申诉过程中，提供虚假凭证的；
3、商家在“不当使用他人权利”的申诉过程中，提供虚假凭证的。
商家出现两次及以上严重违规或产生严重损害消费者体验重大舆情的，属于违规程度极其严重。
四、商家资质不具备特定行业资质的类目经营权，发布超范围经营的商品指什么？
国家明确有资质要求或平台有特定要求的行业，商家未取得行业经营许可（如商家无药品、食品、医疗器械商品、成人用品等的经营资质），且将商品错放在非对应类目的（比如，血压计挂在运动户外类目）。
五、“未按京东要求时间更新、提交符合京东要求的最新资质”指的是什么？
包括但不限于市场监管信息中商家出现了经营异常的情况；商家实际经营信息发生变化但未变更其营业执照登记；商家在其营业执照信息变更完成后未在平台及时更新店铺营业执照信息；市场主体登记信息、与经营业务有关的行政许可信息发生变更而未在平台及时更新的行为等。因商家资质信息未及时、完整更新而造成商家不符合京东平台入驻、经营资质要求的，京东有权根据法律法规、协议、规则等对商家进行处理。
六、商家伪造或提供与事实不符的申诉证据包括什么？
1、 伪造订单信息、交易快照或物流凭证等，包括不限于商家自己下单伪造用户订单，或者商家自行修改收货地址等伪造用户特殊标记等；
2、伪造聊天记录，包括不限于提供咚咚、电话等聊天记录非用户本人沟通记录；
3、提供虚假视频素材等，包括但不限于视频材料存在剪辑拼凑或者多个用户使用相同举证材料等非用户真实恶意行为依据。
4、其他误导平台、用户、其他第三方虚假信息等。
除上述情形外，其他商家不当使用工具，包括但不限于提供虚假问题描述或证明材料误导审核人员判断，造成负面监管、舆情影响时，都属于违规程度严重或者极其严重的伪造申诉证据。

## 附则：
1.商家的行为，发生在本规则生效之日以前的，适用当时的规则。发生在本规则生效之日以后的，适用本规则。
2.京东可根据运营情况随时调整本规则并以公告的形式向商家公示。
3.商家应遵守国家法律、行政法规、部门规章等规范性文件。对任何涉嫌违反国家法律、行政法规、部门规章等规范性文件的行为，本规则已有约定的，适用于本规则。本规则尚无约定的，京东有权酌情处理。但京东对商家的处理不免除其应承担的法律责任。商家在京东的任何行为，应同时遵守与京东及其关联公司签订的各项协议；若商家服务协议约定的违约金金额高于规则约定的，京东有权适用协议约定。
4.本规则于2026年3月30日修订，于2026年4月8日生效。
