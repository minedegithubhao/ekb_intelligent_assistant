# 第一章** **概述
**第一条**为促进京东开放平台商家清晰了解交易订单的结算周期，特制定并公示本规则。
**第二条**本规则适用于入驻京东开放平台的所有商家。
# 第二章** **结算周期说明
**第三条** 若店铺处于0元试运营阶段，结算周期为订单完成后第15日进行结算。
**第四条** 若店铺处于正式运营阶段，结算周期如下：
- 店铺类型为“自然人/个体工商户”的商家，结算周期为订单完成后第7日进行结算。当店铺星级≥4.7星，且无欠款和违规行为时，结算周期缩短至订单完成后第3日进行结算。该结算周期每月调整一次，调整时间为每月最后一个工作日。
- 店铺类型为“普通企业店”的商家，结算周期为订单完成后第7日进行结算。
- 店铺类型为“企业店”的商家，结算周期为订单完成后次日进行结算。
**第五条** 如某店铺被识别为风险商家可能会单独调整为延迟结算，具体结算时间以账单查询页面显示为准。
**第六条** 如商家存在售后服务单且被识别为存在一定风险，则该订单的货款会结算暂停，待售后服务单完成后进行结算。
**第七条** 遇到超过3个自然日的法定假日，如五一、十一、春节等假期，结算时间会延后至假期结束后首个工作日进行结算，具体时间以平台公告为准。
**第八条** 如特殊业务有特定的结算周期，则以其相应的规则/公告展示为准。
**第九条** 京东开发平台可根据运营情况随时调整本规则并向商家公示。
**第十条**本规则于2025年2月20日首次发布，2025年2月27日起正式生效。
