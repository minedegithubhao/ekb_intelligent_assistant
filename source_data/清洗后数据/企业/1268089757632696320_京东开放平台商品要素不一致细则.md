# 京东开放平台商品要素不一致细则

# 1.****【违规定义】
1.1属性不一致：指商品信息中出现的属性词，填写错误或在主图、商详、标题、销售属性、营销属性等位置描述不一致的情形。
1.2销量不一致：指商品信息如标题、卖点、商详等位置的销量信息，与实际销量不一致的情形。
1.3店铺信息不一致：指商品主图中出现的店铺名称信息，与当前商品所属店铺名称不一致的情形。
1.4其他商品要素不一致：指商品信息中除上述以外的其他要素不一致的情形。
# 2.****【处理措施】
2.1 属性不一致
2.1.1违规程度轻微：违规商品预警，需在提示整改期内完成整改；
2.1.2违规程度一般：违规商品下架，扣除违约金500元（M2档）
2.1.3违规程度严重：违规商品下架，扣除违约金500元扣除违规商品近14天成交金额10%，最低100元，最高100,000元违约金（M3档）
2.1.4违规程度极其严重：违规商品下架，扣除违规商品近14天成交金额20%，最低100元，最高100,000元违约金（M4档）；
2.1.5除上述处理措施外，如商家多次出现违规行为或因上述行为引起重大投诉的，京东有权视违规行为对商家采取包括但不限于违规商品下架或修改（商品信息修改）、店铺屏蔽、全店下架、限制使用咚咚、扣除违约金、冻结资金、加收风险保证金、清退等处理措施。
2.2销量不一致/店铺信息不一致/其他商品要素不一致
2.2.1违规程度轻微：违规商品预警，需在提示整改期内完成整改；
2.2.2违规程度一般：违规商品下架，扣除违约金50元（M1档）
2.2.3违规程度严重：违规商品下架，扣除违约金500元（M2档）
2.2.4违规程度极其严重：违规商品下架，扣除违约金800元（M2档）
2.2.5除上述处理措施外，如商家多次出现违规行为或因上述行为引起重大投诉的，京东有权视违规行为对商家采取包括但不限于违规商品下架或修改（商品信息修改）、店铺屏蔽、全店下架、限制使用咚咚、扣除违约金、冻结资金、加收风险保证金、清退等处理措施

# 4.****【附则】
4.1商家的行为，发生在本规则生效之日以前的，适用当时的规则。发生在本规则生效之日以后的，适用本规则。
4.2京东可根据运营情况随时调整本管理规则并以公告的形式向商家公示。
4.3商家应遵守国家法律、行政法规、部门规章等规范性文件。对任何涉嫌违反国家法律、行政法规、部门规章等规范性文件的行为，本规则已有约定的，适用于本规则。本规则尚无约定的，京东有权酌情处理。但京东对商家的处理不免除其应承担的法律责任。商家在京东的任何行为，应同时遵守与京东及其关联公司签订的各项协议；若商家服务协议约定的违约金金额高于规则约定的，京东有权适用协议约定。
4.4本规则于2026年4月9日修订，于2026年4月24日生效。
