# 京东开放平台不当使用关键词细则

# 1.****【违规定义】
1.1关键词混淆：指商家在商品信息（标题、详情页、广告等）中，故意使用与商品无真实关联的热门关键词，通过误导性表述吸引流量或提升转化，但实际商品不具备关键词所描述的特征、授权或功能，属于违规营销行为。
1.2品牌信息不一致：指在标题、宣传语、商品图片及商品详情等位置使用/滥用/堆砌其他品牌词（包括但不限于品牌的变形词或衍生词，如错别字、同音字、故意增加间隔符号等）来描述商品，与商品本身品牌不一致的行为。
# 2.****【处理措施】
## 2.1关键词混淆
2.1.1违规程度轻微：警告；
2.1.2违规程度一般：预警整改，整改失败/超期未整改下架违规商品；
2.1.3违规程度严重：下架违规商品、扣除违约金500元；
2.1.4违规程度极其严重：下架违规商品、扣除违规商品近14天成交金额10%，最低100元，最高10,000元违约金（M3档）；
2.1.5除上述处理措施外，如商家多次出现违规行为或因上述行为引起重大投诉的，京东有权视违规行为对商家采取包括但不限于违规商品下架或修改（商品信息修改）、店铺屏蔽、全店下架、限制使用咚咚、扣除违约金、冻结资金、加收风险保证金、清退等处理措施。
## 2.2品牌信息不一致
2.2.1违规程度轻微：警告，违规商品屏蔽；
2.2.2违规程度一般：违规商品屏蔽、钱包资金冻结、支付近14日全店商品销售额的10%作为违约金，最低100元，最高10,000元（M3档） （实际金额包括但不限于与商家经营收入、违规程度、违规影响等相关）；
2.2.3违规程度严重：店铺屏蔽、钱包资金冻结、支付近14日全店商品销售额的20%作为违约金，最低300元，最高100,000元（M4档） （实际金额包括但不限于与商家经营收入、违规程度、违规影响等相关）；
2.2.4违规程度特别严重：店铺屏蔽、钱包资金冻结、支付近14日全店商品销售额的30%作为违约金，最低500元，最高500,000元（M5档） （实际金额包括但不限于与商家经营收入、违规程度、违规影响等相关）；
2.2.5除上述处理措施外，如商家多次（两次及以上）或大量在售商品出现以上违规行为，或因上述行为引起重大投诉或舆情的，京东有权视违规行为对商家采取包括但不限于违规商品下架、限制创建商品、店铺商品搜索/推荐降权、禁止上架待售商品、永久限制通过接口批量操作商品、限制在售商品阈值、限制提报营销活动、属性屏蔽、店铺屏蔽、商品屏蔽、品牌停用、类目停用、冻结资金、页面止付、降低店铺星级、跨类目经营限制、店铺清退等处理措施。
# 3.****【规则解读】
3.1关键词混淆包括但不限于以下情形：
1）非旗舰店商家宣传“旗舰店”等信息；
2）商家在宣传中强行关联与商品无实际联系的知名IP、卖场、明星或影视作品，以不当获取流量。
3.2品牌信息不一致包括但不限于以下情形：
1）挂靠品牌A，但在标题、主图等位置宣传除A以外其他品牌的相关信息；
2）未挂靠品牌，但在商品信息里宣传与商品本身无关品牌信息。

# 4.****【附则】
4.1商家的行为，发生在本规则生效之日以前的，适用当时的规则。发生在本规则生效之日以后的，适用本规则。
4.2京东可根据运营情况随时调整本管理规则并以公告的形式向商家公示。
4.3商家应遵守国家法律、行政法规、部门规章等规范性文件。对任何涉嫌违反国家法律、行政法规、部门规章等规范性文件的行为，本规则已有约定的，适用于本规则。本规则尚无约定的，京东有权酌情处理。但京东对商家的处理不免除其应承担的法律责任。商家在京东的任何行为，应同时遵守与京东及其关联公司签订的各项协议；若商家服务协议约定的违约金金额高于规则约定的，京东有权适用协议约定。
4.4本规则于2026年3月31日修订，于2026年4月8日生效。
