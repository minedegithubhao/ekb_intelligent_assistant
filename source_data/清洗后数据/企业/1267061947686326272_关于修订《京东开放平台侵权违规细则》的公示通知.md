# 关于修订《京东开放平台侵权违规细则》的公示通知

尊敬的商家朋友：
您好，为适应业务发展需求，将修订《京东开放平台侵权违规细则》，现进行规则公示，公示期：2025年7月28日- 2025年8月3日，规则预计于2025年8月4日生效，公示期满平台将另行发布正式生效规则。
核心调整点：
①调整违规定义中"不当使用他人商标权/著作权/专利权的行为"的定义；
②调整违规处理措施及违约金额。
原规则链接：https://rule.jd.com/rule/ruleDetail.action?ruleId=3407&type=0&btype=1（https://rule.jd.com/rule/ruleDetail.action?ruleId=3407&type=0&btype=1）
修订后正文如下：
1.【违规定义】商家发布的商品或信息涉嫌不当使用他人商标权、著作权、专利权、肖像权以及其他类型权利，或者商家发布、传播或授意他人发布、传播不利于京东的网络信息、音视频等的行为。
1.1 不当使用他人商标权/著作权/专利权的行为：指包括不限于商家所出售的商品或发布的商品信息中、所使用的店铺名、域名中，存在不当使用他人商标权、著作权、专利权等权利的行为；或相关行为产生消费者投诉举报、负面舆情等不良后果。
1.2 不当使用他人肖像权：指商家所发布的商品信息或出售的商品不当使用他人肖像权的行为。
1.3 知识产权投诉：指商家收到权利人发起的知识产权投诉，京东受理权利人所投诉的不当使用他人权利的行为。
1.4 发布影响京东名誉的信息：指商家发布、传播或授意他人发布、传播不利于京东的网络信息、音视频等的行为。
1.5 其他不当使用他人权利的行为：指商家其他不当使用他人权利的行为。
2.【处理措施】
2.1 违规程度轻微的，下架或删除违规商品，并限制创建商品及禁止上架待售商品7日。
2.2 违规程度一般的，下架或删除违规商品，并限制创建商品及禁止上架待售商品7日，扣除违约金 100元-1000元不等。
2.3 违规程度严重的，下架或删除违规商品，并限制创建商品及禁止上架待售商品（7日），扣除违约金2000元-10000元不等。
2.4 违规程度极其严重的，下架或删除违规商品，并限制创建商品及禁止上架待售商品（30日），扣除违约金5000元-100000元不等。
2.5 商家发布、传播或授意他人发布、传播不利于京东的网络信息、音视频等的，京东有权扣收不低于人民币1000000元的违约金；不足以弥补京东损失的部分，商家应继续予以赔偿。
2.6 除上述措施外，京东有权视违规行为对商家采取包括但不限于扣除违约金、违规商品下架或修改（商品信息修改）、禁止发布新商品、禁止上架待售商品、禁止投放或提报活动、禁止投放或播放广告、品牌停用和限制申请品牌、类目停用和限制开通类目、搜索降权、推荐降权、全店商品下架、降低店铺星级、屏蔽降权、账户止付、定额冻结、提高涉事店铺应交保证金金额、延长结算周期、执行店铺清退等处理措施，并且有权视商家违规影响，扣除商家部分或全部违规经营所得（订单款项），并对商家予以追责。
