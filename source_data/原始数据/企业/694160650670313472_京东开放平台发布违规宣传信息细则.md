# 京东开放平台发布违规宣传信息细则

# **1.****【违规定义】**
1.1夸大或虚构功能功效：指商家在商品宣传中声称该商品具备某种特殊功效或特性，但实际商品并未包含相关功能、功效、资质或证明。或商家在商品描述、广告、营销活动或其他渠道中对商品的性能、效果或特点进行不实或过度夸张的宣传，导致消费者形成错误的预期，认为商品能够实现超出实际能力的功能或效果。
1.2使用广告法禁用词汇：指商家在商品的主图、标题、属性、商详等关键展示位置，使用被明确禁止的词汇或表述方式进行商品描述。
1.3虚假营销活动：指商家在商品宣传、广告或其他营销渠道中使用虚假、误导、夸大或过期的活动信息来吸引消费者参与营销活动，而实际情况与宣传内容不符，影响公平竞争和交易环境的行为。
1.4虚假营销服务：指虚构根本不存在的服务或通过虚假、误导性、夸大或隐瞒关键信息的方式推广服务或在商品或服务的宣传中做出不切实际、无法实现或违背客观常识的服务承诺，误导消费者对商品或服务的期望的行为。
1.5AI假图：禁止利用AI等技术合成方式，呈现显著失真的与实际不符的内容，侵害消费者知情权、误导消费者。
1.6其他不符合实际情况的违规宣传：除上述情况外的其他不符合实际情况的违规宣传。
# **2.****【处理措施】**
2.1违规程度轻微：预警整改，整改失败/超期未整改下架违规商品；
2.2违规程度一般： 违规商品下架，扣除违约金200元（M2档）；
2.3违规程度严重：违规商品下架，扣除违规商品近14天成交金额10%，最低100元，最高10,000元（M3档）；
2.4违规程度极其严重：违规商品下架，扣除违规商品近14天成交金额20%，最低300元，最高100,000元（M4档）；
2.5除上述处理措施外，如商家多次出现违规行为或因上述行为引起重大投诉的，京东有权视违规行为对商家采取包括但不限于违规商品下架或修改（商品信息修改）、店铺屏蔽、全店下架、限制使用咚咚、扣除违约金、冻结资金、加收风险保证金、清退等处理措施。

本细则所述违规行为，根据其严重程度，适用的违约金档位范围为M2-M4档，具体如下：
1）夸大或虚构功能功效违规情形包括但不限于以下情形：
- 化妆品宣传具备医疗功效；
- 普通化妆品宣传具有特殊化妆品功效；
- 食品或保健品宣传具有特定的健康效益。
- 保健品商家宣称可以治愈某种疾病，实际没有经过严格的临床试验或获得相关医疗机构的认可；
- 药品、医疗器械商家，宣传的功能功效超出商品说明书、备案或注册证的功能功效。
更多案例可参考[发布违规宣传信息案例](https://learn-jdm.jd.com/knowledge/rule/detail?ruleId=1266955896873947136)
2）使用广告法禁用词汇违规情形包括但不限于以下情形：
- 绝对化词汇：如“顶级”“最高级”“国家级”等；
3）虚假营销活动违规情形包括但不限于以下情形：
- 商家在商品信息中使用“国家补贴”、“政府补贴”、“百亿补贴”等字样，明示或暗示商品享有补贴，但实际上未获得相关补贴或政策支持；
- 商家在商品信息中使用“满xxx元-xx元”等字样，宣传满减优惠活动，但实际上没有相应的折扣或优惠；
- 商家参加的活动结束后，未及时调整店铺或商品中的活动信息，使用已结束活动信息；
4）虚假营销服务违规情形包括但不限于以下情形：
- 发布夸大且无法验证的服务承诺，如终身免费维修、无效免费送等相关描述；
5）AI假图违规情形包括但不限于以下情形：
- 利用AI等技术生成虚假商品展示，夸大不实的功能或效果展示等；
- 利用AI等技术过度美化导致图片、视频等，导致内容效果失真、内容变形等；
- 利用AI等技术生成违背常识、虚构编造、不符合现实或不存在的商品等。
6）其他不符合实际情况的违规宣传违规情形包括但不限于以下情形：
- 过度美化商品不符合客观物理事实，使消费者对商品的真实状态产生错误的认知和期望；
- 伪造授权证书、科研成果、排名信息等，诱导或欺骗消费者进行点击或购买的行为。
# **4.【附则】**
4.1商家的行为，发生在本规则生效之日以前的，适用当时的规则。发生在本规则生效之日以后的，适用本规则。
4.2京东可根据运营情况随时调整本管理规则并以公告的形式向商家公示。
4.3商家应遵守国家法律、行政法规、部门规章等规范性文件。对任何涉嫌违反国家法律、行政法规、部门规章等规范性文件的行为，本规则已有约定的，适用于本规则。本规则尚无约定的，京东有权酌情处理。但京东对商家的处理不免除其应承担的法律责任。商家在京东的任何行为，应同时遵守与京东及其关联公司签订的各项协议；若商家服务协议约定的违约金金额高于规则约定的，京东有权适用协议约定。
4.4本规则于2026年4月1日修订，于2026年4月8日生效。
