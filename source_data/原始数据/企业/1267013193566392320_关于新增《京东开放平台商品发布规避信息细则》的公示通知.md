# 关于新增《京东开放平台商品发布规避信息细则》的公示通知

- active_time: 2025-08-03

尊敬的商家朋友：
您好，为适应业务发展需求，将新增《京东开放平台商品发布规避信息细则》，现进行规则公示，公示期：2025年7月28日- 2025年8月3日，规则预计于2025年8月4日生效，公示期满平台将另行发布正式生效规则。
**核心要点：**
①违规定义：包括在已有商品链接上更换另一款商品、利用视觉效果隐藏关键信息导致消费者误解；
②违规措施：根据不同违规程度设置违规处理措施。
③规则解读：对常见违规场景举例，商家应重点关注避免出现类似违规。
**拟新增规则全文如下：**
**1.****【违规定义】**
1.1更换商品：指商家在已有商品（SKU）链接上将原商品更换为另一款商品进行销售的行为，更换后的商品在类目、品牌、颜色、型号、品种等属性上发生了改变。
1.2图片利用视觉效果隐藏关键信息导致用户误解：指商家通过修改字体/图案大小、色彩等，导致从视觉上完全改变整体含义并产生消费者误解的行为。
**2.****【处理措施】**
**2.1****更换商品：**
2.1.1违规程度一般：扣除违约金500元；
2.1.2违规程度严重：扣除违约金2000元，违规商品下架；
2.1.3违规程度极其严重：扣除违约金5000元，违规商品下架；
2.1.4除上述处理措施外，如商家多次出现违规行为或因上述行为引起重大投诉的，京东有权视违规行为对商家采取包括但不限于违规商品下架或修改（商品信息修改）、店铺屏蔽、全店下架、限制使用咚咚、扣除违约金、冻结资金、加收风险保证金、清退等处理措施。
**2.2****图片利用视觉效果隐藏关键信息导致用户误解：**
2.2.1违规程度一般：警告；
2.2.2违规程度严重：预警整改，超期未整改或整改失败下架违规商品；
2.2.3违规程度极其严重：下架违规商品、并扣除违约金最低500元；
2.2.4除上述处理措施外，如商家多次出现违规行为或因上述行为引起重大投诉的，京东有权视违规行为对商家采取包括但不限于违规商品下架或修改（商品信息修改）、店铺屏蔽、全店下架、限制使用咚咚、扣除违约金、冻结资金、加收风险保证金、清退等处理措施。
**3.****【规则解读】**
3.1商家更换商品包括但不限于以下情形：
1）将虚拟链接（如赠品链接、补差价链接）更换成实体商品
2）将商品更换为不同颜色的同款商品，比如白色裤子变成黑色裤子
3）将商品更换为不同型号的商品
4）将商品更换为不同类目的商品，比如将手机壳更换为手机
5）将单一商品更换为同类商品组合装，比如将洗发水更换为洗发水+护发素的组合装
3.2图片利用视觉效果隐藏关键信息导致用户误解包括但不限于以下情形：
1）图片宣传“上门安装”（48px加粗醒目字体），但“有偿”以极小字号（8px）隐藏于角落；
2）红色背景图片突出“免费”（白色大字），而“首月”用暗红色（与背景色差≤3）近乎隐形。
**4.****【附则】**
4.1商家的行为，发生在本规则生效之日以前的，适用当时的规则。发生在本规则生效之日以后的，适用本规则。
4.2京东可根据运营情况随时调整本管理规则并以公告的形式向商家公示。
4.3商家应遵守国家法律、行政法规、部门规章等规范性文件。对任何涉嫌违反国家法律、行政法规、部门规章等规范性文件的行为，本规则已有约定的，适用于本规则。本规则尚无约定的，京东有权酌情处理。但京东对商家的处理不免除其应承担的法律责任。商家在京东的任何行为，应同时遵守与京东及其关联公司签订的各项协议；若商家服务协议约定的违约金金额高于规则约定的，京东有权适用协议约定。
4.4本规则于2025年7月28日发布，于2025年8月4日生效。
