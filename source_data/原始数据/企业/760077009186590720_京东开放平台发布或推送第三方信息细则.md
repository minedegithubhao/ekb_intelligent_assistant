# 京东开放平台发布或推送第三方信息细则

# **1.****【违规定义】**
**1.1****推送第三方信息**：商家通过咚咚、短信等渠道推送第三方信息（非平台购物链接、非咚咚联系方式、银行账号及其他付款方式、二维码及其他非平台标识等）。
**1.2****页面发布第三方信息**：商家通过京东平台页面等渠道发布第三方信息（非平台购物链接、非咚咚联系方式、银行账号及其他付款方式、二维码及其他非平台标识等）。
**1.3****其他方式发布或推送第三方信息**：商家联系消费者添加京东外联系方式，或引导消费者跳转至第三方网站或客户端或线下进行交易。
# **2.****【处理措施】**
**2.1****推送第三方信息：**
2.1.1违规程度一般：下架违规商品，每次扣除违约金500元；
2.1.2违规程度严重：下架违规商品、每次扣除违约金1000元、加收风险保证金；
2.1.3违规程度极其严重：限制提报营销活动7日、减少店铺曝光3日、删除违规商品，延长结算21日、咚咚禁言、降低店铺星级至2.5星30天、每次扣除违约金10000元、加收风险保证金；
2.1.4除上述处理措施外，如商家多次出现违规行为或因上述行为引起重大投诉的，京东有权视违规行为对商家采取包括但不限于违规商品下架或修改（商品信息修改）、店铺屏蔽、全店下架、限制使用咚咚、扣除违约金100000元、冻结资金、加收风险保证金、清退等处理措施。
**2.2****页面发布第三方信息：**
2.2.1违规程度一般：下架违规商品，每次扣除违约金500元；
2.2.2违规程度严重：下架违规商品、每次扣除违约金1000元、加收风险保证金；
2.2.3违规程度极其严重：限制提报营销活动7日、减少店铺曝光3日、删除违规商品，延长结算21日、咚咚禁言、降低店铺星级至2.5星30天、每次扣除违约金10000元、加收风险保证金；
2.2.4除上述处理措施外，如商家多次出现违规行为或因上述行为引起重大投诉的，京东有权视违规行为对商家采取包括但不限于违规商品下架或修改（商品信息修改）、店铺屏蔽、全店下架、限制使用咚咚、扣除违约金100000元、冻结资金、加收风险保证金、清退等处理措施。

# **【规则解读】**
**一、第三方信息包括哪些方面？**
第三方信息包括但不限于非京东购物链接、二维码、第三方网站信息、非咚咚联系方式、银行账号及其他付款方式、非京东平台标识等。
**二、商家发布或推送第三方信息的情形有哪些？**
商家发布或推送第三方信息的情形包括但不限于以下情形：
1、通过咚咚咨询诱导消费者，比如
（一）商家通过咚咚发送二维码、电话等信息，诱导消费者使用非京东方式进行支付和资金往来，或引导添加微信，电话等联系方式等，如：商家需要为消费者退款，主动要求消费者提供二维码或者其他支付账号；
（二）商家通过咚咚推送非平台购物链接、非京东平台标识等第三方信息，诱导消费者跳转至第三方网站或客户端或线下进行交易等。
2、在京东（JD.COM）页面诱导消费者，比如
（一）在商品标题、宣传语、商品图片及商品详情等位置发布或推送第三方信息；
（二）在店铺首页、店铺名称、店铺logo等位置发布或推送第三方信息。
3、通过其他方式诱导消费者，比如
（一）在实物包裹中包含第三方信息的宣传内容。
（二）通过短信通知等方式诱导到第三方网站或客户端等。
**三、发布或推送第三方信息违规如何区分违规程度的等级？**
1、违规程度一般包括但不限于：商家存在发布或推送第三方信息违规行为。
2、违规程度严重包括但不限于：
（一）商家大量发布或推送第三方信息；
（二）店铺多次发生“一般”违规程度的违规。
3、违规程度极其严重包括但不限于：
（一）店铺多次发生“严重”违规程度的违规；
（二）商家联系消费者添加京东外联系方式，或引导消费者跳转至第三方网站或客户端或线下进行交易；
（三）商家刻意研究、规避京东的管控方式，并加以传播、利用；
（四）商家存在其他恶意对抗管控的特殊情节。
**四、商家如何避免发布或推送第三方信息违规？**
1、商家应避免通过咚咚咨询服务、快递包裹等向消费者发送、传递含有第三方信息的消息及宣传内容；
2、商家应及时对店铺装修页面、商品详情页面进行自查自检；
3、商家在向消费者付款时需使用京东平台已支持的支付功能。
咚咚红包功能使用手册参见链接：[https://helpcenter.jd.com/vender/issue/42901.html](https://helpcenter.jd.com/vender/issue/42901.html)；
直赔系统使用手册参见链接：[https://helpcenter.jd.com/vender/issue/654-3850.html](https://helpcenter.jd.com/vender/issue/654-3850.html)；
小额打款功能介绍参见链接：[https://helpcenter.jd.com/vender/issue/44401.html](https://helpcenter.jd.com/vender/issue/44401.html)。
**附则：**
1.商家的行为，发生在本规则生效之日以前的，适用当时的规则。发生在本规则生效之日以后的，适用本规则。
2.京东可根据运营情况随时调整本规则并以公告的形式向商家公示。
3.商家应遵守国家法律、行政法规、部门规章等规范性文件。对任何涉嫌违反国家法律、行政法规、部门规章等规范性文件的行为，本规则已有约定的，适用于本规则。本规则尚无约定的，京东有权酌情处理。但京东对商家的处理不免除其应承担的法律责任。商家在京东的任何行为，应同时遵守与京东及其关联公司签订的各项协议；若商家服务协议约定的违约金金额高于规则约定的，京东有权适用协议约定。
4.本规则于2025年3月11日修订，于2025年3月18日生效。
