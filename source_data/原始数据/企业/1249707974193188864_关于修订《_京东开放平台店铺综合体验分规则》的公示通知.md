# 京东开放平台店铺星级规则
# **第一章 适用范围及定义**
**第一条** 本规则适用于在京东开放平台入驻开店的企业店铺，个人及个体店铺、外卖店铺不适用本规则。
**第二条 **京东开放平台店铺星级主要从客服咨询、物流履约、售后服务、商品体验等维度，对店铺近期提供的综合服务水平进行评定，不同行业还会结合品牌、价格等店铺其他表现进行补充。店铺星级为消费者提供购物决策依据，助力商家追求更好的全链路服务体验。
# 第二章 店铺星级的展示
**第三条 商家端展示**
商家可通过京麦后台查看商家店铺星级数据情况。
（一） 电脑端：京麦·商家中心首页-店铺星级；
（二） 手机端：京麦APP-我的-店铺信息。
**第四条 消费者端展示**
店铺星级会在电脑/手机APP/微信/手Q/M端面向消费者进行全方位展示，展示形式为星标，展示位置包括但不限于商品搜索页、商品详情页、店铺首页、店铺详情页、店铺关注页等。店铺星级越高，获取的消费者感知相对越强。
# 第三章 店铺星级的应用
**第五条 **店铺星级关联的流量及权益资源包括但不限于以下内容，具体获得权益可在商家后台进行查看：
**（一） 搜索排序因子：**店铺服务表现将影响该店铺中商品的搜索排序结果。
**（二） 营销活动门槛：**在营销活动场域，店铺星级会作为报名活动的筛选条件。
**（三） []商品搜索页展示：**店铺星级4.5-4.9店铺及五星店铺，会在京东APP商品搜索结果页商品卡位置展示。
**（四） 推荐场域门槛：**为了给消费者提供更优质的购物体验，店铺星级会作为首页推荐的筛选条件。
**（五） 频道活动门槛：**活动频道会根据频道特点，选择不同的店铺星级门槛。
**（六） 店铺商品数量资源：**店铺在售商品数量根据店铺星级月度进行调控。
**（七） 特色保险优惠：**店铺星级较高的商家，可以获得开通特色保险的价格优惠。所支持的特色保险优惠将不定期更新，以京东通知为准。

**第七条 **店铺星级中，售后服务、物流履约、商品体验、客服咨询4个因子，根据商家的指标在行业内的表现情况，其得分区间落在【5.5-10】分内，最低为5.5分，最高为10分。因子得分会在商品详情页及店铺详情页以分值或高/中/低向消费者展示。
**第八条 **当商家所在行业开通相关服务或者商家达成更高服务要求，店铺将获得额外加分；当商家的服务不满足消费者体验时，店铺将进行额外减分。根据行业不同，加减分项会进行调整，商家可在京麦商家后台查看加减分情况。具体影响加减分的服务或指标可在[京麦后台-店铺星级-规则解读](https://joyspace.jd.com/pages/9HjGxK5atMR2Sd8SsDJv)内查看，最终以商家后台展示为准。

**第十条 **为帮助商家快速展开经营，对于行业内服务数据不足的店铺赋予店铺星级初始值，默认初始星级为4.0星，同时对各因子赋予一定得分。初始值不代表商家真实服务水平，无法通过计算进行分数对应。店铺产生真实服务数据后，默认得分及星级将根据真实服务水平调整。
**第十一条 **根据各类目的行业情况及服务价值导向，不同类目下的因子权重侧重点不同，并且根据各类目的发展情况，不同时期也会有相应的调整。各类目的因子权重配置情况，以京麦后台展示为准。
**第十二条 **店铺星级同时考察商家的合规经营情况。不同行业还会结合品牌、价格等店铺其他表现，在此基础上对店铺星级进行不同的升级或降级展示。若统计周期内店铺无所在行业，或者商家不在店铺星级统计范围内，店铺星级将无数据，具体以商家后台页面为准。
**第十三条 **异常交易数据会在计算时予以剔除，存在通过不正当方式获取虚假销量、评分、评论或成交金额等行为的店铺，将根据情况严重程度调整该店铺的星级评估结果。
**第十四条 **如出现其他违反法律法规、平台规则、合同约定的相关情形，京东有权根据情况严重程度调整店铺星级结果，京东有权公示调整店铺星级认证标准。
## 第三节 2.5星展示
**第十五条 **当店铺近期存在违法违规或严重影响消费者体验的行为时，京东有权将店铺星级降至2.5星，降星天数根据行为不同有所区别，以商家后台展示为准。
**第十六条 **影响店铺星级的行为不定期更新，商家以页面提示为准。违规行为包括但不限于：（一） 出售假冒商品；（二） 恶意倒卖；（三）侵权违规；（四） 虚假宣传；（五） 不当发布商品或信息；（六） 发布混淆信息；（七）价格违规；（八）发布或推送第三方信息；（九）品牌欺诈；（十）商品质量不合格；（十一）资质不合格；（十二）发货违规；（十三）扰乱经营秩序；（十四）骚扰他人。
# 第五章 五星店铺
**第十七条** 五星店铺全面整合店铺服务、品质体验、经营状况等多个维度因子，代表店铺综合体验的最高水平。京东将对符合要求的店铺给予专属标识、流量及其他权益赋能，进行五星店铺认证打标，并面向消费者在商品搜索页、商品详情页、店铺首页、店铺详情页、店铺关注页等位置展示五星店铺标签。
**第十八条 五星店铺认证规则**
（一） 认证周期及有效期：每周的周二为五星店铺认证日，认证有效期为7天。
（二） 认证范围：在京东开放平台入驻开店的旗舰店、专卖店、专营店、卖场店店铺。
（三） 认证统计时间：以认证前两天的数据为准。
（四） 认证期间标签取消展示：当商家在五星店铺认证期间服务产品出现未开通状态或违规指标不通过，不满足认证标准的情况，五星店铺标签将取消展示。

# 第六章 附则
**第二十三条 **京东商家的行为，发生在本规则生效之日以前的，适用当时的规则。发生在本规则生效之日以后的，适用本规则。
**第二十四条** 京东可根据运营情况随时调整本规则并以公告的形式向商家公示。
**第二十五条** 商家应遵守国家法律、行政法规、部门规章等规范性文件。对任何涉嫌违反国家法律、行政法规、部门规章等规范性文件的行为，本规则已有约定的，适用于本规则。本规则尚无约定的，京东有权酌情处理。但京东对商家的处理不免除其应承担的法律责任。商家在京东的任何行为，应同时遵守与京东及其关联公司签订的各项协议。
**第二十六条 **本规则于2025年6月10日修订，于2025年7月9日正式生效。
