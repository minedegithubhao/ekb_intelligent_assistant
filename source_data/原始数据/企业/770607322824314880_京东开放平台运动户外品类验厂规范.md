**第一章 验厂标准要求**
**第一条 基本资质核验**
下表基本资质核验项目为必须符合的项目，商家应在验厂前提供电子版文件，若出现一项不符合，则不进行后续的验厂检查，直接判定为验厂不合格。

<table>
  <tr>
    <td>序号</td>
    <td>资质核验项目</td>
    <td>要求</td>
  </tr>
  <tr>
    <td>1</td>
    <td>营业执照</td>
    <td>生产企业应具有营业执照，经营范围应包含所生产的商品品类，企业应为存续状态。</td>
  </tr>
  <tr>
    <td>2</td>
    <td>行业许可</td>
    <td>如果企业所生产的产品涉及到国家行业许可，则需要企业持有相应许可证书，且该证书在有效期内。</td>
  </tr>
  <tr>
    <td>3</td>
    <td>合作协议</td>
    <td>如果生产企业与京东店铺不是同一公司主体，则需要商家提供生产企业与店铺主体公司的合作协议或其他证明其合作关系的文件。</td>
  </tr>
</table>

**第二条 质量管理要求**（70分）
（一）具体要求：对生产工厂的质量管理要求主要体现在质量体系、供应商管理、原料管理、生产过程控制、成品检验与仓储、售后质量等几个方面。根据条款1.2.2得出质量管理要求部分的得分。

<table>
  <tr>
    <td>评审条款</td>
    <td>评审要点</td>
  </tr>
  <tr>
    <td rowspan="7">1.1 质量体系</td>
    <td>公司应设立专门的质量部门，质量有关人员，有书面的、正式的和明确的岗位职责定义，员工了解其被赋予的职责。</td>
  </tr>
  <tr>
    <td>有书面的、系统的与质量有关的程序文件且定义清晰。</td>
  </tr>
  <tr>
    <td>员工应进行上岗培训、质量培训计划、专项培训（专业技能培训、安全知识培训、标准要求培训等），并建立培训档案。</td>
  </tr>
  <tr>
    <td>质量人员熟悉与所生产产品相适应的相关标准（国/行/企标），具备相关专业知识。</td>
  </tr>
  <tr>
    <td>质量人员了解国家市场监管部门对于所生产产品的管控手段和有关法律法规。</td>
  </tr>
  <tr>
    <td>所生产产品若涉及到国家行业许可或强制性认证，则企业应持有相应许可证书，该证书应在有效期内且覆盖所有京东平台在售的此类商品。</td>
  </tr>
  <tr>
    <td>管理评审、内审（至少一年一次），所有与标准质量程序存在的偏离的地方必须记录，内审完毕后实施有效的整改措施。</td>
  </tr>
  <tr>
    <td rowspan="3">1.2 供应商管理</td>
    <td>公司应对供应商进行评审，并保存相应的评审标准与评估记录。</td>
  </tr>
  <tr>
    <td>公司应建立合格供应商名录。</td>
  </tr>
  <tr>
    <td>采购原料的供应商应全部在合格供应商名录中， 并保存购买记录，至少抽查3种原料的进料记录，确定供应商存在于合格供应商名录中。</td>
  </tr>
  <tr>
    <td rowspan="10">1.3 原料管理</td>
    <td>公司应对原料及各类辅料建立检验标准。</td>
  </tr>
  <tr>
    <td>应记录和保存原料检验结果，并将相应的不合格情况反馈供应商，要求其改善。</td>
  </tr>
  <tr>
    <td>原料检验人员应经过了相应的培训，并保存培训记录。</td>
  </tr>
  <tr>
    <td>原料仓库应进行分类分区与明显标识。</td>
  </tr>
  <tr>
    <td>原料的摆放应有充足的货架或者托盘，原料不能直接放置在地上。</td>
  </tr>
  <tr>
    <td>原料仓库应保存出入库记录。</td>
  </tr>
  <tr>
    <td>原料仓库应执行先进先出管理程序。</td>
  </tr>
  <tr>
    <td>来料应有足够区域存放,且存放条件满足材料特性（包括但不限于温度、湿度、光照等）。</td>
  </tr>
  <tr>
    <td>定期对物料进行盘点，确定物料的数量正确，同时对物料的有效期进行再次确认。</td>
  </tr>
  <tr>
    <td>对于发现超出有效期或者报废的物料应有处理标准和记录。</td>
  </tr>
  <tr>
    <td rowspan="10">1.4 生产过程控制</td>
    <td>公司应建立领料程序，保存领料记录，并满足可追溯性要求。</td>
  </tr>
  <tr>
    <td>环境应整洁并适宜生产。</td>
  </tr>
  <tr>
    <td>应进行巡检或者首检。</td>
  </tr>
  <tr>
    <td>应配备检验人员，并保留培训记录和巡检记录。</td>
  </tr>
  <tr>
    <td>生产各工序应有相应的作业规范，并在显著位置保存作业指导书。</td>
  </tr>
  <tr>
    <td>公司应对生产员工进行作业规范的培训，并保存培训记录。</td>
  </tr>
  <tr>
    <td>公司应建立生产设备台帐。</td>
  </tr>
  <tr>
    <td>公司应制定生产设备保养计划，并按计划进行点检和维护，保存记录。</td>
  </tr>
  <tr>
    <td>半成品应做相应的标识与分类放置，保存生产流转记录并满足追溯性要求。</td>
  </tr>
  <tr>
    <td>生产过程中的不合格品应有相应的标识分区与处理记录。</td>
  </tr>
  <tr>
    <td rowspan="14">1.5 成品检验与仓储</td>
    <td>公司应建立成品检验标准，按照程序进行检验并保存检验记录。</td>
  </tr>
  <tr>
    <td>公司应对成品检验人员进行相应的岗位培训。</td>
  </tr>
  <tr>
    <td>应保留成品检验人员完整的培训记录。</td>
  </tr>
  <tr>
    <td>公司具有检测能力或委托第三方机构对成品进行安全/性能测试以便满足客户要求。</td>
  </tr>
  <tr>
    <td>应保存定期测试的记录，且报告合规。</td>
  </tr>
  <tr>
    <td>检验不合格的成品应区分放置并明确标识。</td>
  </tr>
  <tr>
    <td>返工返修应保存记录。</td>
  </tr>
  <tr>
    <td>成品仓的存储条件应适宜（包括但不限于温度、湿度、通风、光照、空间堆放高度）。</td>
  </tr>
  <tr>
    <td>成品的放置应有充足的货架、包装或者托盘，成品不应直接放置在地上。</td>
  </tr>
  <tr>
    <td>仓库应对不同产品做了相应的标识作为区分。</td>
  </tr>
  <tr>
    <td>应保存成品出入库记录。</td>
  </tr>
  <tr>
    <td>可以从成品追溯到原材料批次。</td>
  </tr>
  <tr>
    <td>产品应有可追溯的辨识代码。</td>
  </tr>
  <tr>
    <td>应建立检测测量试验仪器与设备的校准程序, 并制定校准计划，保存完整记录。</td>
  </tr>
  <tr>
    <td rowspan="4">1.6 售后质量</td>
    <td>由指定的人员负责退回货物的管理，有程序文件并保存记录。</td>
  </tr>
  <tr>
    <td>对客户反馈和投诉的质量问题进行分析，制订改善计划和措施，并加以落实，保存记录。</td>
  </tr>
  <tr>
    <td>企业应定期对产品出厂合格率进行总结，对生产过程中发生的不良问题进行分析及改善。</td>
  </tr>
  <tr>
    <td>企业应形成定期品质会议制度，对期间发生的品质问题进行汇总，分析和改善，保存会议纪要。</td>
  </tr>
</table>

（二）计分方式：质量管理部分共计48个条款，每个条款按照下表中工厂的不同表现可得到5分、4分、3分或者0分。质量管理总得分=(所有条款得分加和x70)/(48x5)。

<table>
  <tr>
    <td>得分</td>
    <td>具体表现</td>
  </tr>
  <tr>
    <td>5 分</td>
    <td>企业对该流程风险有正确识别，建立了完善的成文标准，对相应行动记录有书面保留，行动的执行完整有效。</td>
  </tr>
  <tr>
    <td>4 分</td>
    <td>企业对该流程风险有正确识别，建立了完善的成文标准，有相关执行记录，但是执行上存在瑕疵，但总体风险可控。</td>
  </tr>
  <tr>
    <td>3 分</td>
    <td>企业对流程风险有识别，存在有效的管理行动，但是未形成成文的记录及标准。</td>
  </tr>
  <tr>
    <td>0 分</td>
    <td>企业对该风险未识别，无实际管理行动。</td>
  </tr>
</table>

**第三条 成品质量检查**（30分）
（一）具体要求：在工厂随机检查5款确定在京东平台销售的成品商品各一件，并根据其是否存在严重缺陷、危害人身安全、三无、无法正常使用或组装、货不对板、配件不全、标志标识不合格、瑕疵、包装污渍等问题，按照条款1.3.2判定其得分。
（二）计分方式：成品质量检查共计检查5款商品各一件，每件商品按照下表中工厂的不同表现可得到6分、4分、2分或者0分。成品质量检查总得分=5件商品得分加和

<table>
  <tr>
    <td>得分</td>
    <td>具体表现</td>
  </tr>
  <tr>
    <td>6 分</td>
    <td>商品无任何瑕疵，标志标识符合国家要求，包装无污渍且可以有效保护商品。</td>
  </tr>
  <tr>
    <td>4 分</td>
    <td>商品存在1处轻微瑕疵，但不影响用户正常使用。</td>
  </tr>
  <tr>
    <td>2 分</td>
    <td>商品存在标志标识不合格，或存在2处及以上细微瑕疵，但不影响用户正常使用。</td>
  </tr>
  <tr>
    <td>0 分</td>
    <td>商品存在严重缺陷、危害人身安全、三无、无法正常使用或组装、货不对板、配件不全等严重影响用户使用的问题。</td>
  </tr>
</table>

**第二章 验厂得分计算方式**
质量管理要求和成品质量检查的得分之和，即为生产工厂验厂检查的总得分。
本规则于2021年10月27日首次发布，并于2021年11月3日生效。
