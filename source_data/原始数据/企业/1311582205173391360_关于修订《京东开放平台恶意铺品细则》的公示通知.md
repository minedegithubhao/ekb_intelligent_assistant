# 关于修订《京东开放平台恶意铺品细则》的公示通知

- rule_id: 1311582205173391360
- source_url: https://learn-jdm.jd.com/knowledge/rule/detail?ruleId=1311582205173391360
- label_names: 
- active_time: 2025-12-04
- update_time: 2025-11-28
尊敬的商家朋友：
您好，为适应业务发展，针对[《京东开放平台恶意铺品细则》](https://learn-jdm.jd.com/knowledge/rule/detail?ruleId=1303597708439298048)进行调整，现进行规则公示，公示期：2025年11月28日- 2025年12月4日，规则预计于2025年12月5日生效，公示期满后平台将另行发布正式生效规则。
核心调整点：
1、修改处理措施，处理措施中增加“全店商品下架和扣除店铺资金（包括保证金、货款等）”
修订后处理措施如下：
京东有权根据违规情节严重程度，对涉事商家店铺及关联店铺采取预警、开具违约单、管控、限权、关店等处理措施。
**管控措施**包括但不限于：违规商品下架且限制重新上架、限制店铺内商品发布数量上限、限制每日新发商品数、全店商品下架、全店商品流量降权 / 屏蔽、限制提报平台营销活动、加收风险保证金、扣除店铺资金（包括保证金、货款等）、限制关联品类商品上架；
**限权措施**包括但不限于：限制新品标识申请、限制厂自模式使用权限、限制 ISV 工具部分功能使用、限制商品信息批量编辑权限。
