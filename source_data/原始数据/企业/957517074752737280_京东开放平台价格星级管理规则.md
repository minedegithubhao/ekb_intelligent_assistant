# 京东开放平台价格星级管理规则

# **第一章 概述**
**第一条 **为营造健康、公开、透明的价格环境，为消费者提供良好的价格体验，特制定本规则。
**第二条 ** 本规则适用于在京东开放平台JD.COM入驻开店的商家。
**第三条** 价格星级是指以商品实价、低价为衡量标准的动态价格竞争力评估体系，旨在为具有价格竞争力的商品提供更多激励，为商家提供更简单、透明的价格运营规则。
# **第二章 价格星级的重要应用**
**第四条 **商品的价格星级指标是商品“搜索排序因子”的重要影响因素。商品的价格星级越高，将给消费者带来的购物体验更好，也有机会获得更多曝光资源。
**（一）搜索排序因子：**为构建更加优质的消费者体验生态体系，价格星级的表现将影响该商品的搜索排序结果，价格星级越高，获取的搜索排序结果相对越好。
**（二）营销活动门槛：**价格星级反映商品的价格竞争力，是商品参与营销活动的基础门槛，价格星级越高，获取的营销活动资源相对越好。
**（三）推荐排序因子：**为了给消费者提供更优质的购物体验，价格星级的表现将影响该商品的推荐排序结果，价格星级越高，获取的推荐场域资源相对越多。
**第五条 **价格星级会在商家运营环节全方位透出，商家可在如下路径中查询：
（一）京麦—商家成长—商家成长中心—价格星级；
（二）京麦—侧栏成长—商品成长—价格星级。
# **第三章 价格星级评定标准**
**第六条 **价格星级由低到高划分1.0星-5.0星，价格星级越高代表当前的价格竞争力越强。
（一）1-2星代表当前商品价格竞争力弱；
（二）3星代表当前商品价格竞争力处于日常水平；
（三）4星代表当前商品价格竞争力良好；
（四）5星代表当前商品价格竞争力优。
**第七条**** **价格星级通过整合商品自身实价、低价及对比同款历史价格和实时价格，并参考市场价格水平，进行价格竞争力判断。
（一）实价：指商品本身没有发生价格违规行为，若发生《[京东开放平台商品价格管理规则](https://rule.jd.com/rule/ruleDetail.action?ruleId=2436&btype=1)》中价格违规行为，京东有权视商家违规程度对价格星级进行无星或降星处理。
（二）低价: 如商品当前的价格对比其自身、同款商品的历史价格及实时价格具备竞争力，京东有权依据本规则对价格星级进行升星处理。
**第八条** 当商品在统计周期内的数据指标不足以判断价格竞争力时，价格星级将暂无数据展示。以下情况将导致价格星级缺失、无数据，具体以商家成长中心-价格星级页面为主。
（一）无有效订单的商品；
（二）当前下柜商品；
（三）无货商品；
（四）违规商品。
**第九条** 商家经营期间，京东有权依据商品自身及同款历史价格和实时价格数据，并参考行业及市场情况，对商品进行价格星级认证及动态调节价格星级认证标准。
**第十条 **价格星级统计期间虚假数据均会予以剔除，京东有权对通过不正当方式，包括但不限于获取虚假销量、设置虚假价格以换取京东资源的行为，根据严重程度采取店铺关停、清退处理，并保留采取其他必要措施的权利。
# **第四章 附则**
**第十一条 **京东开放平台商家的行为，发生在本规则生效之日以前的，适用当时的规则。发生在本规则生效之日以后的，适用本规则。
**第十二条** 京东可根据运营情况随时调整本规则并以公告的形式向商家公示。
**第十三条** 商家应遵守国家法律、行政法规、部门规章等规范性文件。对任何涉嫌违反国家法律、行政法规、部门规章等规范性文件的行为，本规则已有约定的，适用于本规则。本规则尚无约定的，京东有权酌情处理。但京东对商家的处理不免除其应承担的法律责任。商家在京东的任何行为，应同时遵守与京东及其关联公司签订的各项协议。若商家服务协议约定的违约金金额高于规则约定的，京东有权适用协议约定。
**第十四条** 本规则于2024年2月22日修订，于2024年2月29日生效。
