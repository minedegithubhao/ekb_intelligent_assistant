# 京东开放平台安心驼奶粉品质认证标准

# **一、范围**
本标准适用于京东平台销售的驼奶粉，包括纯驼奶粉和调制驼奶粉。

# **三、术语和定义**
3.1 驼奶粉：以单一品种的生驼乳为原料，经加工制成的粉状产品。
3.2 调制驼奶粉：以单一品种的生驼乳和（或）其全乳（或脱脂及部分脱脂）加工制品为主要原料，添加其他原料（不包括其他品种的全乳、脱脂及部分脱脂乳）、食品添加剂、营养强化剂中的一种或多种，经加工制成的粉状产品，其中来自主要原料的乳固体含量不低于70%。
3.3 京东安心驼奶粉：由京东或京东委托的行业独立三方机构，通过对驼奶粉或调制驼奶粉生产企业从源头实地验厂、产品基因检测、售中飞检抽检等生产加工全链路的审核和监管，确保其成分配料添加真实可信，符合相关产品执行标准及相关法律法规要求。
# **四、具体要求**
4.1 源头品控验厂：由京东或京东委托的行业独立三方机构，通过现场审核生产企业的原料供应链管理、过程及关键控制点管理、生产配料投料准确性、配料产能衡算及追溯等验证方案，确保生产环节配料添加真实合规，具备驼奶粉生产的质量保证能力，审核通过后将由京东出具合格审核报告或证书。
4.2 产品基因检测：从工厂生产现场取样或在京东平台匿名购买成品取样，通过检测驼奶粉基因成分以及其他可能掺假造假的动物源性成分，来进一步判定驼奶粉配料的真实性，检测通过后将出具驼奶粉基因成分检测报告。
4.3 售中品质监控：商品在京东平台售卖期间，京东将依靠体验大数据反馈，对可能存在风险的商家及商品，将采取不通知的方式对其工厂进行突击检查，确保其生产配料符合京东安心驼奶粉品质标准，同时不定期匿名下单送检第三方实验室抽检基因，保证正常销售期间产品配料真实。
4.4 商家自我承诺：在完成以上步骤的基础上，参与“京东安心驼奶粉”商家同时需与京东签订品质承诺书，并承担对等的责任和义务。
# **五、上线权益**
5.1 颁发认可证书：通过以上步骤后的商家，将由京东授予颁发《京东安心驼奶粉符合性验证证书》，证书有效性为1年。
5.2 营销资源支持：通过以上步骤的商品，可在京东商城商品主图及详情打上“安心驼奶粉”品质标签，并允许使用京东“安心驼奶粉”统一主图/详情视觉宣传，同时获得搜索营销相关推广活动支持。
# **六、认证要求**
6.1 资质审核：意向参与商家需提交《商家工厂能力介绍》、《生产许可证》、《产品三方检测报告》、《京东真驼乳申请单》提报至京东业务部门，并由京东品控进行审核。
6.2 实地验厂：京东品控或京东委托的三方机构与商家工厂约定现场审核时间，按照京东安心驼奶粉品质审核要求进行实地验厂。
6.3 产品检测：京东品控或京东委托的三方机构从工厂生产现场取样或在京东平台匿名购买成品取样，检测驼奶粉基因成分以及其他可能掺假造假的动物源性成分。
6.4 商家承诺函：商家需与京东签订《京东驼奶粉真实性品质承诺书》，并承担对等的责任和义务。
6.5 营销推广：通过以上步骤后的商家，将由京东授予颁发《京东安心驼奶粉符合性验证证书》，可在京东商城商品主图及详情打上“安心驼奶粉”品质标签，并允许使用京东“安心驼奶粉”统一主图/详情视觉宣传，同时获得搜索营销相关推广活动支持。
# **七、监督要求**
7.1 京东结合站内外用户反馈及舆情相关数据进行研判，对通过“京东安心驼奶粉”品质认证的商品采取进行飞行检查和匿名抽检，如发现商家标签信息不真实、投料配料不真实或不准确、产品检测不合格等违规行为，将按照撤销前台标签展示和营销活动，第一时间下架问题商品，并按照平台规则及采销合同承担对应违约金。
7.2 参与认证的商家需全力配合支持京东上线及监管要求及营销活动的开展支持，不得推诿，如有违反，将撤销商家“安心驼奶粉”提报资质和相关营销活动，同时承担在销售流通环节的相关损失。
# **八、附则**
8.1 京东商家的行为，发生在本管理规则生效之日以前的，适用当时的规则。发生在本管理规则生效之日以后的，适用本规则。
8.2 京东可根据平台运营情况随时调整本管理规则，并向商家进行共识。
8.3 商家应遵守国家法律、行政法规、部门规章等规范性文件。对任何涉嫌违反国家法律、行政法规、部门规章等规范性文件的行为，本规则已有规定的，适用于本规则。本规则尚无规定的，京东有权酌情处理，但京东对商家的处理不免除其应承担的法律责任。商家在京东的任何行为，应同时遵守与京东及其关联公司签订的各项协议。
8.4 本规则于2025年10月15日发布，于2025年10月22日生效。
