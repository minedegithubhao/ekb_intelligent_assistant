# 京东开放平台商品价格保护服务规则

# **第一章 概述**
**第一条**** **“商品价格保护”服务是指用户在京东开放平台购买带有“7天价保”/“15天价保”/“30天价保”/“90天价保”/“180天价保”/“365天价保”服务标识的商品后，商家承诺在用户下单后对应服务时效内，若因商家调整同一SKU价格，出现商品京东价发生降价情形，用户可以在客户端一键申请价格保护，由系统工具自动计算返还差价。
**第二条** 京东开放平台SOP、FBP店铺（京喜/众筹/租赁/新通路/同城购/除外）售卖的所有类目下的商品（二手/定制/拍卖/房地产/公益/挂号特殊商品除外），适用该价格保护服务规则。
**第三条**** **商品价格保护作为保障用户购物体验的基础服务，商家可自主为其店铺开通价保服务，开通服务后商品将按各类目对应时长执行，不同类目商品服务保障时长不同，京东有权进行调整。
# **第二章 服务细则**
**第四条**** **商家承诺提供“商品价格保护”服务的条件：
（一）用户购买的商品带有“7天价保”/ “15天价保”/“30天价保”/“90天价保”/“180天价保”/“365天价保”服务标识。
（二）用户在指定时间内，除下表不支持价保场景之外，其他因商家调整价格导致用户购买的商品降价均可申请差价补退，享受“商品价格保护”服务。
（三）“指定时间”指在用户下单的7天/15天/30天/90天/180天/365天内（从提交订单时间开始计算168/360/720/2160/4320/8760小时内，仅限在线支付订单）。

**第五条**** ****“商品价格保护”计算方式**
（一）单品价保金额=用户下单时基准价-申请价保时基准价-价保系统已经返还的金额；
（二）下单时基准价=下单时商品前台京东价-下单时促销金额（促销金额包含所有促销方式如直降、返现、返京券、满减等）；
（三）申请价保时基准价=申请价保时商品前台京东价-申请价保时促销金额（促销金额包含所有促销方式如直降、返现、返京券、满减等）；
（四）价保所产生的费用由商家承担，如订单产生价保，将由平台先行补偿至用户后，再由商家钱包扣除对应价保款项。
**第六条**** **如遇交通管制、大雨雪、洪涝、冰灾、地震、停电等不可抗力因素造成的未履行服务，商家不承担因此而导致的服务未履行责任。
**第七条** 商家承诺提供“商品价格保护”服务的，并不免除商家根据法律、法规及此前任何对商家具有约束力的协议、承诺所应当履行的销售者义务/责任。
# **第三章 商家管理**
**第八条**** **商家在履行“商品价格保护”服务期间，须履行京东平台价保服务承诺，遵守法律法规、协议及京东相关规则。价保服务履行之后，系统会根据实际价保退款情况，由商家钱包内扣除对应的价保费用。不以为用户提供服务体验保障为目的使用价保服务或不当使用等造成京东权益损失的，一经发现，京东有权立即停止商家的该项服务并追究商家违约责任。
**第九条** 若商家在包括但不限于商品标题、描述、商品详情页等自行承诺价保/保价/补差价等，商家需按承诺履约。
**第十条** 如用户因商家未履行价格保护服务发起纠纷并申请京东介入的，如双方就价格保护有另行约定的，以双方约定为准，京东介入后有权依据用户或商家提供的双方约定证明（如咚咚聊天记录）等进行判定。
（一）如商家在商品详情页提示支持价格保护，且对价格保护服务规则进行详情说明的，京东介入判定用户所购商品符合商家价格保护服务申请条件的，京东有权判定为商家责任，并有权要求商家按该价格保护服务规则为用户返还差价。
（二）如商家在商品详情页提示支持价格保护，但未对价格保护服务服务进行详情说明的，京东若判定用户所购商品符合该价格保护规则条件的，京东有权判定为商家责任，并有权要求商家按该价格保护规则为用户返还差价。
（三）若商家发生上述行为损害用户服务权益的，京东有权依据法律法规、协议、[《京东开放平台商家服务管理规则》](https://rule.jd.com/rule/ruleDetail.action?ruleId=941277149179023360&btype=1#%E7%AC%AC%E4%BA%8C%E8%8A%82%E5%9C%BA%E6%99%AF%E7%B1%BB%E8%BF%9D%E8%A7%84)中违背服务承诺等进行处理。
**第十一条 **如商家恶意逃避“商品价格保护”服务承诺，经核实店铺、商品品牌、品名、型号（包括但不限于颜色、数量、存储容量、口味、香型、包装、款式、材质、尺寸、尺码、规格、生产日期等）、赠品等单一商品维度的基本信息均一致且在售、可售时，京东有权补偿用户并追究商家违约责任。另外，定制商品由于其特殊性，不适用于“同款商品”的定义，需单独识别。
**第十二条** 在商家履行“商品价格保护”服务期间，当商家钱包资金不足支付价保费用时，系统会关闭商家的价保服务。待商家钱包资金可支付价保费用后，商家需自行开通价保服务。
# **第四章 不支持**** ****“商品价格保护”服务场景**
**第十三条** 如出现不可抗力或重大情势变更的情况（包括但不限于重大灾害事件、活动或服务受政府机关指令需要停止举办或调整的、活动或服务遭受严重网络攻击或因系统故障需要暂停的），则商家有权依相关法律法规的规定主张免责。
**第十四条**  如用户存在利用价保服务进行盈利或非法获利、以任何形式转让或转移其所享有的价保权益、非本人使用的，或者商家有合理理由怀疑用户存在不当使用价保服务或补差权益的，商家有权不予补差，并可要求用户退回已价保补差金额。无不支持用户价保补差情形，如下：
（一）通过任何不正当手段或以违反诚实信用原则的方式使用价保服务的，如通过与其他用户串通或利用机器等方式进行作弊、刷取或套取补差等的行为；
（二）用户申请使用价保服务的订单获取的商品或服务并非其个人生活所需，或非用于个人消费或使用的；
（三）用户欠缺交易意愿，如利用退款流程、补差流程的便利性以实现获取权益或利益的不当目的，补差后出现大量退货退款的；
（四）用户被认为存在违反商品或服务提供方的相关协议或规则的行为，或已经被商品或服务提供方限制接受服务或购买商品的权利；
（五）其他违反法律法规、诚实信用原则等的行为。
**第十五条**** **针对某一订单，如用户已获得买贵双倍服务或价保服务中任一服务补偿的，则用户在申请另一服务时须以实付订单金额扣除已获得任一服务（补偿）金额后的价格作为到手价。具体规则如下：
（一）如用户已申请买贵双倍赔服务且获得相关补偿，则用户须以其实付订单金额扣除已获得买贵双倍赔补偿后的金额作为到手价基础（用户实际支付金额-买贵双倍赔补偿金额）申请价保服务；
（二）如用户已申请价保服务且已获得相关补偿，则用户须以其实付订单金额扣除已获得价保的金额作为到手价基础（用户实际支付金额-价保金额）申请买贵双倍赔服务。
# **第五章 附则**
**第十六条**** **京东开放平台商家的行为，发生在本规则生效之日以前的，适用当时的规则。发生在本规则生效之日以后的，适用本规则。
**第十七条**** **京东可根据运营情况随时调整本规则并向商家公示。
**第十八条**** **商家应遵守国家法律、行政法规、部门规章等规范性文件。对任何涉嫌违反国家法律、行政法规、部门规章等规范性文件的行为，本规则已有约定的，适用于本规则，本规则尚无约定的，依照法律法规处理。商家依据相关规则承担的相关责任并不免除其应承担的法律责任。商家在京东的任何行为，应同时遵守与京东及其关联公司签订的各项协议。协议与平台规则对于违约责任、违约金等约定不一致的，两者以高者为准。
**第十九条** 本规则于2024年12月12日修订，于2024年12月19日生效。
