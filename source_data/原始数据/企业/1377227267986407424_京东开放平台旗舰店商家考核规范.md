# 京东开放平台旗舰店商家考核规范

# **第一章 概述**
**1.1 ****目的及依据**
为保障平台（域名为jd.com）健康发展，促进平台与商家共同成长，持续为消费者提供优质的商品、品牌和服务，根据国家法律法规规范，以及“京东 JD.COM”开放平台服务协议、《京东开放平台规则总则》等相关协议和规则，特制定《京东开放平台旗舰店商家考核规范》。
**1.2 ****适用范围**
适用于平台内所有的旗舰店商家（包含大药房旗舰店），以及因考核不达标被从旗舰店降级的商家。
# **第二章 实施细则**
**2.1 ****考核对象**
2.1.1 平台内所有的旗舰店、大药房旗舰店商家均需参加，且需符合对应考核标准，如未达标，将会被从旗舰店降级为专卖店，大药房旗舰店降级为专营店（被降级的商家将无法自行申请店铺类型升级）。新入驻的旗舰店商家享有开店90天内免考期，开店90天后自动进入正式考核范围。
2.1.2 因考核不达标被从旗舰店降级的商家可继续参加考核，下一考核周期考核达标后，平台会恢复该店铺的旗舰店身份（无需商家自行申请升级）。如未达标，将继续维持现有店铺类型。
**2.2 ****考核规则**
2.2.1 考核周期：自然月内每双周为一个考核周期
2.2.2 考核指标：双周内平均店铺星级
店铺星级口径：剔除**默认赋星商家和无星级商家**
**2.2.3 ****考核结果**
- 若店铺达标，则维持旗舰店资格。
- 若店铺不达标（店铺星级≤4星），则直接执行降级处理。
- 已降级店铺仍继续考核，若在后续的考核中，双周内店铺星级持续≥4.2星，平台会恢复该店铺的旗舰店身份（无需商家自行申请升级）。
**2.3 ****考核影响**
商家店铺类型因旗舰店考核发生变更（包含考核不达标从旗舰店降级和降级商家考核达标的升级两种情形）后，存在以下相关影响：
- 店铺命名规范需要符合对应店铺类型的命名规范要求。详情可参考[京东开放平台店铺命名及信息规范](https://learn-jdm.jd.com/knowledge/rule/detail?ruleId=2598)。
- 从旗舰店降级的商家，在店铺类型降级期间，不可发布仅允许旗舰店类型发布的类目/品牌商品。有旗舰店经营要求的类目/品牌，具体可见[京东开放平台商品类目管理规范](https://learn-jdm.jd.com/knowledge/rule/detail?ruleId=826299126802157568)。
- 除上述相关影响外，店铺由旗舰店降级后，平台将依据与降级后店铺类型有关的规则对店铺进行管理。
另外，若商家存在违反法律法规要求、或违反平台规则可能或已经造成平台风险或消费者权益受损的，平台有权对店铺类型做降级处理，包含但不限于以下情形：
- 商家收到涉及侵权、售假、虚构或夸大产品功效、所售商品抽检不合格等违反法律法规的处罚，及涉及监管处罚、产生重大负面舆情等；
- 商家发布容易造成消费者产生混淆的商品和信息的行为，造成一定负向影响的；
- 商家服务能力、经营状态持续不佳，包括但不限于店铺星级较低、经营指标达不到旗舰店相关要求等。
# **第三章 附则**
3.1 京东开放平台商家的行为，发生在本规则生效之日以前的，适用当时的规则。发生在本规则生效之日以后的，适用本规则。
3.2 京东可根据运营情况随时调整本规则并以公告的形式向商家公示。
3.3 商家应遵守国家法律、行政法规、部门规章等规范性文件。对任何涉嫌违反国家法律、行政法规、部门规章等规范性文件的行为，本规则已有约定的，适用于本规则。本规则尚无约定的，京东有权酌情处理。但京东对商家的处理不免除其应承担的法律责任。商家在京东的任何行为，应同时遵守与京东及其关联公司签订的各项协议；若商家服务协议约定的违约金金额高于规则约定的，京东有权适用协议约定。
3.4 本规则于2026年5月21日新增，于2026年5月28日生效。
