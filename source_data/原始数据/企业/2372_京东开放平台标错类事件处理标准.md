# 京东开放平台标错类事件处理标准

# **第一章 概述**
**第一条** 本规则适用于京东开放平台的所有商家。
**第二条** 标错类事件指商品信息在平台信息展示过程中发生标注错误，包括价格、数量、促销语、图片、规格参数等方面信息。
标错价指因商品价格设置、价格调整、促销/优惠券设置、促销/优惠券叠加、限量促销的库存数配置、违规换品错误等因素，导致价格标错，出现诸如商品价格设定明显低于该商品正常价格范围、或导致京东损失(包括但不限于经济损失、舆情影响）等的风险事件。
# **第二章 处理流程**
**第三条** 标错事件报备
对于已产生的标错/低价的订单，商家应按订单约定履行合同义务。商家履约发货的订单，如后续产生投诉等情形将按正常订单进行处理。如商家认为有部分或全部订单可能无法履约发货，商家需在知道或者应当知道事件发生的2小时内通过【京麦平台—促销—管理工具—价格风控服务—风险订单报备】进行报备。报备前产生的消费者投诉（如未能及时发货等）情形均按照正常订单进行处理，报备后，京东会与商家一同协商处理方案，并会持续追踪商家处理进度；
**第四条** 标错事件判定及对应处理
（一）商家报备后，京东有权根据包含但不限于京东价、成交价、历史成交价、常规尺寸等进行判断，如经京东判断为非标错类事件，或京东判定为标错事件但订单损失金额小于￥1万元（含）的，商家应按照订单约定履行发货义务，订单发货后产生相关投诉等情形，均按正常订单的流程进行处理。
（二）经京东判断确为标错类事件且损失金额大于￥1万元的，针对可能无法履约发货的有效订单（订单状态为等待付款、暂停、等待出库均为有效订单），京东可协助商家制定商家及京东均认可的订单处理方案（由此产生的费用由商家承担），商家应按照与京东约定的处理方案在规定时效内完成订单处理；
处理时效要求：
1. 订单量≤500单的，商家需在下单时前台页面宣传的发货时效内，或形成方案起的2个自然日内处理完成全部订单（订单取消/发货视为处理完成）。
2. 500<订单量≤1000单的，商家需在下单时前台页面宣传的发货时效内，或形成方案起的3个自然日内处理完成全部订单（订单取消/发货视为处理完成）；
3. 订单量＞1000单的，商家需在下单时前台页面宣传的发货时效内，或形成方案起的5个自然日内处理完成全部订单（订单取消/发货视为处理完成）；
（三）如京东和商家无法达成均认可的订单处理方案，或商家未在规定时效内完成订单处理，京东有权采取商品下架并限期整改、冻结货款等措施，同时京东有权参考商品历史售价、原京东价、同类商品市场价等价格制定订单的处理方案（由此产生的费用由商家承担），直到处理完成；
（四）本规则中因单次标错价格产生的损失金额计算方式：（正常前台京东价-标错价）*商品数量；新上架商品的损失金额计算中，“京东价”以该商品的市场实际成交价为参考。
（五）本规则中“订单量”指在商家标错商品信息期间，发生的购买标错商品信息商品的已支付订单数量（含货到付款订单）。订单量以京东平台系统数据为准。
# **第三章 商家须知**
**第五条** 如商家发生标错事件，应根据本规则处理流程第一时间报备，并优先解决因标错产生的订单问题。如商家因标错产生重大损失的，依据相关法律规定，若商家对标价存在重大误解，可通过诉讼或仲裁的方式行使撤销权。
**第六条** 京东开放平台非国家行政或司法机构，亦非专业的纠纷解决机构，京东对于商家及消费者之间纠纷的处理完全是基于相关法规的规定、协议的约定及买卖双方的意愿，京东仅能以普通非专业人士的知识水平和能力对消费者和商家提交的相关证据材料进行鉴别和认定，京东对据此作出的交易纠纷处理结果等无法保证完全正确，也不对此承担任何责任。京东对消费者主张交易引发的额外损失或法定赔偿事宜无权认定及执行，故消费者对该等事宜的赔偿主张经平台调解双方无法达成一致的，由买卖双方自行协商或通过其他途径处理。
**第七条** 商家应当加强内部管理，熟练掌握及充分理解平台各类优惠措施配置、叠加等逻辑，反复校验，审慎对待每一次营销活动设置或商品信息发布。
# **第四章 违规处理**
**第八条** 如商家拒绝配合处理、未在规定的时间内完成处理工作或因处理不当导致大量客诉的，京东有权按照《京东开放平台商家违规管理总则》中“履约违规”对商家进行处理。
**第九条** 连续3个月内同一商家发生3次及以上标错类事件或者单次标错类事件产生重大影响的，京东有权按照《京东开放平台商家违规管理总则》中”扰乱经营秩序”对商家进行处理。
# **第五章 附则**
**第十条** 京东开放平台商家的行为，发生在本管理规则生效之日以前的，适用当时的规则。发生在本管理规则生效之日以后的，适用本规则。
**第十一条 **京东可根据平台运营情况随时调整本管理规则并以“京东开放平台”公告的形式向商家公示；商家入驻京东即表示接受京东其后不时调整、颁布的管理规则。
**第十二条** 商家应遵守国家法律、行政法规、部门规章等规范性文件。对任何涉嫌违反国家法律、行政法规、部门规章等规范性文件的行为，本规则已有规定的，适用于本规则。本规则尚无规定的，京东有权酌情处理。但京东对商家的处理不免除其应承担的法律责任。商家在京东的任何行为，应同时遵守与京东及其关联公司签订的各项协议；如有违约即视为对本规则的违反。
**第十三条** 本标准于2025年11月27日修订，于2025年12月4日生效。
