# 关于时尚类目商品、商家质量问题管控的公告

- rule_id: 1177074635410378752
- source_url: https://learn-jdm.jd.com/knowledge/rule/detail?ruleId=1177074635410378752
- label_names: 
- active_time: 2024-11-29
- update_time: 2024-11-22
尊敬的各位商家：
为保障广大消费者的权益，提高平台上鞋靴、服饰内衣、运动户外、珠宝首饰、钟表眼镜、箱包皮具、美妆护肤、母婴（仅童装、童鞋）、鲜花/奢侈品（仅奢侈品）类目商品的质量与信誉度，针对上述类目商品质量表现持续处于尾部的商品和商家实施必要的管控措施，具体管控方式如下：
管控周期：双周
管控方式：警告、降权、屏蔽（仅商品维度）
开始时间：2024年11月30日起开始实施
具体方案：
- 商品维度
|
**商品管控措施（双周）**
|
|
**管控条件**
|
**管控阈值（商品质量指数）**
|
**管控措施**
|
|
商品质量指数C级
|
落入C等级1~6次
|
预警
|
|
落入C等级累计≥7次
|
降权7天（降权50%）
|
|
商品质量指数D级
|
落入D级累计1-3次
|
预警
|
|
落入D等级累计4-6次
|
降权7天（降权50%）
|
|
落入D等级累计≥7次
|
屏蔽7天
|
- 商家维度
|
**商家管控措施（双周）**
|
|
**管控条件**
|
**管控阈值（店铺+三级类目QCR）**
|
**管控措施**
|
|
双周内完成订单数≥20单
|
均≥1.5倍所属三级类目均值
|
警告
|
|
均≥2倍所属三级类目均值
|
降权7天（降权50%）
|
|
均≥3倍所属三级类目均值
|
降权14天（降权50%）
|
|
管控方式：双周一期统计数据，针对不达标店铺触发预警，6期内累计4期（含）不达标，基于第4期不达标数据进行全店管控。
|
各位商家，我们诚邀您与我们共同努力，持续提高商品质量，让消费者在这里购物放心、满意。
本次调整于2024年11月22日公示通知，将于2024年11月30日正式生效。
