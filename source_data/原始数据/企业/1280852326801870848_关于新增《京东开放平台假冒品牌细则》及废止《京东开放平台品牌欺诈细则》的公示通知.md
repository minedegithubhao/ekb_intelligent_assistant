# 关于新增《京东开放平台假冒品牌细则》及废止《京东开放平台品牌欺诈细则》的公示通知

尊敬的商家朋友：
您好，为适应业务发展需求，将新增《京东开放平台假冒品牌细则》及废止《京东开放平台品牌欺诈细则》，现进行规则公示，公示期：2025年9月4日- 2025年9月10日，规则预计于 2025年9月11日生效，公示期满平台将另行发布正式生效规则。
**拟新增****规则全文如下：**
**1、【违规定义】**
1.1 假冒品牌：商家在商品发布、销售、售后等阶段可能导致消费者对商品品牌产生认知偏差 的行为。
**2、【处理措施】**
2.1 违规程度轻微：警告、下架违规商品、限制创建商品、禁止上架待售商品、违规品牌停用 等措施；
2.2 违规程度一般：警告、下架违规商品、限制创建商品、禁止上架待售商品、违规品牌停 用、资金冻结、延长结算周期、账户止付、类目限制或停用等措施；
2.3 违规程度严重：扣除违约金 5000-30000 元不等（实际金额与商家经营收入、违规程度、 违规后果等相关）、违规商品下架、限制创建商品、禁止上架待售商品、违规品牌停用、类目 限制或停用、资金冻结、延长结算周期、账户止付、搜索和推荐降权、禁止投放或提报活动、 禁止投放或播放广告、降低商品阈值等措施；
2.4 违规程度极其严重：扣除违约金 5000-100000 元不等（实际金额与商家经营收入、违规程 度、违规后果等相关）、全店商品下架、限制创建商品、禁止上架待售商品、违规品牌停用、 类目限制或停用、资金冻结、延长结算周期、账户止付、搜索和推荐降权、禁止投放或提报活 动、禁止投放或播放广告、降低商品阈值、店铺清退等措施。
2.5 除上述措施外，如商家累计两次违规或单次违规大量出现以上违规行为或因上述行为引起 重大投诉或舆情的，平台有权基于违规程度、违规后果进行加重处置。
**除了上述处理措施之外，京东有权视商家违规影响、情形等，同时采取包括但不限于以下措 施：** 违规商品下架、全店商品下架、限制创建商品、禁止上架待售商品、限制通过接口批量操作商 品、限制新品标签权益、降低店铺在售商品阈值、禁止投放或提报活动、禁止投放或播放广 告、品牌停用和限制申请品牌、类目停用和限制开通类目、搜索降权、推荐降权、降低店铺综 合体验分、屏蔽降权、扣收违约金、账户止付、定额或超额冻结、扣除店铺资金（包括质保 金、货款等）、增加或提高涉事店铺应交保证金金额、扣收保证金延长结算周期、店铺清退等 处理措施。同时，京东有权对涉事商家的关联方或关联店铺采取一项或多项管控措施，具体管 控措施详见《京东开放平台关联店铺管理规则》，如违约金不足以弥补京东损失的，商家仍应 赔偿京东损失，京东有权视商家违规影响，扣除商家部分或全部违规经营所得（订单款项）， 并对商家予以追责。
# **【规则解读】**
**一、假冒品牌有哪些常见违规情形？** 包括但不限于： 1）信息层：商品挂靠品牌 A，但商品标题、图片、规格选项、详情介绍等任一或多处位置显 示商品为品牌 B 或无品牌； 2）实物层：商品页面宣传品牌 A，消费者实际收到的商品实物为品牌 B 或无品牌；
**二、商家因假冒品牌被开具违约单后，可提供哪些材料进行申诉？** 包含但不限于： 1）与违规商品信息对应的商品实物信息、照片等； 2）采购、进货凭证； 3）库存证明； 4）其他可证明页面品牌与实物品牌一致的有效证明。
**三、商家应做好哪些资料留存？** 商家应保存在售商品和过往销售商品的相关授权资质和采购链路凭证，以备查验。
**四、假冒品牌违规程度严重、极其严重的情形包括哪些？** 包括但不限于： 1）大量或多次发布造成消费者产生品牌认知偏差信息的； 2）发布造成消费者产生品牌认知偏差信息且造成恶劣影响的，如产生消费者资损或引起消费 者投诉举报等； 3）发布造成消费者产生品牌认知偏差商品或信息，已造成舆情或监管事件的，或其他对京东 产生负面影响的情形。
**附则：**
1.京东开放平台商家的行为，发生在本规则生效之日以前的，适用当时的规则。发生在本规则 生效之日以后的，适用本规则。
2.京东可根据运营情况随时调整本规则并以公告的形式向商家公示。
3.商家应遵守国家法律、行政法规、部门规章等规范性文件。对任何涉嫌违反国家法律、行政 法规、部门规章等规范性文件的行为，本规则已有约定的，适用于本规则。本规则尚无约定 的，京东有权酌情处理。但京东对商家的处理不免除其应承担的法律责任。商家在京东的任何 行为，应同时遵守与京东及其关联公司签订的各项协议；若商家服务协议约定的违约金金额高 于规则约定的，京东有权适用协议约定。
4.本规则于2025年9月4日新增，于2025年9月11日生效。
