# 关于修订《京东开放平台“0元开店”试运营管理规则》公示通知

- source_url: https://learn-jdm.jd.com/knowledge/rule/detail?ruleId=1321670857456599040
- label_names: 
- active_time: 2025-12-31

尊敬的商家朋友：
您好，为适应业务发展，针对《[京东开放平台"0元开店"试运营管理规则](https://learn-jdm.jd.com/knowledge/rule/detail?ruleId=915123374395625472)》进行调整，现进行规则公示，公示期：2025年12月25日- 2025年12月31日，规则预计于2026年1月1日生效，公示期满后平台将另行发布正式生效规则。
**核心调整点：**针对部分表述进行优化，将参与“0元开店”试运营的类目单独以表格形式呈现，同时在《[京东开放平台类目资费规则](https://learn-jdm.jd.com/knowledge/rule/detail?ruleId=638209647311982592)》删除【试运营阶段】字段。
**修订后内容如下：**
# **第二章 试运营规则**
**第二条** 从2023年4月1日起，针对新入驻京东开放平台的POP模式商家，部分类目支持“0元开店”试运营，即试运营期间不缴纳保证金，试运营结束进入正式运营阶段后再缴纳保证金，
参与“0元开店”试运营的类目可详见：0元试运营类目明细。
注：具体明细以最终公示为准。
