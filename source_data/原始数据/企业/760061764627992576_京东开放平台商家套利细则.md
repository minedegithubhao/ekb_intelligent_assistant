# 京东开放平台商家套利细则

# **1.****【违规定义】**
**1.1****商家套利：**指商家不遵守诚信原则，实施谋私舞弊、虚假操作、套利或套垫付款行为（包括但不限于套取京东款项、京东优惠券、促销、京豆、红包、佣金、垫付款及/或其他京东经营资源等）或实施其他侵害京东、客户、平台其他商家利益的行为。本规则所称商家，包括商家加盟门店等关联机构；商家应保证其关联机构同样遵守平台规则的约定，商家对其关联机构的行为造成的一切后果承担连带保证责任。
# **2.****【处理措施】**
2.1违规程度轻微的，剔除商家优惠券、京豆、红包等补贴、优惠或奖励资源的使用资格30日；
2.2违规程度一般的，剔除商家优惠券、京豆、红包等补贴、优惠或奖励资源的使用资格60日；
2.3违规程度严重的，剔除商家优惠券、京豆、红包等补贴、优惠或奖励资源的使用资格90日；
2.4违规程度极其严重的，清退店铺。
2.5除上述处理措施外，京东有权视违规程度追缴套利或套垫付款款项、延迟结算、关闭/停用价保等权益、关闭/停用工具、商品下架、全店限售、钱包冻结、暂停分账、限制服务市场交易、门店置休、门店下线、永久限制参与各类型营销优惠、限制同主体下店铺使用营销优惠、关闭违规商家店铺，商家应向京东赔偿全部损失，除此之外还应支付违约金壹佰万元或按所涉商品销售额2倍支付违约金，二者从高适用。对于商家恶意索赔侵害京东关联方利益的行为，京东有权开具违约单并扣除违约金。 此外，视上述违规程度，京东有权冻结违规商家及其关联商家在京东的款项；商家行为构成犯罪的，京东有权向公安部门报案以进一步追究商家刑事责任。
# **3.【规则解读】**
**3.1****京东如何区分违规程度的轻微、一般、严重、极其严重？**
京东将根据包括但不限于商家违规套取平台补贴金额、套取平台补贴的次数等来判定违规程度。
**3.2****商家通过不正当方式套取平台补贴，常用的手段包括哪些？**
商家套利的常用手段包括但不限于以下情形：
（1）商家通过非真实交易来套取京东平台补贴或其他京东款项的，包括但不限于：
1）自买自卖：商家使用自有账号、操纵或利用他人账号购买自家所售商品的；同一用户（包括但不限于同一设备、同一IP等），在同一商家店铺内高频购买同一个/同一类商品，超出单一消费者合理消费需求，套取京东或京东关联方款项。
2）异常出入库：同一批商品当天入库当天或短时间内出库且主营商品物流出入库未发生实质变化的；
3）拍A发B，出库商品与售卖链接商品不符的。
（2）商家将不得使用特定类目平台补贴的商品，违规发布在该特定商品类目的；
（3）商家将一件商品拆分为多个不同形式或页面发布，重复使用平台补贴。包含但不仅限于以下情况：商品和商品的运费分开发布或一个商品拆分不同价格打包出售；
（4）商家通过提高商品价格方式套取平台补贴，包括但不限于以下情况：提高价格搭配店铺促销套取平台补贴；利用平行优惠套取平台补贴；价格设置高于同平台或市场售卖价格套取平台补贴；
（5）商家利用用户套取平台补贴，包括但不限于以下情况：用户使用平台补贴后支付多件商品，但商家只发货其中一件商品；商家提高商品价格，诱导或同意用户使用平台补贴后返还消费者相应差价；
（6）商家利用平行优惠政策，根据京东营销活动资源的让利力度，自行降低店铺原有优惠力度，使得京东营销资源与店铺优惠叠加后，用户未能享受到应有的优惠金额；
（7）商家在交易过程中，骗取京东款项的行为，如：利用非法手段骗取京东保证金、京东优先赔付或用非法手段套取京东货款等款项的；利用京东退款服务骗取退款款项的；利用京东返利服务骗取京东返利款项的；
（8）商家实施的其他套取平台补贴等资源的不正当手段。
**3.3****商家通过恶意索赔套取京东关联公司理赔金额的常用手法有哪些？**
商家恶意索赔的常用手段包括但不限于以下情形：
（1）使用与事实不符的索赔资料向京东物流索赔（如使用重复破损照片；货物到货后蓄意破坏形态；谎报货物状态、数量、成交价格；）
（2）因收寄双方交易纠纷、非物流问题等原因，多次产生网络舆情投诉、邮政投诉的行为；
（3）使用虚假下单、指定付费场所签收等方式，以达到获得赔偿、牟利等目的的行为；
（4）商家实施的其他恶意索赔获取利益资源的不正当手段。
# **4.附则**
4.1商家的行为，发生在本规则生效之日以前的，适用当时的规则。发生在本规则生效之日以后的，适用本规则。
4.2京东可根据运营情况随时调整本规则并以公告的形式向商家公示。
4.3商家应遵守国家法律、行政法规、部门规章等规范性文件。对任何涉嫌违反国家法律、行政法规、部门规章等规范性文件的行为，本规则已有约定的，适用于本规则。本规则尚无约定的，京东有权酌情处理。但京东对商家的处理不免除其应承担的法律责任。商家在京东的任何行为，应同时遵守与京东及其关联公司签订的各项协议；若商家服务协议约定的违约金金额高于规则约定的，京东有权适用协议约定。
4.4本规则于2025年6月30日修订，于2025年7月7日生效。
