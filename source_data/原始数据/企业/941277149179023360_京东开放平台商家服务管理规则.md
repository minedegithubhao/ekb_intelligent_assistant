# **第一章 概述**
**第一条**为维护京东开放平台的正常经营秩序，保障京东消费者的合法权益，实现商家店铺的规范化运营，根据国家法律法规规范，以及“京东 JD.COM”开放平台服务协议、[《京东开放平台规则总则》](https://rule.jd.com/rule/ruleDetail.action?ruleId=2368)等相关协议和规则制定本规则。
**第二条**本规则适用于京东开放平台第三方商家。
**第三条**服务违规指商家在店铺经营过程中出现咨询违规、售后违规、处理投诉违规等影响消费者交易体验的行为。
**第四条**若商家因违反法律法规、协议、本规则等被采取治理措施的，商家可在违规处理之日起7日内（京东审核时间除外），通过京东商家后台提交违规申诉申请，申诉材料提交后京东有权依据[《京东开放平台违规申诉管理规则》](https://rule.jd.com/rule/ruleDetail.action?ruleId=2370&type=0&btype=1)等进行核实处理。
# **第二章 服务违规管理**
**第五条 服务质量考核**
**1.****【违规定义】**
1.1 3分钟人工回复率不达标：指月度考核周期内，当店铺有效咨询服务量（在线咨询量+留言咨询量）大于200单时，工作时间段内（每天9:00:00~22:59:59）消费者的咨询及留言会话中，每一个消息轮次的等待时间均在3分钟内的在线咨询会话比例小于70%。
1.2 咨询差评率不达标：指月度考核周期内，当店铺咚咚有效咨询量大于100单时，消费者在咚咚咨询后对商家客服的不满意和非常不满意评价量占同周期内咚咚咨询量的比例大于等于1%。
1.3 售后处理超时率不达标：指月度考核周期内，当店铺命中超时的售后服务单量大于3单时，店铺整体超时的售后服务单量占同周期内暂完结售后服务单总量比例大于20%。
1.4 工单24小时完结率不达标：指月度考核周期内，当店铺工单创建量大于10单时，商家的合格工单数量占同周期内工单总数量的比例小于等于70%。合格工单指①消费者投诉工单（催单工单、安装求助工单、催开发票工单、物流投诉工单）—商家在24小时内回复且该订单6天内未被京东介入；②平台协同工单—商家在24小时内回复且未出现追加工单让商家重新处理。
【备注】以上考核项的考核周期为每个自然月，每月7日公布上一个周期的考核结果。
1.5发货能力不达标：指周度考核周期内，当商家的延迟单量和延迟发货率同时满足以下任一种情况（①延迟单量4到5单且延迟发货率>40%；②延迟单量6到9单且延迟发货率>30%；③延迟单量10到20单且延迟发货率>20%；④延迟单量21到50单且延迟发货率>10%；⑤延迟单量>50单且延迟发货率>5%），则视为商家发货能力不达标。延迟发货率计算方式：超承诺发货时间仍未发货的单量 / 应发货单量。若商家在经营过程中，出现批量订单违规发货或导致消费者严重投诉，京东有权适用本规定对商家进行处置，不参考上述延迟单量和延迟发货率的要求。
【备注】发货能力不达标的考核周期为每个自然周，每周公布上一个周期的考核结果。
**2.****【处理措施】**
2.1月度考核周期：若商家月度考核不达标，京东有权视违规程度对商家进行预警、开具违约单，并根据每月商家不达标的指标个数、次数及扣除违约金。每月每项不达标指标扣除违约金最低200元，最高800元（M2档）。考核周期内，商家服务质量考核不达标违约金为各项不达标指标违约金的总和。

**第六条 服务态度违规**
**1.****【违规定义】**
1.1 咨询服务场景违规：商家咨询服务场景违规包括但不限于以下情况：
1.1.1 商家服务效率差：商家对消费者的咨询不做回复，或回复较慢等不活跃表现，在工作时间（每天9:00:00~22:59:59）内商家未提供在线客服服务等行为。
1.1.2 消费者咨询体验差：消费者对商家提供的咨询服务普遍存在不满意或非常不满意的评价结果等行为。
1.1.3 商家服务态度差：商家在与消费者沟通过程中存在答非所问、敷衍不耐烦、嘲讽、不文明/不尊重/不礼貌”等消极态度及言论，或在服务过程中出现拖沓、回复模糊、推诿不解决、不兑现承诺等消极处理行为。
1.1.4 恶意拉黑消费者：商家在消费者咨询过程中无故将消费者拉至黑名单，导致消费者无法正常享有客服服务，或产生投诉的行为。
1.1.5 未告知消费者详细且真实的商品信息，是否存在随机发货等情况，未及时处理相关的配送或售后等咨询问题。
1.2商家拒绝与京东客服沟通：处理用户问题时，商家与京东客服无法达成一致，商家出现2次及以上直接挂断，或其他恶意拒绝与京东客服沟通的行为。
1.3 骚扰他人：指商家对包括但不限于消费者、京东工作人员、其他与京东开放平台相关的关联人员等，实施诽谤、骚扰、跟踪、诋毁、谩骂或使用任何引起他人不满的字句，妨害他人合法权益的行为。
**2.****【处理措施】**
2.1 咨询服务场景违规：若商家发生上述咨询服务场景违规行为，京东有权视商家违规程度进行警告、平台介入处理、向消费者推荐其他店铺商品、开启智能应答工具、要求客服进行学习及考试、限制使用咚咚产品功能、扣除200-800元/单的违约金（M2档）关闭咨询入口等处理措施。若消费者因上述咨询服务违规行为投诉至京东，京东有权视商家违规程度并依据规则，为消费者执行退货退款等措施。
2.2商家拒绝与京东客服沟通：若商家存在拒绝与京东客服沟通的服务态度违规行为，京东有权视商家违规程度采取警告、扣除200-800元/单的违约金（M2档）等措施。
2.3 骚扰他人：若商家存在骚扰他人的服务态度违规行为，京东有权视商家违规程度进行警告、要求客服进行学习及考试、限制使用咚咚产品功能、扣除800-100000元/单的违约金（M2/M3/M4档）、关闭咨询入口、店铺屏蔽、限制提报营销活动，限制创建商品、禁止上架待售商品、下架违规商品、降低店铺星级至2.5星、冻结资金、加收风险保证金、清退等处理措施。
**具体处理措施如下：**
- 违规程度轻微的，警告；
- 违规程度一般的，扣除违约金800元（M2档）；
- 违规程度严重的，扣除商家近14天成交金额的10%，最低100元，最高10,000元（M3档），限制提报营销活动7日，限制创建商品7日，禁止上架待售商品7日；
- 违规程度极其严重的，扣除商家近14天成交金额的20%，最低300元，最高100,000元（M4档），且京东有权冻结店铺资金（包括货款、保证金等），并执行店铺清退。
- 除上述处理措施外，如商家多次出现违规行为或因上述行为引起重大投诉的，京东有权视违规行为对商家采取包括但不限于店铺屏蔽，全店商品下架，限制使用咚咚，降低店铺星级至2.5星，冻结资金，加收风险保证金，清退等处理措施。
**3.****【赔付措施】**
针对骚扰他人违规，消费者发起投诉经京东判定投诉成立且不属于下述规则解读中不支持赔付的场景的，商家需每次向消费者赔付30000个京豆。
**4.****【规则解读】**
**4.1****骚扰他人包括但不限于以下情形：**
4.1.1商家在沟通过程中或者在评价或者在评价补充回复中等出现包括但不限于挑衅、质问、诋毁、辱骂等词语（含商家发送后被系统拦截的违规信息），或使用任何形式的有人身攻击、侮辱性等不文明语言的，严重影响消费者购物体验的语言等；
4.1.2商家在日常运营中骚扰、谩骂、威胁、恐吓、诅咒京东工作人员或其他与京东开放平台相关的关联人员等，散播恶意攻击京东的消息等情形的。
4.1.3商家在短时间内通过咚咚卡片、电话、短信、邮件等方式反复联系消费者，或已经遭到明确拒绝后仍然进行联系的。
**4.2****骚扰他人违规如何区分违规程度的等级**
4.2.1 违规程度一般包括但不限于：
- 通过电话、短信、咚咚、邮件等方式对他人实施侮辱、恐吓威胁、语言攻击的行为（如：带有辱骂含义的词汇、动物指代词汇、诅咒词汇等不文明用语）；
- 通过电话、短信、咚咚、邮件等方式频繁联系他人，影响他人正常生活的行为。
4.2.2 违规程度严重包括但不限于：
- 严重影响他人正常生活，给他人身心造成极大伤害或造成严重恶劣影响的行为，如商家未经他人允许私自上门、向消费者邮寄商品进行恐吓等行为；
- 商家多次产生“违规程度一般”情形；
4.2.3违规程度极其严重包括但不限于：多次产生“违规程度严重”情形或产生严重损害消费者体验重大客诉或舆情的。
**4.3****什么情形会判定消费者投诉成立但不支持赔付？**
出现下述任一情形的：
1、消费者与卖家均存在骚扰、辱骂等相关行为，但消费者先行出现前述行为，且卖家未产生促使消费者情绪升级、矛盾激化等行为的。如，卖家未有激烈（基于频次、程度等判断）回骂、向消费者邮寄商品进行恐吓、威胁上门等。
2、经京东认定，消费者在交易前不以成交为目的，交易后不以解决售后争议为目的，滥用本规则以实现其牟利意图的。如，出现挑衅、故意激怒卖家等行为，诱使或致使卖家违规。
**第七条 售后服务流程违规**
**1.****【违规定义】**
1.1未与消费者协商一致处理售后：指商家在审核处理售后服务单环节，存在未与消费者核实确认、未协商一致情况下，不合理拒绝售后或要求消费者变更诉求、擅自更改服务单处理方式、不合理费用扣除、无正当理由拒收原返或直接关闭消费者的售后服务单等消极售后行为。
商家未与消费者协商一致不合理处理售后包括但不限于以下情形：
1.1.1未与消费者协商一致修改售后信息：指商家无正当理由要求消费者修改售后类型、售后申请原因、返件方式、退款金额等，且修改后的售后信息非消费者真实诉求。
1.1.2未与消费者协商一致拒绝售后申请：指商家未与消费者沟通一致直接拒绝售后、或未按照与消费者已沟通一致的内容执行，导致消费者对协商或拒绝结果不认可。或在消费者售后过程中，刻意提出不合理要求和限制条件等，刁难阻碍消费者正常售后诉求。
1.1.3未与消费者协商一致费用扣除：指商家处理售后退款时，未与消费者协商一致，无故给消费者少退款、或以运费、包装费、服务费等理由进行扣减要求消费者承担，导致消费者收到的退款金额少于实际支付金额、消费者无法再次发起售后等情况。
1.1.4未与消费者协商一致拒收/原返/关单：指商家未妥善解决消费者售后诉求，在未沟通一致的情况下，无正当理由拒收返件商品、操作原返、或强制关闭服务单。
1.1.5未与消费者协商一致擅自更改服务单处理：指商家在售后服务单处理环节，未与消费者提前协商一致，对消费者原售后诉求为换货/补发/维修类的服务单直接操作退款，消费者对处理结果不认可。
1.2售后服务超时：指商家未按照[《京东开放平台售后服务管理规则》](https://rule.jd.com/rule/ruleDetail.action?ruleId=2595&btype=1)、[《京东开放平台取消订单服务管理规则》](https://learn-jdm.jd.com/knowledge/rule/detail?ruleId=590903538377428992)等规定时效内及时响应及处理消费者所提出的售后服务申请，或超过与消费者所约定时效，包括但不限于：超时未审核售后服务单（含未收货退款服务单）、超时未处理售后服务单、超时未确认收货、超时未操作退款、超时未履行与消费者所约定的售后服务承诺等。
1.3售后处理不规范：指商家在售后服务过程中存在售后承诺与平台服务产品不一致、未有效履行承诺、服务单处理多次出现问题、不合理增加消费者售后成本、违背协议或规则向消费者提供服务等妨害消费者权益的行为。
商家售后处理不规范包括但不限于以下情形：
1.3.1售后承诺与平台服务产品不一致：指商家自行承诺的特色服务、商品退换等售后服务与商家开通的平台服务产品不一致或存在过度售后承诺。“售后服务承诺与平台服务产品不一致” 规则解读可查阅：[售后承诺与平台服务产品不一致规则解读](https://learn-jdm.jd.com/knowledge/rule/detail?ruleId=1295653344178184192)
1.3.2售后承诺不兑现：
1）承诺承担费用或补偿不兑现：指商家在售后过程中，自行向消费者承诺承担服务费用，如售后返件运费等，或与消费者约定给予其他形式补偿，但实际未履行。
2）承诺承担服务不兑现：指商家在售后过程中，应承担特色服务产品或其他如安装、退换货、补发、维修、等保障义务、或与消费者约定的其他售后相关责任，但实际商家拒绝承担、提供虚假售后信息逃避承担、或未有效履行。
3）商家违反了京东开放平台服务协议中关于消费者权益保障要求的。
4）消费者因购买、使用商品或者接受服务受到人身、财产损害，商家拒绝赔偿的。
5）商家所提供售后服务低于京东要求，包括但不限于商家七天无理由设置低于京东要求等。
1.3.3售后地址不规范： 指商家未完善必要信息或通过设置不合理条件排除或限制消费者申请售后，减轻或免除商家应承担的售后责任，增加消费者售后成本的，如：
1）设置境内偏远地区售后地址，或非全球购/全球售商家设置境外(含港澳台地区)售后地址，增加消费者售后费用的。
2）商家所提供售后地址不精确或无效的，包括但不限于提供错误地址等导致消费者寄出退货快递后实际无人签收/丢件；或商家提供空白售后地址、无法与一二三级地址匹配的四级地址等，导致消费者无法正常寄出退货商品。
3）商家所提供售后联系方式有误或无效的，包括但不限于退货联系电话有误，实际接通并非商家工作人员；或类似13800000000的无效号码，始终为无人接听、关机、停机、占线、不在服务区状态，消费者因此无法正确寄出退货商品、或寄出后快递实际无人签收/丢件。
4）商家未在商家后台完善正确的自主售后退货地址并操作设置有效的默认地址，因商家原因导致消费者返件地址有误或正常售后流程受阻。
1.3.4不合理指定快递公司：指商家在售后服务过程中，不合理强制要求消费者仅可使用商家指定物流公司，导致消费者售后体验损伤、升级京东介入。
1.3.5售后服务瑕疵：
1）指商家在售后过程中，未一次有效解决消费者售后诉求，增加消费者售后流程费力度，包括但不限于进行售后退款、退货、换货、补发、维修等服务中反复出现瑕疵，导致消费者不满、升级京东介入。
2）商家在发布商品时所填写商品重量、体积等基础信息有误或无效，导致消费者售后成本增加或物流无法揽收，包括但不限于实际边长、重量与系统维护不一致，由此造成消费者运费损失等情况。
1.4 发票违规：商家未按照《[京东开放平台发票服务管理规则](https://learn-jdm.jd.com/knowledge/rule/detail?ruleId=562814072702242816)》对其销售的商品或者提供的服务及时向消费者开具发票，包括但不限于开票过程中存在开具虚假发票、未依照订单信息或承诺方式提供发票、超期未开具发票、未按时在【京麦-消费者发票】上传电子发票、开错发票、拒绝开票、无故驳回消费者发票申请等行为。
**2.****【处理措施】**
2.1违规程度轻微：若经京东巡检识别，店铺存在上述违规行为，京东有权采取包括但不限于要求商家依承诺提供服务、依据对应特色服务产品规则进行处理、京东介入处理、对商家进行预警整改、下架违规商品 、对商家进行预警、向消费者赔付、扣除50元/单的违约金（M1档）、为消费者执行退货退款等措施，退货运费及相应损失由商家承担。
2.2违规程度一般：若经消费者投诉核实或京东巡检识别店铺自然年内存在两次及以上轻微违规，京东有权采取包括但不限于要求商家依承诺提供服务、依据对应特色服务产品规则进行处理、京东介入处理、下架违规商品 、向消费者赔付、扣除200-800元/单的违约金（M2档）、为消费者执行退货退款等措施，退货运费及相应损失由商家承担。
2.3违规程度严重：若经京东巡检识别或消费者投诉，店铺产生严重损害消费者体验重大舆情、监管投诉、监管举报、司法诉讼等的，京东有权采取包括但不限于要求商家依承诺提供服务、依据对应特色服务产品规则进行处理、京东介入处理、下架违规商品 、向消费者赔付、单次违规扣除违规商家近14天成交金额的10%作为违约金（M3档）、为消费者执行退货退款等措施，退货运费及相应损失由商家承担。
**第八条综合服务场景违规**
**1.****【违规定义】**
1.1 售后服务体验不达标：指商家在售后处理过程中，因处理售后不及时或消费者对商家售后处理结果不满意等，带给消费者负面体验或无法达到平台服务要求的。售后服务体验不达标主要考核售后服务因子得分，每个自然月内，店铺星级售后服务因子得分7.8分及以下视为不达标。每月7日公布上一个周期的考核结果。若商家未妥善处理消费者售后服务诉求，导致消费者升级京东介入，京东会视商家售后纠纷的商责情况动态调整处置周期。
1.2平台介入率不达标：考核周期内，因商家未妥善处理消费者服务请求，导致平台介入率超过平台管控阈值。
1.3监管投诉成立违规：考核周期内，商家在经营期间，产生监管投诉记录并被京东判定为商责的订单量大于等于2单。其中考核周期指自然月考核，每月7日公布上一个周期的考核结果。
1.4赔付欠款不归还：商家经营期间，京东依据相关纠纷标准等协议规则判定，因商家原因导致的消费者申请退款、赔付诉求成立时，在优先处理消费者退赔诉求后，由此产生的费用商家未归还的情形。
