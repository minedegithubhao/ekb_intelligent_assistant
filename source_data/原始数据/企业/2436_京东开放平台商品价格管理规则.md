# 京东开放平台商品价格管理规则

# **第一章 概述**
**第一条 **为保障消费者享有良好的商品价格体验，倡导公平合理的定价行为，根据国家法律法规、平台协议等制定本规则。
**第二条 **本规则适用于入驻京东开放平台JD.COM的所有商家。
**第三条 **商品价格类型
（一） 京东价：指商家在京东开放平台销售商品时设置的价格。
（二） 划线价：商品展示的划横线价格为参考价，并非原价，该价格可能是品牌专柜标价、商品吊牌价或由品牌供应商提供的正品零售价（如厂商指导价、建议零售价等）或其他真实有依据的价格。
（三） 普惠实时到手价：基于商家报名时设置的前台京东价及优惠，计算出用户最容易享受到的成交价格。
（四） 未来到手价：未来指定时刻的预计成交价格。
（五） 拼购价：指用户通过参与拼购活动而获得的商品价格。
（六） 企业会员价：企业用户专享的商品价格。
（七） 企业新人价：企业用户注册未首购以及首购订单下单后30天内，可享受的优惠价格。
（八） 新人价：新用户在购买完成第一笔订单时，可享受的首单新人优惠价格。
（九） 学生价：符合认证条件的学生用户专享的商品价格。
（十） 店铺会员价：按不同会员等级设置的店铺会员专享的单品促销优惠价格。
（十一） 山姆价：山姆会员用户专享的商品价格。
（十二） 陪伴价：母婴建档用户的专属商品价格。
（十三） 秒杀价：指通过参与秒杀活动而获得的商品价格。参见[《京东开放平台单品秒杀管理规则》](https://rule.jd.com/rule/ruleDetail.action?ruleId=816253070936772608)。
（十四） 超级补贴价：指通过参与超级补贴活动而获得的商品价格。参见[《京东开放平台超级补贴管理规则》](https://rule.jd.com/rule/ruleDetail.action?ruleId=1031746067852562432)。
（十五）预售商品价格：指通过参与预售活动而获得的商品价格。参见[《京东开放平台预售管理规则》](https://rule.jd.com/rule/ruleDetail.action?ruleId=817929024989958144)。
# **第二章 商品定价原则**
**第四条 **为保障消费者享有公平合理的价格体验，商家设置的商品价格（包括但不限于商品销售价、优惠价、到手价、成交价等）应当具备价格竞争力。损害消费者价格体验的行为包括但不限于：在京东平台所售卖的商品价格，高于同一时间段在其他渠道所售的同款商品价格。
**第五条 **政府指导价、政府定价
商家适用政府指导价或者政府定价的商品的京东价，需执行政府指导价或政府定价，不得改动或者超出浮动幅度。
**第六条 **商品价格、运费
商品价格和运费描述应真实准确，不应偏离实际运费，同时应符合市场规律和[《京东行业运营标准》](https://psm.jd.com/index.html)。
**第七条 **商品价格宣传
商品售价应以京东前端页面展示出来的京东价为准，商品发布的价格需与商品描述中标注的价格一致（如：商品发布京东价为100元，但却在商品描述中标注产品其他销售价格的，京东有权判定为不符合本条要求）。
**第八条 **促销商品价格
（一） 为保障商家平稳参与大促营销活动，商家在参与平台促销活动前，除市场价格浮动情形外不应随意上调包含活动价、京东价在内的商品价格。
（二） 除本规则约定外，大促时商品价格设置还应遵守大促期间活动商品价格力等相关平台规则、公告，具体以平台发布内容为准。
**第九条 **特殊品类商品价格
（一） 经营商品二级类目为二手书的商家，所销售图书的定价不得过高于京东图书类目下同ISBN商品的最低售价（售价指实际销售价格）。
（二）吊牌价：有商品吊牌的类目（包括服饰内衣、鞋靴、钟表眼镜、美妆护肤、运动户外、母婴、鲜花/奢侈品、箱包皮具、居家布艺、钟表眼镜、珠宝首饰）吊牌价格填写需真实、有依据。
**第十条 **商家其他标价行为
（一） 商家采用与其他经营者或者其他销售业态进行价格比较的方式开展促销活动，应当准确标明被比较价格的含义，且能够证明标示的被比较价格真实有依据。
（二） 商家采取馈赠物品或者服务等方式开展促销活动，如果馈赠物品或者服务标示价格（价值）的，所标示的价格（价值）应当真实明确，不得以过高或过低的价格发布赠品。
# **第三章 价格违规管理**
**第十一条 **价格违规管理要求
商家需严格遵守《中华人民共和国价格法》、《中华人民共和国反不正当竞争法》、《明码标价和禁止价格欺诈规定》等相关法律法规，遵循市场规律，确保可提供任一价格的合法依据或可供比较的出处，不得出现任何欺骗性价格标识以及价格违规行为，比如商家利用虚假的或者使人误解的标价形式或者价格手段，欺骗、诱导消费者与其进行交易等。
**第十二条 **价格违规行为类型
价格违规行为包括但不限于：不合理制定价格、提价促销、违背价格承诺、运费虚高、价格波动异常、宣传价格与实际不符、不当发布价格信息、价格引流、其他价格违规行为等。
**第十三条 **各类价格违规行为
1.【违规定义】
1.1价格违规
1.1.1不合理制定价格：商品价格（包括但不限于商品销售价、优惠价、到手价、成交价等），出现包括但不限于以下情况：
1）商家所设置的商品价格明显高于同一时间段平台内或者其他渠道的同款商品价格；
2）显著高于平台认定的行业一般价格水平的；
3）商家所设置的商品价格较该商品历史价格有大幅度上涨的。
1.1.2提价促销：商家开展促销优惠活动，刻意抬高京东价，其京东价显著高于促销商品日常售价，商品将被判定为提价促销。
举例：某一商品日常售价100元，开展满200减100促销活动，大幅提高售价至200元，实际成交价仍为100元，恶意诱导消费者的。
1.1.3违背价格承诺：指商家销售商品和提供服务时有价格承诺，但实际执行时不履行或者不完全履行，未实际让利消费者或导致消费者产生投诉。
举例1：承诺商品当前价格为全年最低价，但活动结束后商品价格更低的。
举例2：商家与消费者约定按100元预售价售出该商品，但实际交易时要求消费者支付额外差价，提高现价或拒不交易的。
举例3：如商家参与平台促销活动时，承诺参与活动价格为历史一段时期内相对低价，但实际活动期间的商品价格高于历史价格，违背诚信原则，未能有效履行价格承诺义务的。
1.1.4运费虚高：商家发布实物商品时设置不符合行业标准的高运费，或者发布虚拟物品时设置运费。
举例：一件衣服运费价格设置为300元；某商家在店铺内发布商品为浙江移动100元在线充值，商品信息中标注“此商品需要5元运费”。
1.1.5低价高邮：商家展示价格设置低价，同时设置高邮费（邮费相较于商品本身到手价价格较高）拉高消费者实际支付价。如：展示价格1元的商品，绑定10元的邮费。
1.1.6价格波动异常：某商品单日京东价调价频率显著高于同类商品，其调价频率已影响到消费者的正常购买行为。
1.1.7宣传价格与实际不符：包括但不限于商品主图价格信息与实际成交不相符，利用虚假的或者使人误解的价格信息，欺骗、诱导消费者与其进行交易。
1.1.8不当发布价格信息：设定与实物不相符的价格，超出行业等定义的价格范围。
1.1.9价格引流：商家通过给SKU设置异常低的价格谋取不当利益或吸引流量的方式，包括但不限于将不同类商品作为多个SKU堆砌在同一商品（SPU）链接里，其中部分SKU设置极低的价格。
1.1.10提价套取补贴：商家通过虚标价格、变相涨价等形式套取补贴，实现补贴后到手价高于商品日常售价的行为。 举例：如商品日常售价为500元，参加补贴活动后商家将商品价格提高到700元，用户到手价为560元高于日常售价500元。
1.1.11其他价格违规：除不合理制定价格、提价促销、违背价格承诺、运费虚高、宣传价格与实际不符、不当发布价格信息、价格引流等行为之外的其他违反法律法规、协议、规则等的价格违规行为。
2.【处理措施】
2.1违规程度分级
一般违规：存在价格违规行为。
严重违规：通过价格违规手段牟取不当利益，或多次违反平台规则。出现包括但不限于以下任一情形：
1）累计触发3次及以上一般违规；
2）价格违规获取不当得利的；
3）发生客诉或舆情的；
极其严重违规：恶意对抗平台规则、造成重大客诉、舆情或平台重大利益损失。出现包括但不限于以下任一情形：
1）累计触发10次及以上一般违规；
2）通过各种手段，逃避平台处置、对抗平台规则的；
3）价格违规造成严重后果、被追究或可能追究行政或刑事责任等情形的；
4）产生重大客诉或舆情。
2.2客诉与舆情判定标准
1）平台内客诉：用户通过订单评价、在线客服、投诉工单等官方渠道反馈价格问题；
2）社会化媒体舆情：用户于微信、微博、抖音、小红书等公共平台发布价格违规证据；
被主流媒体转载报道或登上社交平台热搜榜单；
3）政府监管介入：用户向市监局、消协、法院等部门举报/诉讼。
2.3处置规则
1）违规程度一般的：平台会采取包括但不限于预警、搜推降权/屏蔽、下架违规商品、限制营销活动、限制广告活动、降低店铺星级、限制商品上架/发布、钱包/账户冻结、类目/品牌停用等；
2）违规程度严重的：平台会采取包括但不限于预警、搜推降权/屏蔽、下架违规商品、限制营销活动、限制广告活动、降低店铺星级、限制商品上架/发布、钱包资金冻结、处以商家50元以上20000元以下违约金等处置；
3）违规程度极其严重的：平台会采取包括但不限于搜推降权/屏蔽、禁止营销/广告活动、降低店铺星级、全店商品下架、禁止商品上架/发布、类目/品牌停用、店铺关停清退等处置，同时视恶意程度冻结账户、处以商家20000元以上1000000元以下违约金等处置。
# **第四章 附则**
**第十四条 **京东商家的行为，发生在本规则生效日之前的，适用当时的规则；发生在本规则生效之日以后的，适用本规则。
**第十五条 **京东可根据运营情况随时调整本规则并向商家公示。
**第十六条 **商家应遵守国家法律、行政法规、部门规章等规范性文件。对任何涉嫌违反国家法律、行政法规、部门规章等规范性文件的行为，本规则已有约定的，适用于本规则。本规则尚无约定的，京东有权酌情处理。但京东对商家的处理不免除其应承担的法律责任。商家在京东的任何行为，应同时遵守与京东及其关联公司签订的各项协议；若商家服务协议约定的违约金金额高于规则约定的，京东有权适用协议约定。
**第十七条 **本规则于2025年8月13日修订，于2025年8月20日生效。
