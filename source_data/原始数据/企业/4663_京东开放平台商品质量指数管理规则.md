# 京东开放平台商品质量指数管理规则

# **第一章 概述**
**第一条** 京东开放平台“商品质量指数”是体现京东开放平台品质价值主张、商品质量情况的模型。为提高消费者购物体验，帮助商家了解消费者对商品品质的需求痛点以及了解店铺商品所处行业水平，提升商品品质竞争力，基于消费者反馈的商品品质信息，构建商品质量指数模型，制定此规则。
# **第二章 定义**
**第二条 **商品质量指数定义
商品质量指数：指综合消费者评论、商品退换修、客诉、赔付等多维度对商品的质量反馈，并考察商责纠纷情况，构建大数据模型，对京东开放平台上的商品给出质量等级，用来评估判断商品质量在行业内的综合质量表现。商品质量指数分为AA、A、B、C、D五个等级。
**第三条** 适用范围
本规则适用于入驻京东开放平台的商家所经营的实物类商品，服务和虚拟类目下的商品除外。
**第四条 **等级标准
等级AA：指商品质量表现明显高于京东开放平台上同类在售商品质量的行业均值；
等级A：指商品质量表现高于京东开放平台上同类在售商品质量的行业均值；
等级B：指商品质量表现位于京东开放平台上同类在售商品质量的行业均值；
等级C：指商品质量表现低于京东开放平台上同类在售商品质量的行业均值，此类商品将被限制参加营销活动等；
等级D：指商品质量表现明显低于京东开放平台上同类在售商品质量的行业均值，此类商品将被限制参加营销活动等。
注：行业均值指京东平台上同类目在售商品的消费者质量平均抱怨率。
# **第三章 商品质量指数构成**
**第五条 **指标构成
商品质量指数是以消费者质量抱怨率为基础，在商品各自类目下排序，得出质量指数得分和等级。
消费者质量抱怨率：指近30日内，在该商品妥投完成的订单行中，商品质量问题抱怨订单行占比，订单数据来源包括因商品质量引起的咚咚售后咨询、客户投诉、评价、售后退换修、赔付、市监投诉、商责纠纷，计算时会按照订单号去重，不会重复计算。
**第六条 **查询路径
商家后台-店铺体检-店铺品质：[https://illegal-jdm.shop.jd.com/shopquality](https://illegal-jdm.shop.jd.com/shopquality)
# **第四章 商品质量指数的重要应用**
**第七条 **商品质量指数等级体现京东开放平台品质价值主张，影响商品在平台获得的多种权益或活动，具有多种扩展应用，包括但不限于本章以下各条内容。对于本规则未概述完整的，以合作协议、相关规则具体公示内容为准。
**第八条 **商品质量指数对参加营销活动的影响
商品质量指数会被各活动场选择作为商品参与营销活动的基本要求之一，商品质量指数等级为D级的商品将无法报名参加营销活动（包括但不限于秒杀、闪购、超级补贴等）。同时，各类目、各活动、频道等可能会根据其特性设置更高的要求，若商品不符合要求，将被限制报名参加活动。
# **第五章 附则**
**第九条** 京东开放平台商家的行为，发生在本规则生效之日以前的，适用当时的规则。发生在本规则生效之日以后的，适用本规则。
**第十条 **京东可根据运营情况随时调整本规则并向商家公示。
**第十一条 **商家应遵守国家法律、行政法规、部门规章等规范性文件。对任何涉嫌违反国家法律、行政法规、部门规章等规范性文件的行为，本规则已有规定的，适用于本规则。本规则尚无规定的，京东有权酌情处理，但京东对商家的处理不免除其应承担的法律责任。商家在京东的任何行为，应同时遵守与京东及其关联公司签订的各项协议。
**第十二条** 本规则于2023年8月17日修订，2023年8月24日生效。

**FAQ**
**Q****：质量指数排名多久更新一次？**
A：每日更新，时效T+1，即当日显示昨日质量指数结果，根据每项指标的取数周期进行取值计算。
**Q****：有什么办法可以提升店内商品质量指数等级呢？**
A：商品质量指数以消费者质量抱怨率为基础，降低消费者抱怨率可以提高商品质量指数等级，以下方案可参考：
- 控制消费者抱怨订单数量，比如优化店铺选品逻辑、关注商品本身质量、采取有效措施提高商品品质；换位思考理解好消费者质量需求，完善商品详情介绍和提示信息，做好售前售中的询问提醒等服务。
- 提升商品订单销量，在商品订单上升速度超过消费者抱怨订单增长速度时，消费者抱怨率自然降低。注意：对于刷单等虚假交易数据，一旦发现，模型会进行剔除。
- 及时处理质量原因的售后退换修，不升级成纠纷。
**Q****：新品的质量指数是多少？是否会被限制提报活动？**
A：近30日无有效订单的商品（如无销售量的新品），没有商品质量指数等级，不会因商品质量而影响提报营销活动。
**Q****：为什么好评率很高商品还被评为D？**
A：商品质量指数目前考察商品质量问题反馈，计算的是近30日的数据并去除刷单等虚假数据，历史数据并不计算在内。
