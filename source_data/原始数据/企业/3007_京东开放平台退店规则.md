# 京东开放平台退店规则

# **第一章 概述**
**第一条 **本规则主要明确店铺因主动申请退店、合同到期或平台清退等原因导致店铺停用的相关事项。
# **第二章 退店场景**
**第二条 **退店是指商家退出京东平台停止店铺经营，具体场景包括主动退出、合同到期和店铺清退，退店后店铺无法重新启用。
**（一）主动退出：**商家在合同有效期内主动向京东申请关闭店铺并停止经营。
**1. ****条件：**对已达成的交易，商家须履行完毕发货、退换货、维权投诉处理等交易保障义务，并承诺依照相关法律法规、规则及商家自主承诺履行质保维修等义务。
**2. ****流程：**商家可于商家后台提交退店申请，自商家提交退店申请次日起，店铺首页将进行退店公示，持续三十天，公示期满即完成退店，退店后店铺名称可供其他商家申请使用。若商家申请退店前已全额退保证金并完成30天公示，则退店阶段无需再次公示。公示期内可撤回退店申请并恢复经营。
**3. ****管控：**为保障消费者权益，经营中商家提交退店申请后，店铺首页进行退店公示、下架全部商品且无法发布新品及上架待售商品。试运营店铺若申请退店，将冻结资金账户；待店铺公示期结束且所有订单履约完结，所有售后、纠纷、赔付处理完毕，无未完成的结算单、收付费单、账单等单据，无待核销的权益发票，无倒挂欠款等问题后解冻，并于1-3个工作日内退还消费者保证金至钱包余额。
**（二）合同到期：**店铺与京东合作期已满且未完成合同换签或续签。合同到期1年后店铺名称可供其他商家申请使用。
**（三）店铺清退：**店铺因违反国家法律、行政法规、部门规章等规范性文件，《“京东 JD.COM”开放平台服务协议》、《京东开放平台商家违规管理总则》等协议/规则的，京东有权对店铺执行清退，并采取包括但不限于扣除违约金、延迟结算、资金冻结、页面止付等处置措施。清退1年后店铺名称可供其他商家申请使用。
**第三条** 退保证金的相关要求，详见《[京东开放平台商家保证金管理规则](https://rule.jd.com/rule/ruleDetail.action?ruleId=2472&btype=1)》 第七章 保证金退还。
# **第三章 附则**
**第四条 **京东商家的行为，发生在本规则生效之日以前的，适用当时的规则。发生在本规则生效之日以后的，适用本规则。
**第五条 **京东可根据平台运营情况随时调整本管理规则并向商家公示。
**第六条 **商家应遵守国家法律、行政法规、部门规章等规范性文件。对任何涉嫌违反国家法律、行政法规、部门规章等规范性文件的行为，本规则已有约定的，适用于本规则。本规则尚无约定的，京东有权酌情处理。但京东对商家的处理不免除其应承担的法律责任。商家在京东的任何行为，应同时遵守与京东及其关联公司签订的各项协议。
**第七条 **本规则于2024年6月24日修订，于2024年7月1日生效。
