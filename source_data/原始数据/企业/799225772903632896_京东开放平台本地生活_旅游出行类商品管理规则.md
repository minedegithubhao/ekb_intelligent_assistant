# [][]第一章 概述
**第一条 **为保障消费者的合法权益，规范商家本地生活/旅游出行类商品（以下或称“旅游产品/服务”）经营，特制定本规则。
**第二条 **本规则适用于在京东开放平台（以下或称“京东”）上经营本地生活/旅游出行类商品（包括但不限于机票预定服务、火车票预定服务、景区门票服务、酒店预定服务、演出票务服务、生活服务/生活缴费服务、签证送关服务、旅游度假服务）的商家。
**第三条** 商家需符合[《京东开放平台本地生活/旅游出行/到家服务/二手回收类目资质标准》](https://rule.jd.com/rule/ruleDetail.action?ruleId=723066649191780352&btype=1)下对应类目的资质要求，旅游产品/服务的发布与经营需遵守[《生活旅行业务行业标准》](https://psm.jd.com/search/getSearchList?key=%25E7%2594%259F%25E6%25B4%25BB%25E6%2597%2585%25E8%25A1%258C%25E4%25B8%259A%25E5%258A%25A1%25E8%25A1%258C%25E4%25B8%259A%25E6%25A0%2587%25E5%2587%2586)。
**第四条 **商家向消费者提供旅游产品/服务的，应遵守下述约定：
（一）按照国家相关规定，与消费者签署相关合同，明确商家与消费者在旅游服务法律关系中的相应权利与义务；
（二）商家保证消费者在购买旅游产品/服务并支付成功后，向消费者开具合法合规发票、行程单及消费者所必需的票据、证明及其他资料，保证消费者可以正常使用旅游产品/服务。商家不得因旅游产品/服务价格变动，向消费者加价或以任何形式、名义向消费者额外收取任何费用；
（三）除异常天气、航空管制等不可抗力原因外，因任何原因导致座位被取消、出票失败或其他消费者无法正常使用旅游产品/服务的情形，商家应及时通知京东并负责联系消费者，由商家向消费者重新提供不劣于原订单价位及标准的旅游产品/服务，保证消费者出行和正常使用旅游产品/服务，相关差价及给京东或消费者造成的全部损失（包括但不限于实际损失、预期利益损失、商誉损失及其他直接和间接损失等，如损失难以计算的，以商家因违约行为获得的全部收益视为损失）由商家全额承担。若商家未能重新提供不劣于原订单价位及标准的旅游产品/服务，或消费者拒绝接受商家重新提供的旅游产品/服务，商家应全额退回消费者已支付的订单款项，并赔偿消费者和京东的全部损失。如商家未能及时处理消费者投诉，京东有权介入处理并先行予以赔付（非京东义务），且有权在商家应结款项、保证金、京东钱包余额等账户中扣除相应的赔付金额；
（四）无论在合作期内还是停止合作后，商家保证在消费者提出退票、退款申请时，按照旅游产品/服务实际提供商对外规定及时为消费者提供退票、退款服务（如商家对消费者作出更有利承诺或本规则对消费者更有利的，以商家承诺或本规则为准）。如因商家未及时为消费者办理退票、退款导致消费者投诉、产生争议、纠纷，应由商家独立承担全部法律责任，并赔偿消费者与京东的全部损失。
# [][]第二章 交易经营规范
## [][]第一节 机票预定服务
**第五条 **国内机票消费者拍下机票商品并支付成功后，商家须在30分钟内出票；国际机票商家应在60分钟内出票，特殊产品不应超过24小时。若商家与消费者关于出票时间另有约定的，则商家应在约定时间内出票。
**第六条 **商品票价不得高于实际票价，不得降舱销售。
**第七条** 若消费者自愿退票的，商家需在接到消费者退票申请后1个工作日内发送退款消息至京东；若因异常天气、航司变更、疫情等不可抗力因素造成消费者非自愿退票的，商家需在接到消费者退票申请后1-7个工作日内发送退款消息至京东。
**第八条** 商家需在收到航班变更信息1小时内通知消费者。起飞当天航班变更的，商家需在收到变更信息30分钟内告知消费者；如未联系上消费者，航班起飞前商家需至少持续保持5次联系，并发送航变信息到消费者预留联系方式。
**第九条** 商家需在商品详情页显著位置明示退票手续费收费标准，并按航司要求收取手续费。
**第十条** 若消费者需要改期的，商家需在消费者支付改期费起1小时内进行处理，不得无理由拒绝消费者改期需求。
**第十一条** 消费者申请行程单的，商家应在航班起飞后3天内邮寄并回填相关邮寄及行程单信息。
## [][]第二节 火车票预定服务
**第十二条** 消费者下单后商家需完整提供车次信息，包括车次、站点、时间、座位/座席、预计发售时间等信息。
**第十三条** 若消费者自愿退票的，商家需在接到消费者退票申请后24小时内发送退款消息至京东；若因天气、铁路公司/铁路局变更、疫情等不可抗力因素造成消费者非自愿退票的，商家需在接到消费者退票申请后3个工作日内发送退款消息至京东；退票手续费按12306规定收取。
**第十四条** 若消费者申请退票，商家须于30分钟内进行处理。
**第十五条** 若产生车次时间变动或停运，商家需在2小时内及时通知京东。
## [][]第三节 酒店预定服务
**第十六条 **如消费者预订当日入住的，商家须在消费者预订成功后2小时内完成订单确认；消费者次日入住的，商家须在消费者入住当日10:00前完成订单确认；消费者次日以后入住的，商家须在消费者预订成功后12小时内完成订单确认。
**第十七条**** 【到店无房】**如消费者到店无房，商家须在30分钟内为消费者安排入住，优先原单确认；若因酒店满房无法原单确认，则商家须确保原酒店升级房型安排消费者入住；若消费者所订房型或更高级别房型全部满房（酒店满房)，则商家应与消费者协调安排附近区域（2公里以内）同档次或更高档次酒店同级别房间入住，如有差价，由商家全额承担。商家应予赔付因此造成的消费者或京东全部损失，但赔付上限不超过首晚房费。当日及次日入住订单，商家需在得知满房消息起2小时内给出答复及解决方案；次日后入住订单，商家需在24小时内给出答复及解决方案。
**第十八条 【****到店变价**】商家需按原单价格确认安排客人入住，不得到店变价，不得以任何形式、任何名义加价。
**第十九条 ****【退改订服务标准】**消费者退订或改订时，商家需按在京东开放平台明示的规则退还消费者房费。消费者因不可抗力导致的无法入住，商家须协助消费者退订并安排全额退款。
## [][]第四节 演出票务服务
**第二十条 **商家须按文化和旅游部等主管部门的要求，在演出/项目获得相关审批文件后方可对门票进行售卖（包括预售）。
**第二十一条 **因不可抗力等因素导致主办方变更或取消演出的，商家须主动与消费者确认变更或退款事宜，不得收取任何手续费。
**第二十二条 **因主办方自身原因导致演出变更或取消的，商家须根据主办方通知为消费者办理变更或退款。
## [][]第五节 生活服务/生活缴费服务
**第二十三条 **提供生活缴费/生活服务服务的商家（如生活服务APP直充、油卡直充等），需在商品详情页显著位置标明商品信息，包括但不限于：
（一）充值服务适用范围及有效期；
（二）充值方式、充值流程、充值查询、到账时间等；
（三）售后服务，发票领取、充值信息填写错误责任声明等常见问题。
**第二十四条 **不得以分期交付或实际上以分期交付的形式经营充值类商品。
**第二十五条 **不得通过任何方式向消费者索取非交易必要的信息，例如验证码、密码等。
## []第六节 签证送关服务
**第二十六条：**若商家提供的旅游产品/服务中涉及签证送关服务的，商家还应遵守下述规则：
**（一）签证服务 **指由商家通过京东开放平台按消费者的具体需求为消费者提供代办签证的服务。
**（二）服务内容 **商家为消费者提供签证办理服务，服务内容包括但不限于向消费者发送办理签证所需资料清单，提供办理签证材料准备阶段的疑问解答，递送消费者提报资料、签证办理成功后的配送服务、发票开具及售后服务等。
**（三）服务承诺与保证**
1. 商家保证按照不同签证国需求，将签证办理所需材料、流程、周期、材料邮递地址、注意事项等相关规则在商品详情页显著位置明确公示，并应保证按照国家文化和旅游部相关规定、自身服务承诺及与京东的约定，提供商品服务及售后服务。
2. 消费者在线支付成功后，申请退款的，如消费者并未实际向商家邮递提报资料，或消费者已向商家邮递提报原件但使馆尚未受理消费者提报材料的，商家应为消费者办理退款，所涉及的配送费用，由消费者自行承担。
3. 商家保证妥善保管消费者材料，并对消费者信息严格保密。如消费者办理签证所需原件丢失（含因商家原因丢失、在使馆丢失、商家发货途中丢失），或发生任何消费者信息泄露情形，商家应独立向消费者承担赔偿责任。如商家怠于履行赔偿等义务，京东有权主动介入商家与消费者间的争议（非京东义务），依据一般人的认知程度对该争议进行判断和处置，商家应当予以执行。京东因此发生的全部损失，商家应予承担。
4. 商家出现如下情形的，京东有权随时终止合作并解除与商家签署的任何协议，如造成京东损失的，京东有权从待与商家结算的所有费用、保证金、商家京东钱包余额中扣除相应款项，不足以弥补损失的，京东保留向商家追偿的权利：
（1）因商家提供的服务不符合国家或行业标准或自行承诺标准，导致京东消费者投诉每月达2起的；
（2）因商家提供服务导致京东被举报、投诉、出现媒体负面报道事件每月达1起的。
## []第七节 旅游度假服务
**第二十七条 【商品上架】**对于度假类型的产品，商家需要按照不同目的地、不同出行日期进行店铺产品分类，团期、行程等所有旅行相关信息需要完整呈现在商品详情页中，并且对其中容易引起争议的部分进行详细的解答与提示。因商品详情页不完善、商详信息有误造成的消费者及京东全部损失，由商家100%承担。
**第二十八条 【订单修改】**商家不得擅自更改消费者已购商品的行程（不可抗力除外）。如因不可抗力原因造成消费者实际行程与商品详情页不符，商家应在第一时间以书面形式告知消费者，更改内容需消费者书面同意方生效。如商家在未获得消费者同意的前提下擅自修改行程，商家应全额赔偿消费者因更改部分造成的损失，消费者亦有权在出发前无责全额退款。
就“自由行/跟团游”“出境度假”“邮轮游”“国内旅游”“周边/当地游”“私人定制游”类目商品订单，以及定制旅行商品订单，商家未经消费者书面同意发生以下更改，按以下约定处理，且消费者亦有权在出发前无责全额退款：
（1）商家降低酒店等级、更改协议中入住酒店的，应向消费者支付违约金，违约金金额为订单金额的5%乘以订单中入住酒店的天数，但总额不超过订单金额的20%。
（2）商家擅自更改交通信息（包括飞机、高铁、火车、邮轮、大巴车等交通方式），如更改车次、航空公司、航班号、舱位等，应向消费者支付订单金额的10%作为违约金。
（3）商家擅自增加购物行程或其他消费者自费项目。商家应在商品详情页及行前与消费者签订的旅游合同中明确约定购物店/消费者自费项目的数量及时长。商家每擅自增加一个购物店/消费者自费项目数量或每延长购物/自费消费时间30分钟以上的，应以订单金额20%为标准向消费者支付违约金，但最高不超过订单金额的50%。
（4）商家未经消费者书面同意增加团队人数的，应向消费者支付订单金额的10%作为违约金。
（5）减少旅游景点或服务项目，或缩短游览时间30分钟及以上。商家发生该情形的，除向消费者支付订单金额10%的违约金外，还应根据减少的景点/项目及缩短的游览时间向消费者全额退回相关款项。
（6）商家以任何形式对消费者行程进行加价，商家应向消费者退回已收取的加价款，并应向消费者支付订单金额的10%作为违约金。
**第二十九条 【保密义务】**商家应对消费者与出行相关的任何信息进行保密，不得将消费者资料存储在第三方服务器上，不得以任何形式对外披露消费者的姓名、身份等信息。否则，商家应向消费者支付订单金额的50%作为违约金，且商家还因赔偿消费者、京东因商家信息泄露造成的全部损失。
# [][]第三章 违规责任
**第三十条** 若商家未按照国家法律法规、京东平台规则或其他要求（包括但不限于《京东行业运营标准》、规范、行业标准等），发生包括但不限于资质不合规、页面信息不合规、履约售后违规、服务违规及《京东开放平台商家违规管理总则》等平台规则约定的其他违规行为，京东有权依据所签协议、平台规则等进行治理，采取包括但不限于下架违规商品、店铺商品搜索降权、限制创建商品和禁止上架待售商品、停用相关品牌或类目、冻结并扣除商家保证金、应结款项、京东钱包等账户余额等处理措施及平台规则约定的其他处理措施。
**第三十一条 **京东有权依据商家经营情况，要求商家补交包括但不限于监管账户的备案金证明等资质信息，对商家店铺执行包括但不限于提高保证金、钱包阈值管理、延迟结算、中止/暂停相关服务等措施。
**第三十二条 **京东有权视商家违规严重程度，依据法律法规、所签协议、《京东开放平台关联店铺管理规则》等平台规则等，对违规商家的关联店铺立即做出处理。
# [][]第四章 终止合作
**第三十三条** 若因商家原因主动退店的，商家需提前30个工作日向京东进行申请，京东有权要求商家提供已出售商品明细，包括但不限于消费者卡券兑换进程、售卡数量等；商家对已售出卡券商品需提供对应的售后服务，对退店后不能继续履约的商品应支持消费者全额退款、退卡或与消费者协商沟通解决等。
**第三十四条** 京东有权视商家店铺实际经营情况，延长店铺退店、退保证金时间。
# [][]第五章 附则
**第三十五条** 京东开放平台商家的行为，发生在本管理规则生效之日以前的，适用当时的规则。发生在本管理规则生效之日以后的，适用本规则。
**第三十六条** 京东可根据平台运营情况随时调整本管理规则并向商家公示。
**第三十七条** 商家应遵守国家法律、行政法规、部门规章、政策文件等规范性文件。对任何涉嫌违反国家法律、行政法规、部门规章、政策文件等规范性文件的行为，本规则已有约定的，适用于本规则。本规则尚未约定的，京东有权依据其他平台规则处理。但京东对商家的处理不免除其应承担的法律责任。商家在京东的任何行为，应同时遵守与京东/京东关联方签订的各项协议。
**第三十八条** 本规则约定的商家应遵守的交易经营规范及违规责任，如与商家与京东/京东关联方所签协议（包括通过京东/京东关联方管理后台或邮件等方式确认的协议）或其他平台规则约定不一致的，以更严格的约定为准。
**第三十九条** 本规则于2025年4月23日修订，于2025年4月30日生效。
