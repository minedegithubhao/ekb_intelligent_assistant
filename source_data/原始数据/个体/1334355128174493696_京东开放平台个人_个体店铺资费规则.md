# 第一章 特别说明
**第一条** 京东可根据运营情况调整资费规则，相关规则调整将提前公示，客户订单下单时间发生在规则调整生效之日以前的，适用当时的规则，发生在规则调整生效之日以后的，适用最新的规则。
**第二条** 本规则将根据公示内容定期进行修订更新，如果公示的规则与本规则不一致，以最新公示的规则为准。
**第三条** 个人/个体店各经营类目对应的资费标准详见第三章 京东开放平台个人/个体店铺资费标准。
# 第二章 费项简要说明
**第四条 运营支持服务费**
商家根据经营类目在达成每一单交易时按比例向京东缴纳的费用，该费用从商家每笔订单的结算货款中扣除。
**第五条 交易服务费**
基于京东开放平台提供的服务产生的费用。2023年1月1日起，针对京东开放平台的POP商家，统一按0.6%费率收取交易服务费，即商家产生的每一单交易会产生一笔费用，该费用按照每单订单货款0.6%的比例扣除，然后结算给商家电子账户。2024年9月13日起，因用户原因导致的取消、售后退款，则退还对应的交易服务费。
